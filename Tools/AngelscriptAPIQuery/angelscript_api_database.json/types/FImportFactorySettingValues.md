# FImportFactorySettingValues

Holds FProperty names and values to customize factory settings

## 属性

### SettingName
- **类型**: `FString`
- **描述**: Name of the property to change.  Nested settings can be modified using "Outer.Property

### Value
- **类型**: `FString`
- **描述**: Value to import for the specified property.

## 方法
