# FRigUnit_CollectionChildren

Creates a collection based on the direct or recursive children
of a provided parent item. Returns an empty collection for an invalid parent item.

## 属性

### Parent
- **类型**: `FRigElementKey`

### bIncludeParent
- **类型**: `bool`

### bRecursive
- **类型**: `bool`

### TypeToSearch
- **类型**: `ERigElementType`

### Collection
- **类型**: `FRigElementKeyCollection`
