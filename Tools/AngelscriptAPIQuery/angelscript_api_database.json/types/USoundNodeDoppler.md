# USoundNodeDoppler

**继承自**: `USoundNode`

Computes doppler pitch shift

## 属性

### DopplerIntensity
- **类型**: `float32`
- **描述**: How much to scale the doppler shift (1.0 is normal).

### bUseSmoothing
- **类型**: `bool`
- **描述**: Whether or not to do a smooth interp to our doppler

### SmoothingInterpSpeed
- **类型**: `float32`
- **描述**: Speed at which to interp pitch scale
