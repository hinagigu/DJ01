# FRigUnit_GetControlInteger

GetControlFloat is used to retrieve a single Integer from a hierarchy.

## 属性

### Control
- **类型**: `FName`

### IntegerValue
- **类型**: `int`
- **描述**: The current value of the control.

### Minimum
- **类型**: `int`
- **描述**: The minimum value of the control.

### Maximum
- **类型**: `int`
- **描述**: The maximum value of the control.
