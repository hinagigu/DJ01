# FCacheRecorderUserParameters

## 属性

### bMaximizeViewport
- **类型**: `bool`
- **描述**: Whether to maximize the viewport (enter Immersive Mode) when recording

### EngineTimeDilation
- **类型**: `float32`
- **描述**: The engine time dilation to apply during the recording

### bResetPlayhead
- **类型**: `bool`
- **描述**: Reset playhead to beginning of the playback range when starting recording

### bStopAtPlaybackEnd
- **类型**: `bool`
- **描述**: Automatically stop recording when reaching the end of the playback range
