# FInterpCurveFloat

Describes an entire curve that is used to compute a float output value from a float input.
@note This is a mirror of TInterpCurve<float>, defined in InterpCurve.h

## 属性

### Points
- **类型**: `TArray<FInterpCurvePointFloat>`

### bIsLooped
- **类型**: `bool`

### LoopKeyOffset
- **类型**: `float32`
