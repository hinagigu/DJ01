# FCustomAttributeSetting

## 属性

### Name
- **类型**: `FString`
- **描述**: Name of the custom attribute

### Meaning
- **类型**: `FString`
- **描述**: Optional property describing the meaning (or role) of the custom attribute, allowing to add context to an attribute
