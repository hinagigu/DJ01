# URemeshProperties

**继承自**: `UMeshConstraintProperties`

## 属性

### SmoothingStrength
- **类型**: `float32`
- **描述**: Amount of Vertex Smoothing applied within Remeshing

### bFlips
- **类型**: `bool`
- **描述**: Enable edge flips in Remeshing (tends to lose edges/detail)

### bSplits
- **类型**: `bool`
- **描述**: Enable edge splits in Remeshing

### bCollapses
- **类型**: `bool`
- **描述**: Enable edge collapses in Remeshing
