# FRigUnit_GetShapeTransform

GetShapeTransform is used to retrieve single control's shape transform.
This is typically only used during the Construction Event.

## 属性

### Control
- **类型**: `FName`

### Transform
- **类型**: `FTransform`
- **描述**: The shape transform to set for the control

## 方法

### opAssign
```angelscript
FRigUnit_GetShapeTransform& opAssign(FRigUnit_GetShapeTransform Other)
```
