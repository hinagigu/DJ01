# UMovieSceneNiagaraSystemSpawnSection

**继承自**: `UMovieSceneSection`

## 属性

### SectionStartBehavior
- **类型**: `ENiagaraSystemSpawnSectionStartBehavior`
- **描述**: Specifies what should happen to the niagara system from before the section evaluates up until the first frame of the section.

### SectionEvaluateBehavior
- **类型**: `ENiagaraSystemSpawnSectionEvaluateBehavior`
- **描述**: Specifies what should happen to the niagara system when section is evaluating from the 2nd frame until the last frame.

### SectionEndBehavior
- **类型**: `ENiagaraSystemSpawnSectionEndBehavior`
- **描述**: Specifies what should happen to the niagara system when section evaluation finishes.

### AgeUpdateMode
- **类型**: `ENiagaraAgeUpdateMode`
- **描述**: Specifies how sequencer should update the age of the controlled niagara system.

### bAllowScalability
- **类型**: `bool`
