# UUVEditorUVTransformProperties

**继承自**: `UUVEditorUVTransformPropertiesBase`

UV Transform Settings

## 属性

### Scale
- **类型**: `FVector2D`
- **描述**: Scale applied to UVs, potentially non-uniform

### Rotation
- **类型**: `float32`
- **描述**: Rotation applied to UVs after scaling, specified in degrees

### Translation
- **类型**: `FVector2D`
- **描述**: Translation applied to UVs, and after scaling and rotation

### TranslationMode
- **类型**: `EUVEditorTranslationMode`
- **描述**: Translation applied to UVs, and after scaling and rotation

### PivotMode
- **类型**: `EUVEditorPivotType`
- **描述**: Transformation origin mode used for scaling and rotation

### ManualPivot
- **类型**: `FVector2D`
- **描述**: Manual Transformation origin point

### QuickTranslateOffset
- **类型**: `float32`

### QuickRotationOffset
- **类型**: `float32`

### QuickTranslation
- **类型**: `FVector2D`

### QuickRotation
- **类型**: `float32`
