# APhysicsVolume

**继承自**: `AVolume`

PhysicsVolume: A bounding volume which affects actor physics.
Each AActor is affected at any time by one PhysicsVolume.

- **类型**: `float32`

### Priority
- **类型**: `int`

### FluidFriction
- **类型**: `float32`

### bWaterVolume
- **类型**: `bool`

### bPhysicsOnContact
- **类型**: `bool`
