# UTransferMeshToolProperties

**继承自**: `UInteractiveToolPropertySet`

Standard properties of the Transfer operation

## 属性

### bTransferMaterials
- **类型**: `bool`

### bTransferCollision
- **类型**: `bool`

### SourceLOD
- **类型**: `FString`
- **描述**: Specify which LOD from the Source (First) mesh to copy from

### TargetLOD
- **类型**: `FString`
- **描述**: Specify which LOD on the Target (Second) mesh to copy to
