# UInputModifierDeadZone

**继承自**: `UInputModifier`

Dead Zone
Input values within the range LowerThreshold -> UpperThreshold will be remapped from 0 -> 1.
Values outside this range will be clamped.

## 属性

### LowerThreshold
- **类型**: `float32`

### UpperThreshold
- **类型**: `float32`

### Type
- **类型**: `EDeadZoneType`
