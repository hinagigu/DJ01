# UGameFeatureAction_DataRegistrySource

**继承自**: `UGameFeatureAction`

Specifies a list of source assets to add to Data Registries when this feature is activated

## 属性

### SourcesToAdd
- **类型**: `TArray<FDataRegistrySourceToAdd>`
- **描述**: List of sources to add when this feature is activated

### bPreloadInEditor
- **类型**: `bool`
- **描述**: If true, this will preload the sources when the feature is registered in the editor to support the editor pickers
