# FCollectionTransformSelectionTargetLevelDataflowNode

Selects the root bones in the Collection

## 属性

### TargetLevel
- **类型**: `int`
- **描述**: Level to select

### bSkipEmbedded
- **类型**: `bool`
- **描述**: Whether to avoid embedded geometry in the selection (i.e., only select rigid and cluster nodes)

- **类型**: `bool`
