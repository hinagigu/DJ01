# UNewPluginDescriptorData

**继承自**: `UObject`

An object used internally by the New Plugin Wizard to set user-defined plugin descriptor information.
This should not be used outside of the New Plugin Wizard dialog

## 属性

### CreatedBy
- **类型**: `FString`
- **描述**: The author of this content

### CreatedByURL
- **类型**: `FString`
- **描述**: Optional hyperlink for the author's website

### Description
- **类型**: `FString`
- **描述**: A description for this content

### bIsBetaVersion
- **类型**: `bool`
- **描述**: Marks this content as being in beta
