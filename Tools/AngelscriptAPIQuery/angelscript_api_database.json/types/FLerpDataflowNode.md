# FLerpDataflowNode

Linearly interpolates between A and B based on Alpha. (100% of A when Alpha = 0.0 and 100% of B when Alpha = 1.0)

## 属性

### A
- **类型**: `float32`

### B
- **类型**: `float32`

### Alpha
- **类型**: `float32`

### bActive
- **类型**: `bool`

### opAssign
