# UTileSetEditorSettings

**继承自**: `UObject`

Settings for the Paper2D tile set editor

## 属性

### DefaultBackgroundColor
- **类型**: `FColor`
- **描述**: Default background color for new tile set assets

### bShowGridByDefault
- **类型**: `bool`
- **描述**: Should the grid be shown by default when the editor is opened?

### ExtrusionAmount
- **类型**: `int`
- **描述**: The amount to extrude out from the edge of each tile (in pixels)

### bPadToPowerOf2
- **类型**: `bool`
- **描述**: Should we pad the conditioned texture to the next power of 2?

### bFillWithTransparentBlack
- **类型**: `bool`
- **描述**: Should we use transparent black or white when filling the texture areas that aren't covered by tiles?
