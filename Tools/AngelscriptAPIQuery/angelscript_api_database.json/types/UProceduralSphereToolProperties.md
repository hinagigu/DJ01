# UProceduralSphereToolProperties

**继承自**: `UProceduralShapeToolProperties`

## 属性

### Radius
- **类型**: `float32`
- **描述**: Radius of the sphere

### SubdivisionType
- **类型**: `EProceduralSphereType`
- **描述**: Type of subdivision for the sphere

### Subdivisions
- **类型**: `int`
- **描述**: Number of subdivisions for each side of the sphere. This is only available for spheres with Box subdivision.

### HorizontalSlices
- **类型**: `int`
- **描述**: Number of horizontal slices of the sphere. This is only available for spheres with Lat Long subdivision.

### VerticalSlices
- **类型**: `int`
- **描述**: Number of vertical slices of the sphere. This is only available for spheres with Lat Long subdivision.
