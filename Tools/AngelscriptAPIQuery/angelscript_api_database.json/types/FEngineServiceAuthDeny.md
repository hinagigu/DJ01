# FEngineServiceAuthDeny

Implements a message for denying service access to a remote user.

## 属性

### UserName
- **类型**: `FString`
- **描述**: Holds the name of the user that denied access.

### UserToDeny
- **类型**: `FString`
- **描述**: Holds the name of the user that access is denied to.
