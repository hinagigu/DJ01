# UChanceToApplyGameplayEffectComponent

**继承自**: `UGameplayEffectComponent`

Applies a probablity to the application conditions of the Gameplay Effect.

## 属性

### ChanceToApplyToTarget
- **类型**: `FScalableFloat`
- **描述**: Probability that this gameplay effect will be applied to the target actor (0.0 for never, 1.0 for always)
