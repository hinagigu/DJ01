# UGameFeatureData

**继承自**: `UPrimaryDataAsset`

Data related to a game feature, a collection of code and content that adds a separable discrete feature to the game

## 属性

### Actions
- **类型**: `TArray<TObjectPtr<UGameFeatureAction>>`
- **描述**: List of actions to perform as this game feature is loaded/activated/deactivated/unloaded

### PrimaryAssetTypesToScan
- **类型**: `TArray<FPrimaryAssetTypeInfo>`
- **描述**: List of asset types to scan at startup
