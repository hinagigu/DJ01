# FControlRigSnapperSelection

Selection from the UI to Snap To. Contains a set of Actors and/or Control Rigs to snap onto or from.

## 属性

### Actors
- **类型**: `TArray<FActorForWorldTransforms>`

### ControlRigs
- **类型**: `TArray<FControlRigForWorldTransforms>`
