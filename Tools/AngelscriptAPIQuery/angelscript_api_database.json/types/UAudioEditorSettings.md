# UAudioEditorSettings

**继承自**: `UDeveloperSettings`

## 属性

### bUseAudioAttenuation
- **类型**: `bool`
- **描述**: Whether or not should Audio Attenuation be used by default, for Non-Game Worlds

### bPinSoundCueInAssetMenu
- **类型**: `bool`
- **描述**: Whether to pin the Sound Cue asset type when creating new assets. Requires editor restart to take effect.

### bPinSoundCueTemplateInAssetMenu
- **类型**: `bool`
- **描述**: Whether to pin the Sound Cue Template asset type when creating new assets. Requires editor restart to take effect.

### bPinSoundAttenuationInAssetMenu
- **类型**: `bool`
- **描述**: Whether to pin the Sound Attenuation asset type when creating new assets. Requires editor restart to take effect.

### bPinSoundConcurrencyInAssetMenu
- **类型**: `bool`
- **描述**: Whether to pin the Sound Concurrency asset type when creating new assets. Requires editor restart to take effect.
