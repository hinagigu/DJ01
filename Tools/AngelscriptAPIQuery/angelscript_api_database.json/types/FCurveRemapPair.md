# FCurveRemapPair

## 属性

### SourceCurve
- **类型**: `FName`
- **描述**: The curve name on the SOURCE skeletal mesh to copy animation data from.

### TargetCurve
- **类型**: `FName`
- **描述**: The curve name on the TARGET skeletal mesh to receive animation data.
