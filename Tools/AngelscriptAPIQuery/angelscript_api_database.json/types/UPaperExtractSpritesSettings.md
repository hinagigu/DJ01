# UPaperExtractSpritesSettings

**继承自**: `UObject`

## 属性

### SpriteExtractMode
- **类型**: `ESpriteExtractMode`
- **描述**: Sprite extract mode

### OutlineColor
- **类型**: `FLinearColor`
- **描述**: The color of the sprite boundary outlines

### ViewportTextureTint
- **类型**: `FLinearColor`
- **描述**: Apply a tint to the texture in the viewport to improve outline visibility in this editor

### BackgroundColor
- **类型**: `FLinearColor`
- **描述**: The viewport background color

### NamingTemplate
- **类型**: `FString`
- **描述**: The name of the sprite that will be created. {0} will get replaced by the sprite number.

### NamingStartIndex
- **类型**: `int`
- **描述**: The number to start naming with
