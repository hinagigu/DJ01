# UTileSheetPaddingFactory

**继承自**: `UFactory`

Factory used to pad out each individual tile in a tile sheet texture

## 属性

### ExtrusionAmount
- **类型**: `int`
- **描述**: The amount to extrude out from each tile (in pixels)

### bPadToPowerOf2
- **类型**: `bool`
- **描述**: Should we pad the texture to the next power of 2?

### bFillWithTransparentBlack
- **类型**: `bool`
- **描述**: Should we use transparent black or white when filling the texture areas that aren't covered by tiles?
