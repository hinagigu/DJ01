# FMode2DLayer

2D layer settings

## 属性

### Name
- **类型**: `FString`
- **描述**: A descriptive name for this snap layer.

### Depth
- **类型**: `float32`
- **描述**: The position of this snap layer's plane along the Snap Axis.
