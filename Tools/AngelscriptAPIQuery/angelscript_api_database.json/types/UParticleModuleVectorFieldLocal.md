# UParticleModuleVectorFieldLocal

**继承自**: `UParticleModuleVectorFieldBase`

## 属性

### VectorField
- **类型**: `UVectorField`
- **描述**: Vector field asset to use.

### RelativeTranslation
- **类型**: `FVector`
- **描述**: Translation of the vector field relative to the emitter.

### RelativeRotation
- **类型**: `FRotator`
- **描述**: Rotation of the vector field relative to the emitter.

### RelativeScale3D
- **类型**: `FVector`
- **描述**: Scale of the vector field relative to the emitter.

### Intensity
- **类型**: `float32`
- **描述**: Intensity of the local vector field.

### Tightness
- **类型**: `float32`
- **描述**: Tightness tweak value: 0: Force 1: Velocity.

### bIgnoreComponentTransform
- **类型**: `bool`

### bTileX
- **类型**: `bool`

### bTileY
- **类型**: `bool`

### bTileZ
- **类型**: `bool`

### bUseFixDT
- **类型**: `bool`
