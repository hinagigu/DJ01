# FMVVMBlueprintViewBinding

## 属性

### BindingType
- **类型**: `EMVVMBindingMode`

### OverrideExecutionMode
- **类型**: `EMVVMExecutionMode`

### bEnabled
- **类型**: `bool`
- **描述**: Whether the binding is enabled or disabled by default. The instance may enable the binding at runtime.

### bCompile
- **类型**: `bool`
- **描述**: The binding is visible in the editor, but is not compiled and cannot be used at runtime.
