# UCircularThrobber

**继承自**: `UWidget`

A throbber widget that orients images in a spinning circle.

* No Children
* Spinner Progress

## 属性

### Image
- **类型**: `FSlateBrush`

### bEnableRadius
- **类型**: `bool`

### NumberOfPieces
- **类型**: `int`

### Period
- **类型**: `float32`

### Radius
- **类型**: `float32`

## 方法

### SetNumberOfPieces
```angelscript
void SetNumberOfPieces(int InNumberOfPieces)
```
Sets how many pieces there are.

### SetPeriod
```angelscript
void SetPeriod(float32 InPeriod)
```
Sets the amount of time for a full circle (in seconds).

### SetRadius
```angelscript
void SetRadius(float32 InRadius)
```
Sets the radius of the circle.
