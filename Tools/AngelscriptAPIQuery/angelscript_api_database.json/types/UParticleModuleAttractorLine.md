# UParticleModuleAttractorLine

**继承自**: `UParticleModuleAttractorBase`

## 属性

### EndPoint0
- **类型**: `FVector`
- **描述**: The first endpoint of the line.

### EndPoint1
- **类型**: `FVector`
- **描述**: The second endpoint of the line.

### Range
- **类型**: `FRawDistributionFloat`
- **描述**: The range of the line attractor.

### Strength
- **类型**: `FRawDistributionFloat`
- **描述**: The strength of the line attractor.
