# UImplicitOffsetProperties

**继承自**: `UInteractiveToolPropertySet`

Properties for Implicit Offsetting

## 属性

### Smoothness
- **类型**: `float32`
- **描述**: How tightly we should constrain the constrained implicit offset to the explicit offset

### bPreserveUVs
- **类型**: `bool`
- **描述**: If this is false, triangles will be reshaped to be more regular, which will distort UVs
