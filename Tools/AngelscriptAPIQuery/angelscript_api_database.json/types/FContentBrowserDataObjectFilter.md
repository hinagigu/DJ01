# FContentBrowserDataObjectFilter

Data used to filter object instances by their name and tags.
@note This will typically limit your query to returning assets.

- **类型**: `TArray<FName>`

### ObjectNamesToExclude
- **类型**: `TArray<FName>`

### bOnDiskObjectsOnly
