# UTimelineTemplate

**继承自**: `UObject`

## 属性

### TimelineLength
- **类型**: `float32`
- **描述**: Length of this timeline

### LengthMode
- **类型**: `ETimelineLengthMode`
- **描述**: How we want the timeline to determine its own length (e.g. specified length, last keyframe)

### MetaDataArray
- **类型**: `TArray<FBPVariableMetaDataEntry>`
- **描述**: Metadata information for this timeline

### bAutoPlay
- **类型**: `bool`

### bLoop
- **类型**: `bool`

### bReplicated
- **类型**: `bool`

### bIgnoreTimeDilation
- **类型**: `bool`
