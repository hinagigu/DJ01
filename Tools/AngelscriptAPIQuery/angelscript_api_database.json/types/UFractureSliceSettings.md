# UFractureSliceSettings

**继承自**: `UFractureToolSettings`

## 属性

### SlicesX
- **类型**: `int`
- **描述**: Number of slices along the X axis

### SlicesY
- **类型**: `int`
- **描述**: Number of slices along the Y axis

### SlicesZ
- **类型**: `int`
- **描述**: Number of slices along the Z axis

### SliceAngleVariation
- **类型**: `float32`
- **描述**: Maximum angle (in degrees) to randomly rotate each slicing plane

### SliceOffsetVariation
- **类型**: `float32`
- **描述**: Maximum distance (in cm) to randomly shift each slicing plane
