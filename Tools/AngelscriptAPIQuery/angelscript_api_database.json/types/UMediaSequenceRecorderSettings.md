# UMediaSequenceRecorderSettings

**继承自**: `UObject`

## 属性

### bRecordMediaPlayerEnabled
- **类型**: `bool`
- **描述**: Whether to enabled MediaPlayer recording into this sequence.

### MediaPlayerSubDirectory
- **类型**: `FString`
- **描述**: The name of the subdirectory MediaPlayer will be placed in. Leave this empty to place into the same directory as the sequence base path
