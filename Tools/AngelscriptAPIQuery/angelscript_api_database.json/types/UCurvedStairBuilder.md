# UCurvedStairBuilder

**继承自**: `UEditorBrushBuilder`

## 属性

### InnerRadius
- **类型**: `int`
- **描述**: The radius of the inner curve of the stair

### StepHeight
- **类型**: `int`
- **描述**: The height of each step

### StepWidth
- **类型**: `int`
- **描述**: The width of each step

### AngleOfCurve
- **类型**: `int`
- **描述**: The angle of the total arc described by this stair

### NumSteps
- **类型**: `int`
- **描述**: The number of steps

### AddToFirstStep
- **类型**: `int`
- **描述**: The distance below the first step

### CounterClockwise
- **类型**: `bool`
