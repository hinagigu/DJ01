# UCubeBuilder

**继承自**: `UEditorBrushBuilder`

## 属性

### X
- **类型**: `float32`
- **描述**: The size of the cube in the X dimension

### Y
- **类型**: `float32`
- **描述**: The size of the cube in the Y dimension

### Z
- **类型**: `float32`
- **描述**: The size of the cube in the Z dimension

### WallThickness
- **类型**: `float32`
- **描述**: The thickness of the cube wall when hollow

### Hollow
- **类型**: `bool`

### Tessellated
- **类型**: `bool`
