# FRigUnit_GetControlDrivenList

GetControlDrivenList is used to retrieve the list of affected controls of an indirect control

## 属性

### Control
- **类型**: `FName`

### Driven
- **类型**: `TArray<FRigElementKey>`
- **描述**: The list of affected controls
