# __EPlayOnBuildMode

Determines whether to build the executable when launching on device. Note the equivalence between these settings and EProjectPackagingBuild.

## 属性

### PlayOnBuild_Always
- **类型**: `EPlayOnBuildMode`

### PlayOnBuild_Never
- **类型**: `EPlayOnBuildMode`

### PlayOnBuild_Default
- **类型**: `EPlayOnBuildMode`

### PlayOnBuild_IfEditorBuiltLocally
- **类型**: `EPlayOnBuildMode`

### PlayOnBuild_MAX
- **类型**: `EPlayOnBuildMode`
