# FRigUnit_GetControlFloat

GetControlFloat is used to retrieve a single Float from a hierarchy.

## 属性

### Control
- **类型**: `FName`

### FloatValue
- **类型**: `float32`
- **描述**: The current value of the control.

### Minimum
- **类型**: `float32`
- **描述**: The minimum value of the control.

### Maximum
- **类型**: `float32`
- **描述**: The maximum value of the control.

## 方法

### opAssign
```angelscript
FRigUnit_GetControlFloat& opAssign(FRigUnit_GetControlFloat Other)
```
