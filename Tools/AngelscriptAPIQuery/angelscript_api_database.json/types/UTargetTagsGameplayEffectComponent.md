# UTargetTagsGameplayEffectComponent

**继承自**: `UGameplayEffectComponent`

Handles granting Tags to the Target (sometimes referred to as the Owner) of the Gameplay Effect

## 属性

### InheritableGrantedTagsContainer
- **类型**: `FInheritedTagContainer`
- **描述**: These tags are applied (granted) to the target actor of the Gameplay Effect.  The Target would then "own" these Tags.
