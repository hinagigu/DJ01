# UAbilityTask_PlayMontageAndWait

**继承自**: `UAbilityTask`

Ability task to simply play a montage. Many games will want to make a modified version of this task that looks for game-specific events

## 属性

- **类型**: `FMontageWaitSimpleDelegate`

### OnBlendOut
- **类型**: `FMontageWaitSimpleDelegate`

### OnInterrupted
- **类型**: `FMontageWaitSimpleDelegate`

### OnCancelled
- **类型**: `FMontageWaitSimpleDelegate`
