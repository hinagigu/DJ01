# FRigUnit_RemoveMetadata

Removes an existing metadata filed from an item

## 属性

### Item
- **类型**: `FRigElementKey`

### Name
- **类型**: `FName`

### NameSpace
- **类型**: `ERigMetaDataNameSpace`

### Removed
- **类型**: `bool`
- **描述**: True if the metadata has been removed

### ExecuteContext
- **类型**: `FControlRigExecuteContext`

## 方法

### opAssign
```angelscript
FRigUnit_RemoveMetadata& opAssign(FRigUnit_RemoveMetadata Other)
```
