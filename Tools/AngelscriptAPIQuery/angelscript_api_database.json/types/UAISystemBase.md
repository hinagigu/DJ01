# UAISystemBase

**继承自**: `UObject`

## 属性

### AISystemClassName
- **类型**: `FSoftClassPath`
- **描述**: List of specific AI system class to create, can be game-specific

### AISystemModuleName
- **类型**: `FName`
- **描述**: Name of a module used to spawn the AI system. If not empty, this module has to implement IAISystemModule
