# UBlendSpace1D

**继承自**: `UBlendSpace`

Allows multiple animations to be blended between based on input parameters

## 属性

### bScaleAnimation
- **类型**: `bool`
- **描述**: If you have input smoothing, whether to scale the animation speed. E.g. for locomotion animation,
the speed axis will scale the animation speed in order to make up the difference between the target
and the result of blending the samples.
