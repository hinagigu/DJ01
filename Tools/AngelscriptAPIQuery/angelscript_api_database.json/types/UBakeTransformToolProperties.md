# UBakeTransformToolProperties

**继承自**: `UInteractiveToolPropertySet`

Standard properties

## 属性

### bApplyToAllLODs
- **类型**: `bool`
- **描述**: If checked, the baked transform will be applied to all available LODs. Has no effect on selections without LODs.

### bBakeRotation
- **类型**: `bool`
- **描述**: Bake rotation

### BakeScale
- **类型**: `EBakeScaleMethod`
- **描述**: Bake scale

### bRecenterPivot
- **类型**: `bool`
- **描述**: Recenter pivot after baking transform
