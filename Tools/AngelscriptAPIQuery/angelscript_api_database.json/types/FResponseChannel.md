# FResponseChannel

Describes response for a single collision response channel

## 属性

### Channel
- **类型**: `FName`
- **描述**: This should match DisplayName of ECollisionChannel
    Meta data of custom channels can be used as well

### Response
- **类型**: `ECollisionResponse`
- **描述**: Describes how the channel behaves
