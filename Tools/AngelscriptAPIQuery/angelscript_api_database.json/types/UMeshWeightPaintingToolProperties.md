# UMeshWeightPaintingToolProperties

**继承自**: `UMeshVertexPaintingToolProperties`

## 属性

### TextureWeightType
- **类型**: `EMeshPaintWeightTypes`
- **描述**: Texture Blend Weight Painting Mode

### PaintTextureWeightIndex
- **类型**: `EMeshPaintTextureIndex`
- **描述**: Texture Blend Weight index which should be applied during Painting

### EraseTextureWeightIndex
- **类型**: `EMeshPaintTextureIndex`
- **描述**: Texture Blend Weight index which should be erased during Painting
