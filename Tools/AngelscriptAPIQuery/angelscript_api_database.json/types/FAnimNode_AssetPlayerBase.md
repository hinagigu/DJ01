# FAnimNode_AssetPlayerBase

Base class for any asset playing anim node

## 属性

### BlendWeight
- **类型**: `float32`
- **描述**: Last encountered blendweight for this node

### InternalTimeAccumulator
- **类型**: `float32`
- **描述**: Accumulated time used to reference the asset in this node
