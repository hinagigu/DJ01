# UEnhancedInputEditorSettings

**继承自**: `UDeveloperSettingsBackedByCVars`

A collection of useful individual user settings when using the EnhancedInputEditorSubsystem.

## 属性

### VisibleEventPinsByDefault
- **类型**: `uint8`
- **描述**: A bitmask of what event pins are visible when you place an Input Action event node in blueprints.

### bLogAllInput
- **类型**: `bool`

### bAutomaticallyStartConsumingInput
- **类型**: `bool`
