# UFractureSettings

**继承自**: `UObject`

Settings specifically related to viewing fractured meshes *

## 属性

### ExplodeAmount
- **类型**: `float32`
- **描述**: Amount to expand the displayed Geometry Collection bones into an 'exploded view'

### FractureLevel
- **类型**: `int`
- **描述**: Current level of the Geometry Collection displayed

### bHideUnselected
- **类型**: `bool`
- **描述**: When active, only show selected bones
