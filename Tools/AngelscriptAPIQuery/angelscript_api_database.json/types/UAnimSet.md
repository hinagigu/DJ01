# UAnimSet

**继承自**: `UObject`

## 属性

### UseTranslationBoneNames
- **类型**: `TArray<FName>`
- **描述**: Names of bones that should use translation from the animation, if bAnimRotationOnly is set. // Do not change private - they will go away

### ForceMeshTranslationBoneNames
- **类型**: `TArray<FName>`
- **描述**: List of bones which are ALWAYS going to use their translation from the mesh and not the animation. // Do not change private - they will go away

### bAnimRotationOnly
- **类型**: `bool`
