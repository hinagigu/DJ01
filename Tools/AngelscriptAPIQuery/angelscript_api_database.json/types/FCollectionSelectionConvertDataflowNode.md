# FCollectionSelectionConvertDataflowNode

Converts Vertex/Face/Transform selection into Vertex/Face/Transform selection

## 属性

### bAllElementsMustBeSelected
- **类型**: `bool`
- **描述**: If true then for converting vertex/face selection to transform selection all vertex/face must be selected for selecting the associated transform

### bActive
- **类型**: `bool`
