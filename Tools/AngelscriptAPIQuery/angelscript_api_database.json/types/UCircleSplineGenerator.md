# UCircleSplineGenerator

**继承自**: `USplineGeneratorBase`

## 属性

### NumberOfPoints
- **类型**: `int`
- **描述**: Number of points making up shape

### Radius
- **类型**: `float32`
- **描述**: Radius of circle

### bReverseDir
- **类型**: `bool`
- **描述**: If enabled, will reverse the direction of the circle

### bKeepFirstKeyTangent
- **类型**: `bool`
- **描述**: If enabled, will start the shape tangent to the current path

### bBranchRight
- **类型**: `bool`
- **描述**: If enabled, will switch the circle's center to the right of the curve
