# __EScrollWhenFocusChanges

Set behavior when user focus changes inside this scroll box

## 属性

### NoScroll
- **类型**: `EScrollWhenFocusChanges`

### InstantScroll
- **类型**: `EScrollWhenFocusChanges`

### AnimatedScroll
- **类型**: `EScrollWhenFocusChanges`

### EScrollWhenFocusChanges_MAX
- **类型**: `EScrollWhenFocusChanges`
