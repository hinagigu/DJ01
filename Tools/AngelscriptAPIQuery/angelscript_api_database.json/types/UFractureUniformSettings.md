# UFractureUniformSettings

**继承自**: `UFractureToolSettings`

## 属性

### NumberVoronoiSitesMin
- **类型**: `int`
- **描述**: Minimum Number of Voronoi sites. The amount of sites per Voronoi diagram will be chosen at random between Min and Max

### NumberVoronoiSitesMax
- **类型**: `int`
- **描述**: Maximum Number of Voronoi sites. The amount of sites per Voronoi diagram will be chosen at random between Min and Max
