# FUIActionKeyMapping

## 属性

### Key
- **类型**: `FKey`
- **描述**: A key that triggers this action

### HoldTime
- **类型**: `float32`
- **描述**: How long must the key be held down for the action to be executed?

### HoldRollbackTime
- **类型**: `float32`
- **描述**: Time (in seconds) for hold progress to go from 1.0 (completed) to 0.0.
