# UMaterialExpressionParameter

**继承自**: `UMaterialExpression`

## 属性

### ParameterName
- **类型**: `FName`
- **描述**: The name of the parameter

### Group
- **类型**: `FName`
- **描述**: The name of the parameter Group to display in MaterialInstance Editor. Default is None group

### SortPriority
- **类型**: `int`
- **描述**: Controls where the this parameter is displayed in a material instance parameter list.  The lower the number the higher up in the parameter list.
