# FNavMeshResolutionParam

## 属性

### CellSize
- **类型**: `float32`
- **描述**: Horizontal size of voxelization cell

### CellHeight
- **类型**: `float32`
- **描述**: Vertical size of voxelization cell

### AgentMaxStepHeight
- **类型**: `float32`
- **描述**: Largest vertical step the agent can perform
