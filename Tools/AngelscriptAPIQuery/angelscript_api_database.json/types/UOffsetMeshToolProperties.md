# UOffsetMeshToolProperties

**继承自**: `UInteractiveToolPropertySet`

Base properties of Offset

## 属性

### OffsetType
- **类型**: `EOffsetMeshToolOffsetType`
- **描述**: Type of Offsetting to apply

### Distance
- **类型**: `float32`
- **描述**: Offset Distance in World Units

### bCreateShell
- **类型**: `bool`
- **描述**: If true, create a thickened shell, instead of only moving the input vertices
