# UPatternTool_GridSettings

**继承自**: `UInteractiveToolPropertySet`

Settings for Grid Patterns in the Pattern Tool
TODO: maybe we can just re-use UPatternTool_LinearSettings for this??

## 属性

### SpacingX
- **类型**: `EPatternToolAxisSpacingMode`
- **描述**: Spacing Technique used to distribute Pattern Elements along the Main axis

### CountX
- **类型**: `int`
- **描述**: Number of Pattern Elements to place along the Main axis

### StepSizeX
- **类型**: `float`
- **描述**: Fixed Increment used to place Pattern Elements along the Main axis

### ExtentX
- **类型**: `float`
- **描述**: Length/Extent of Pattern falong the Main Axis

### bCenteredX
- **类型**: `bool`
- **描述**: If true, Pattern is centered at the Origin along the Main axis, otherwise Pattern starts at the Origin

### SpacingY
- **类型**: `EPatternToolAxisSpacingMode`
- **描述**: Spacing Technique used to distribute Pattern Elements along the Secondary axis

### CountY
- **类型**: `int`
- **描述**: Number of  Pattern Elements to place along the Secondary axis

### StepSizeY
- **类型**: `float`
- **描述**: Fixed Increment used to place Pattern Elements along the Secondary axis

### ExtentY
- **类型**: `float`
- **描述**: Length/Extent of Pattern falong the Secondary Axis

### bCenteredY
- **类型**: `bool`
- **描述**: If true, Pattern is centered at the Origin along the Secondary axis, otherwise Pattern starts at the Origin
