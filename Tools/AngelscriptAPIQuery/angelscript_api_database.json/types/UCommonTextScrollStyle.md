# UCommonTextScrollStyle

**继承自**: `UObject`

* ---- All properties must be EditDefaultsOnly, BlueprintReadOnly !!! -----
* We return the CDO to blueprints, so we cannot allow any changes (blueprint doesn't support const variables)

## 属性

### Speed
- **类型**: `float32`

### StartDelay
- **类型**: `float32`

### EndDelay
- **类型**: `float32`

### FadeInDelay
- **类型**: `float32`

### FadeOutDelay
- **类型**: `float32`

### Clipping
- **类型**: `EWidgetClipping`
