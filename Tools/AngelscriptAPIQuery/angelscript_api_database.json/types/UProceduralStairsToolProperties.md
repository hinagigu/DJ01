# UProceduralStairsToolProperties

**继承自**: `UProceduralShapeToolProperties`

## 属性

### StairsType
- **类型**: `EProceduralStairsType`
- **描述**: Type of staircase

### NumSteps
- **类型**: `int`
- **描述**: Number of steps

### StepWidth
- **类型**: `float32`
- **描述**: Width of each step

### StepHeight
- **类型**: `float32`
- **描述**: Height of each step

### StepDepth
- **类型**: `float32`
- **描述**: Depth of each step of linear stairs

### CurveAngle
- **类型**: `float32`
- **描述**: Angular length of curved stairs. Positive values are for clockwise and negative values are for counterclockwise.

### SpiralAngle
- **类型**: `float32`
- **描述**: Angular length of spiral stairs. Positive values are for clockwise and negative values are for counterclockwise.

### InnerRadius
- **类型**: `float32`
- **描述**: Inner radius of curved and spiral stairs
