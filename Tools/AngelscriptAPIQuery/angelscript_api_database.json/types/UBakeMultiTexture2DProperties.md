# UBakeMultiTexture2DProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### MaterialIDSourceTextures
- **类型**: `TArray<TObjectPtr<UTexture2D>>`
- **描述**: For each material ID, the source texture that will be resampled in that material's region

### UVLayer
- **类型**: `FString`
- **描述**: UV channel to use for the source mesh texture

### AllSourceTextures
- **类型**: `TArray<TObjectPtr<UTexture2D>>`
- **描述**: The set of all source textures from all input materials
