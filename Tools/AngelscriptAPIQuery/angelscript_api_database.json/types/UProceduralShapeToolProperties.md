# UProceduralShapeToolProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### PolygroupMode
- **类型**: `EMakeMeshPolygroupMode`
- **描述**: How PolyGroups are assigned to shape primitives.

### TargetSurface
- **类型**: `EMakeMeshPlacementType`
- **描述**: How the shape is placed in the scene.

### PivotLocation
- **类型**: `EMakeMeshPivotLocation`
- **描述**: Location of pivot within the shape

### Rotation
- **类型**: `float32`
- **描述**: Rotation of the shape around its up axis

### bAlignToNormal
- **类型**: `bool`
- **描述**: If true, aligns the shape along the normal of the surface it is placed on.

### bShowGizmo
- **类型**: `bool`
- **描述**: Show a gizmo to allow the mesh to be repositioned after the initial placement click.
