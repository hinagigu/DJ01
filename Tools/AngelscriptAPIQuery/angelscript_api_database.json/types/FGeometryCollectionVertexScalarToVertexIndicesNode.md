# FGeometryCollectionVertexScalarToVertexIndicesNode

Convert an vertex float array to a list of indices

## 属性

### VertexAttributeName
- **类型**: `FString`
- **描述**: The name of the vertex attribute to generate indices from.

### SelectionThreshold
- **类型**: `float32`
- **描述**: The value threshold for what is included in the vertex list.

### bActive
- **类型**: `bool`
