# FRigUnit_GetControlVector2D

GetControlVector2D is used to retrieve a single Vector2D from a hierarchy.

## 属性

### Control
- **类型**: `FName`

### Vector
- **类型**: `FVector2D`
- **描述**: The current value of the control.

### Minimum
- **类型**: `FVector2D`
- **描述**: The minimum value of the control.

### Maximum
- **类型**: `FVector2D`
- **描述**: The maximum value of the control.
