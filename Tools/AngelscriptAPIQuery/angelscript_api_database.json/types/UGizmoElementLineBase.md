# UGizmoElementLineBase

**继承自**: `UGizmoElementBase`

Base class for 2d and 3d primitive objects which support line drawing,
intended to be used as part of 3D Gizmos.

## 属性

### HoverLineThicknessMultiplier
- **类型**: `float32`
- **描述**: Multiplier applied to line thickness when hovering

### InteractLineThicknessMultiplier
- **类型**: `float32`
- **描述**: Multiplier applied to line thickness when interacting
