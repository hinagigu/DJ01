# UInternationalizationExportSettings

**继承自**: `UObject`

## 属性

### CulturesToGenerate
- **类型**: `TArray<FString>`
- **描述**: Which cultures should be exported

### CommandletClass
- **类型**: `FString`
- **描述**: The commandlet to run

### SourcePath
- **类型**: `FString`
- **描述**: Source for the localization data

### DestinationPath
- **类型**: `FString`
- **描述**: Destination for the localization data

### PortableObjectName
- **类型**: `FString`
- **描述**: Filename for the Portable Object format file

### ManifestName
- **类型**: `FString`
- **描述**: Name of the manifest file

### ArchiveName
- **类型**: `FString`
- **描述**: Name of the archive file

### bExportLoc
- **类型**: `bool`
- **描述**: Whether or not to export localization data

### bImportLoc
- **类型**: `bool`
- **描述**: Whether or not to import localization data

### bUseCultureDirectory
- **类型**: `bool`
- **描述**: Whether or not to use culture path
