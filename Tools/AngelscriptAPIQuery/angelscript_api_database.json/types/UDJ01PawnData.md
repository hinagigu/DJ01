# UDJ01PawnData

**继承自**: `UPrimaryDataAsset`

UDJ01PawnData

定义 Pawn 属性的不可变数据资产
仿照 Lyra 的 PawnData 系统，用于配置角色的各种属性

## 属性

### PawnClass
- **类型**: `TSubclassOf<APawn>`

### AbilitySets
- **类型**: `TArray<TObjectPtr<UDJ01AbilitySet>>`

### InputConfig
- **类型**: `UDJ01InputConfig`

### DefaultCameraMode
- **类型**: `TSubclassOf<UDJ01CameraMode>`
