# UDJ01AbilitySet

**继承自**: `UPrimaryDataAsset`

UDJ01AbilitySet

     Non-mutable data asset used to grant gameplay abilities and gameplay effects.

## 属性

### GrantedGameplayAbilities
- **类型**: `TArray<FDJ01AbilitySet_GameplayAbility>`
- **描述**: Gameplay abilities to grant when this ability set is granted.

### GrantedGameplayEffects
- **类型**: `TArray<FDJ01AbilitySet_GameplayEffect>`
- **描述**: Gameplay effects to grant when this ability set is granted.

### GrantedAttributes
- **类型**: `TArray<FDJ01AbilitySet_AttributeSet>`
- **描述**: Attribute sets to grant when this ability set is granted.
