# USmoothFillBrushOpProps

**继承自**: `UBaseSmoothBrushOpProps`

## 属性

### Strength
- **类型**: `float32`
- **描述**: Strength of the Brush

### Falloff
- **类型**: `float32`
- **描述**: Amount of falloff to apply (0.0 - 1.0)

### bPreserveUVFlow
- **类型**: `bool`
- **描述**: If true, try to preserve the shape of the UV/3D mapping. This will limit Smoothing and Remeshing in some cases.
