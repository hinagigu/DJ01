# UGameplayTagsDeveloperSettings

**继承自**: `UDeveloperSettings`

## 属性

### DeveloperConfigName
- **类型**: `FString`
- **描述**: Allows new tags to be saved into their own INI file. This is make merging easier for non technical developers by setting up their own ini file.

### FavoriteTagSource
- **类型**: `FName`
- **描述**: Stores the favorite tag source, used as the default ini when adding new tags, can be toggled on/off using the button next to the tag source picker
