# UVolumeTexture

**继承自**: `UTexture`

## 属性

### Source2DTexture
- **类型**: `UTexture2D`
- **描述**: A (optional) reference texture from which the volume texture was built

### Source2DTileSizeX
- **类型**: `int`
- **描述**: The reference texture tile size X

### Source2DTileSizeY
- **类型**: `int`
- **描述**: The reference texture tile size Y

### AddressMode
- **类型**: `TextureAddress`
