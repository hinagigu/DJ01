# FBuildPromotionNewProjectSettings

Holds settings for the new project stage of the build promotion test

## 属性

### NewProjectFolderOverride
- **类型**: `FDirectoryPath`
- **描述**: The path for the new project

### NewProjectNameOverride
- **类型**: `FString`
- **描述**: The name of the project *
