# UPropertyViewBase

**继承自**: `UWidget`

Base of property view allows you to display the value of an object properties.

## 属性

### bAutoLoadAsset
- **类型**: `bool`
- **描述**: Load the object (if it's an asset) when the widget is created.

### OnPropertyChanged
- **类型**: `FOnPropertyValueChanged`
