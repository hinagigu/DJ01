# UMeshElementsVisualizerProperties

**继承自**: `UInteractiveToolPropertySet`

Visualization settings UMeshElementsVisualizer

## 属性

### bVisible
- **类型**: `bool`
- **描述**: Should any be mesh elements be shown

### bShowWireframe
- **类型**: `bool`
- **描述**: Should mesh wireframe be shown

### bShowBorders
- **类型**: `bool`
- **描述**: Should mesh boundary edges be shown

### bShowUVSeams
- **类型**: `bool`
- **描述**: Should mesh uv seam edges be shown

### bShowNormalSeams
- **类型**: `bool`
- **描述**: Should mesh normal seam edges be shown

### bShowTangentSeams
- **类型**: `bool`
- **描述**: Should mesh tangent seam edges be shown

### bShowColorSeams
- **类型**: `bool`
- **描述**: Should mesh color seam edges be shown

### ThicknessScale
- **类型**: `float32`
- **描述**: multiplier on edge thicknesses

### WireframeColor
- **类型**: `FColor`
- **描述**: Color of mesh wireframe

### BoundaryEdgeColor
- **类型**: `FColor`
- **描述**: Color of mesh boundary edges

### UVSeamColor
- **类型**: `FColor`
- **描述**: Color of mesh UV seam edges

### NormalSeamColor
- **类型**: `FColor`
- **描述**: Color of mesh normal seam edges

### TangentSeamColor
- **类型**: `FColor`
- **描述**: Color of mesh tangent seam edges

### ColorSeamColor
- **类型**: `FColor`
- **描述**: Color of mesh color seam edges

### DepthBias
- **类型**: `float32`
- **描述**: depth bias used to slightly shift depth of lines
