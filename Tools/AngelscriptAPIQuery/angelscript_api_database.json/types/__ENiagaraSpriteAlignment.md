# __ENiagaraSpriteAlignment

This enum decides how a sprite particle will orient its "up" axis. Must keep these in sync with NiagaraSpriteVertexFactory.ush

## 属性

### Unaligned
- **类型**: `ENiagaraSpriteAlignment`

### VelocityAligned
- **类型**: `ENiagaraSpriteAlignment`

### CustomAlignment
- **类型**: `ENiagaraSpriteAlignment`

### Automatic
- **类型**: `ENiagaraSpriteAlignment`

### ENiagaraSpriteAlignment_MAX
- **类型**: `ENiagaraSpriteAlignment`
