# UVertexColorBaseBrushOpProps

**继承自**: `UMeshSculptBrushOpProps`

## 属性

### Strength
- **类型**: `float32`
- **描述**: Strength of the Brush

### Falloff
- **类型**: `float32`
- **描述**: Amount of falloff to apply

### BlendMode
- **类型**: `EVertexColorPaintBrushOpBlendMode`

### bApplyFalloff
- **类型**: `bool`
- **描述**: If bApplyFalloff is disabled, 1.0 is used as "falloff" for all vertices
