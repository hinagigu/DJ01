# UProceduralTorusToolProperties

**继承自**: `UProceduralShapeToolProperties`

## 属性

### MajorRadius
- **类型**: `float32`
- **描述**: Major radius of the torus, measured from the torus center to the center of the torus tube

### MinorRadius
- **类型**: `float32`
- **描述**: Minor radius of the torus, measured from the center ot the torus tube to the tube surface

### MajorSlices
- **类型**: `int`
- **描述**: Number of radial slices along the torus tube

### MinorSlices
- **类型**: `int`
- **描述**: Number of radial slices around the torus tube
