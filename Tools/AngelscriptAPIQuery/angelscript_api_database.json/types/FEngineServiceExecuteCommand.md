# FEngineServiceExecuteCommand

Implements a message for executing a console command.

## 属性

### Command
- **类型**: `FString`
- **描述**: Holds the command to execute.

### UserName
- **类型**: `FString`
- **描述**: Holds the name of the user that wants to execute the command.
