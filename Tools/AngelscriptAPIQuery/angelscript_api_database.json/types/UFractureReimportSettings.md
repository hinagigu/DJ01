# UFractureReimportSettings

**继承自**: `UFractureToolSettings`

Settings related to geometry collection -> static mesh conversion *

## 属性

### bOddMaterialsAreInternal
- **类型**: `bool`
- **描述**: If checked, materials will be updated on export to ensure that odd-numbered material IDs correspond to internal faces, and this will be used on re-import to assign correct material IDs
