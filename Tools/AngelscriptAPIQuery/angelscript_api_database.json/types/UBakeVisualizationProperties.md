# UBakeVisualizationProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### bPreviewAsMaterial
- **类型**: `bool`
- **描述**: Preview the texture as an input to the respective material parameter (ex. Normal, Ambient Occlusion)

### Brightness
- **类型**: `float32`
- **描述**: Adjust the brightness of the preview material; does not affect results stored in textures

### AOMultiplier
- **类型**: `float32`
- **描述**: Ambient Occlusion multiplier in the viewport; does not affect results stored in textures
