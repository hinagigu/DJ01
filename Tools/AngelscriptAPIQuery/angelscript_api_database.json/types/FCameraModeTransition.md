# FCameraModeTransition

A camera transition.

## 属性

### Conditions
- **类型**: `TArray<TObjectPtr<UCameraModeTransitionCondition>>`
- **描述**: The list of conditions that must pass for this transition to be used.

### Blend
- **类型**: `UBlendCameraNode`
- **描述**: The blend to use to blend a given camera mode in or out.
