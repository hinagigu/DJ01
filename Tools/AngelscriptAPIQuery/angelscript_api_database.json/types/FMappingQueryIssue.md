# FMappingQueryIssue

Potential issue raised with a mapping request

## 属性

### Issue
- **类型**: `EMappingQueryIssue`

### BlockingContext
- **类型**: `const UInputMappingContext`
- **描述**: Input context that contains a blocking action bound to the queried key

### BlockingAction
- **类型**: `const UInputAction`
- **描述**: Action within the input context that caused the blockage
