# UBakeMultiMeshAttributeMapsToolProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### MapTypes
- **类型**: `int`
- **描述**: The map types to generate

### MapPreview
- **类型**: `FString`
- **描述**: The map type index to preview

### Resolution
- **类型**: `EBakeTextureResolution`
- **描述**: The pixel resolution of the generated map

### BitDepth
- **类型**: `EBakeTextureBitDepth`
- **描述**: The channel bit depth of the source data for the generated textures

### SamplesPerPixel
- **类型**: `EBakeTextureSamplesPerPixel`
- **描述**: Number of samples per pixel

### SampleFilterMask
- **类型**: `UTexture2D`
- **描述**: Mask texture for filtering out samples/pixels from the output texture
