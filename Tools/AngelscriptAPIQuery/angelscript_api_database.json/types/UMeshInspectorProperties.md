# UMeshInspectorProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### bWireframe
- **类型**: `bool`
- **描述**: Toggle visibility of all mesh edges

### bBoundaryEdges
- **类型**: `bool`
- **描述**: Toggle visibility of open boundary edges

### bBowtieVertices
- **类型**: `bool`
- **描述**: Toggle visibility of bowtie vertices

### bPolygonBorders
- **类型**: `bool`
- **描述**: Toggle visibility of polygon borders

### bUVSeams
- **类型**: `bool`
- **描述**: Toggle visibility of UV seam edges

### bUVBowties
- **类型**: `bool`
- **描述**: Toggle visibility of UV bowtie vertices

### bMissingUVs
- **类型**: `bool`
- **描述**: Toggle visibility of triangles with missing UVs

### bNormalSeams
- **类型**: `bool`
- **描述**: Toggle visibility of Normal seam edges

### bTangentSeams
- **类型**: `bool`
- **描述**: Toggle visibility of Tangent seam edges

### bNormalVectors
- **类型**: `bool`
- **描述**: Toggle visibility of normal vectors

### bTangentVectors
- **类型**: `bool`
- **描述**: Toggle visibility of tangent vectors

### bDrawHiddenEdgesAndSeams
- **类型**: `bool`
- **描述**: Toggle visibility of occluded boundary edges and seams

### NormalLength
- **类型**: `float32`
- **描述**: Length of line segments representing normal vectors

### TangentLength
- **类型**: `float32`
- **描述**: Length of line segments representing tangent vectors

### ShowIndices
- **类型**: `EMeshInspectorToolDrawIndexMode`
- **描述**: Draw the mesh indices of the selected type. A maximum of 500 visible indices will be rendered.
