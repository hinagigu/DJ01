# UFacialAnimationBulkImporterSettings

**继承自**: `UObject`

## 属性

### SourceImportPath
- **类型**: `FDirectoryPath`
- **描述**: The path to import files from

### TargetImportPath
- **类型**: `FDirectoryPath`
- **描述**: The path to import files to

### CurveNodeName
- **类型**: `FString`
- **描述**: Node in the FBX scene that contains the curves we are interested in
