# FPoseDriverTransform

Translation and rotation for a particular bone at a particular target

## 属性

### TargetTranslation
- **类型**: `FVector`
- **描述**: Translation of this target

### TargetRotation
- **类型**: `FRotator`
- **描述**: Rotation of this target
