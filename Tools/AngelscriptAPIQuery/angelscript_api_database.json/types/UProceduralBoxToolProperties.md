# UProceduralBoxToolProperties

## 属性

### Width
- **类型**: `float32`
- **描述**: Width of the box

### Depth
- **类型**: `float32`
- **描述**: Depth of the box

### Height
- **类型**: `float32`
- **描述**: Height of the box

### WidthSubdivisions
- **类型**: `int`
- **描述**: Number of subdivisions along the width

### DepthSubdivisions
- **类型**: `int`
- **描述**: Number of subdivisions along the depth

### HeightSubdivisions
- **类型**: `int`
- **描述**: Number of subdivisions along the height
