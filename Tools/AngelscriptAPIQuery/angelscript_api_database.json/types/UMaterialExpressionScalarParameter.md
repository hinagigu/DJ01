# UMaterialExpressionScalarParameter

**继承自**: `UMaterialExpressionParameter`

## 属性

### DefaultValue
- **类型**: `float32`

### bUseCustomPrimitiveData
- **类型**: `bool`

### PrimitiveDataIndex
- **类型**: `uint8`

### SliderMin
- **类型**: `float32`
- **描述**: Sets the lower bound for the slider on this parameter in the material instance editor.

### SliderMax
- **类型**: `float32`
- **描述**: Sets the upper bound for the slider on this parameter in the material instance editor.
The slider will be disabled if SliderMax <= SliderMin.
