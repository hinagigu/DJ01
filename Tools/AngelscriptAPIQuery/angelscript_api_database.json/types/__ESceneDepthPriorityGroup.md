# __ESceneDepthPriorityGroup

A priority for sorting scene elements by depth.
Elements with higher priority occlude elements with lower priority, disregarding distance.

## 属性

### SDPG_World
- **类型**: `ESceneDepthPriorityGroup`

### SDPG_Foreground
- **类型**: `ESceneDepthPriorityGroup`

### SDPG_MAX
- **类型**: `ESceneDepthPriorityGroup`
