# UStackBoxSlot

**继承自**: `UPanelSlot`

The Slot for the UStackBox, contains the widget that is flowed vertically or horizontally.

## 属性

### Padding
- **类型**: `FMargin`

### Size
- **类型**: `FSlateChildSize`

### HorizontalAlignment
- **类型**: `EHorizontalAlignment`

### VerticalAlignment
- **类型**: `EVerticalAlignment`
