# UUVEditorDistortionVisualizationProperties

**继承自**: `UInteractiveToolPropertySet`

Visualization settings for the UUVEditorDistortionVisualization

## 属性

### bVisible
- **类型**: `bool`
- **描述**: Should the visualization be shown.

### Metric
- **类型**: `EDistortionMetric`
- **描述**: The specific metric used to classify and visualize distortion.

### bRespectUDIMTextureResolutions
- **类型**: `bool`
- **描述**: When computing average texel density, take into account specific UDIM tile resolutions for a more accurate representation
