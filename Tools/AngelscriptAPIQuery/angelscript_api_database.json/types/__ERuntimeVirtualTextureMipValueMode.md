# __ERuntimeVirtualTextureMipValueMode

Set how Mip levels are calculated.
Internally we will convert to ETextureMipValueMode which is used by internal APIs.

## 属性

### RVTMVM_None
- **类型**: `ERuntimeVirtualTextureMipValueMode`

### RVTMVM_MipLevel
- **类型**: `ERuntimeVirtualTextureMipValueMode`

### RVTMVM_MipBias
- **类型**: `ERuntimeVirtualTextureMipValueMode`

### RVTMVM_RecalculateDerivatives
- **类型**: `ERuntimeVirtualTextureMipValueMode`

### RVTMVM_MAX
- **类型**: `ERuntimeVirtualTextureMipValueMode`
