# FFractureModeCustomSectionColor

Defines a color to be used for a particular Tool Palette Section

## 属性

### SectionName
- **类型**: `FString`
- **描述**: Name of Section in Fracture Mode Tool Palette

### Color
- **类型**: `FLinearColor`
- **描述**: Custom Header Color
