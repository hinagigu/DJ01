# FEditorDialogLibraryObjectDetailsViewOptions

## 属性

### bShowObjectName
- **类型**: `bool`

### bAllowSearch
- **类型**: `bool`

### bAllowResizing
- **类型**: `bool`

### MinWidth
- **类型**: `int`
- **描述**: The minimum dialog width. If zero, default to the medium window width defined by the Editor style.

### MinHeight
- **类型**: `int`
- **描述**: The minimum dialog height. If zero, default to the medium window height defined by the Editor style.

### ValueColumnWidthRatio
- **类型**: `float32`
- **描述**: The column size proportion, between 0 and 1, used to display the property value. The remaining of the horizontal space will be used to display the property name.
