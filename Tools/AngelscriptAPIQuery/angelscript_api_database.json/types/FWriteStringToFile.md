# FWriteStringToFile

Write a string to a file

## 属性

### FilePath
- **类型**: `FString`
- **描述**: Where file should be written on disk

### FileContents
- **类型**: `FString`
- **描述**: Contents of the file to write

### bActive
- **类型**: `bool`

## 方法

### opAssign
```angelscript
FWriteStringToFile& opAssign(FWriteStringToFile Other)
```
