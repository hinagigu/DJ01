# FLaunchOnTestSettings

Holds settings for the editor Launch On With Map Iterations test.

## 属性

### LaunchOnTestmap
- **类型**: `FFilePath`
- **描述**: Map to be used for the Launch On test *

### DeviceID
- **类型**: `FString`
- **描述**: This is the device to be used for launch on. Example: WindowsClient, Android, IOS, Linux

## 方法

### opAssign
```angelscript
FLaunchOnTestSettings& opAssign(FLaunchOnTestSettings Other)
```
