# FGameModeName

Helper structure, used to associate GameModes with shortcut names.

## 属性

### Name
- **类型**: `FString`
- **描述**: Abbreviation/prefix that can be used as an alias for the class name

### GameMode
- **类型**: `FSoftClassPath`
- **描述**: GameMode class to load

## 方法

### opAssign
```angelscript
FGameModeName& opAssign(FGameModeName Other)
```
