# FGameplayTagResponseTableEntry

## 属性

### Positive
- **类型**: `FGameplayTagReponsePair`
- **描述**: Tags that count as "positive" toward to final response count. If the overall count is positive, this ResponseGameplayEffect is applied.

### Negative
- **类型**: `FGameplayTagReponsePair`
- **描述**: Tags that count as "negative" toward to final response count. If the overall count is negative, this ResponseGameplayEffect is applied.
