# UNavArea

**继承自**: `UNavAreaBase`

Class containing definition of a navigation area

## 属性

### DefaultCost
- **类型**: `float32`
- **描述**: travel cost multiplier for path distance

### FixedAreaEnteringCost
- **类型**: `float32`
- **描述**: entering cost

### DrawColor
- **类型**: `FColor`
- **描述**: area color in navigation view

### SupportedAgents
- **类型**: `FNavAgentSelector`
- **描述**: restrict area only to specified agents
