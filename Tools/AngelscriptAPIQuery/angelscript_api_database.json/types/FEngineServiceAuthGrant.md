# FEngineServiceAuthGrant

Implements a message for granting service access to a remote user.

## 属性

### UserName
- **类型**: `FString`
- **描述**: Holds the name of the user that granted access.

### UserToGrant
- **类型**: `FString`
- **描述**: Holds the name of the user that access is granted to.
