# UTranslationUnit

**继承自**: `UObject`

## 属性

### Translation
- **类型**: `FString`
- **描述**: Translations

### HasBeenReviewed
- **类型**: `bool`
- **描述**: Whether the changes have been reviewed

### TranslationBeforeImport
- **类型**: `FString`
- **描述**: If this Translation Unit had a different translation before import, it will be stored here

### LocresPath
- **类型**: `FString`
- **描述**: Optional: Which Locres File this translation is in
