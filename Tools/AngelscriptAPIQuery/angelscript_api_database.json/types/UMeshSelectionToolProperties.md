# UMeshSelectionToolProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### SelectionMode
- **类型**: `EMeshSelectionToolPrimaryMode`
- **描述**: The Selection Mode defines the behavior of the selection brush

### AngleTolerance
- **类型**: `float32`
- **描述**: Angle in Degrees used for Angle-based Selection Modes

### bHitBackFaces
- **类型**: `bool`
- **描述**: Allow the brush to hit back-facing parts of the surface

### bShowPoints
- **类型**: `bool`
- **描述**: Toggle drawing of highlight points on/off

### FaceColorMode
- **类型**: `EMeshFacesColorMode`
- **描述**: Color each triangle based on the selected mesh attribute
