# UBlueprintHeaderViewSettings

**继承自**: `UDeveloperSettings`

Settings for the Blueprint Header View Plugin

## 属性

### SyntaxColors
- **类型**: `FHeaderViewSyntaxColors`
- **描述**: Syntax Highlighting colors for Blueprint Header View output

### SelectionColor
- **类型**: `FLinearColor`
- **描述**: Highlight color for selected items in the Blueprint Header View output

### FontSize
- **类型**: `int`
- **描述**: Font Size for the Blueprint Header View output

### SortMethod
- **类型**: `EHeaderViewSortMethod`
- **描述**: Sorting Method for Header View Functions and Properties
