# UCylinderBuilder

**继承自**: `UEditorBrushBuilder`

## 属性

### Z
- **类型**: `float32`
- **描述**: Distance from base to tip of cylinder

### OuterRadius
- **类型**: `float32`
- **描述**: Radius of cylinder

### InnerRadius
- **类型**: `float32`
- **描述**: Radius of inner cylinder (when hollow)

### Sides
- **类型**: `int`
- **描述**: How many sides this cylinder should have

### AlignToSide
- **类型**: `bool`

### Hollow
- **类型**: `bool`
