# UPaperExtractSpriteGridSettings

**继承自**: `UObject`

## 属性

### CellWidth
- **类型**: `int`
- **描述**: The width of each sprite in grid mode

### CellHeight
- **类型**: `int`
- **描述**: The height of each sprite in grid mode

### NumCellsX
- **类型**: `int`
- **描述**: Number of cells extracted horizontally. Can be used to limit the number of sprites extracted. Set to 0 to extract all sprites

### NumCellsY
- **类型**: `int`
- **描述**: Number of cells extracted vertically. Can be used to limit the number of sprites extracted. Set to 0 to extract all sprites

### MarginX
- **类型**: `int`
- **描述**: Margin from the left of the texture to the first sprite

### MarginY
- **类型**: `int`
- **描述**: Margin from the top of the texture to the first sprite

### SpacingX
- **类型**: `int`
- **描述**: Horizontal spacing between sprites

### SpacingY
- **类型**: `int`
- **描述**: Vertical spacing between sprites
