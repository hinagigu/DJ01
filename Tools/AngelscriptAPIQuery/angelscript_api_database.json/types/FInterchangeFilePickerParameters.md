# FInterchangeFilePickerParameters

## 属性

### bAllowMultipleFiles
- **类型**: `bool`
- **描述**: If true, the user will be able to select multiple files.

### Title
- **类型**: `FText`
- **描述**: If not empty, this will override the default title.

### DefaultPath
- **类型**: `FString`
- **描述**: Set the default open path that the dialog will show to the user.
