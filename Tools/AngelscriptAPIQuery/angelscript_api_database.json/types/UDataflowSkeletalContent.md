# UDataflowSkeletalContent

**继承自**: `UDataflowBaseContent`

Dataflow content owning dataflow and skelmesh assets that that will be used to evaluate the graph

## 属性

### SkeletalMesh
- **类型**: `USkeletalMesh`
- **描述**: Data flow skeletal mesh

### AnimationAsset
- **类型**: `UAnimationAsset`
- **描述**: Animation asset to be used to preview simulation

### Skeleton
- **类型**: `USkeleton`
- **描述**: Data flow skeleton
