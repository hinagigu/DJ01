# FInterpCurveQuat

Describes an entire curve that is used to compute a quaternion output value from a float input.
@note This is a mirror of TInterpCurve<FQuat>, defined in InterpCurve.h

## 属性

### Points
- **类型**: `TArray<FInterpCurvePointQuat>`

### bIsLooped
- **类型**: `bool`

### LoopKeyOffset
