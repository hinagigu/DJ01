# UConeBuilder

**继承自**: `UEditorBrushBuilder`

## 属性

### Z
- **类型**: `float32`
- **描述**: Distance from base to tip of cone

### CapZ
- **类型**: `float32`
- **描述**: Distance from base to the tip of inner cone (when hollow)

### OuterRadius
- **类型**: `float32`
- **描述**: Radius of cone

### InnerRadius
- **类型**: `float32`
- **描述**: Radius of inner cone (when hollow)

### Sides
- **类型**: `int`
- **描述**: How many sides this cone should have

### AlignToSide
- **类型**: `bool`

### Hollow
- **类型**: `bool`
