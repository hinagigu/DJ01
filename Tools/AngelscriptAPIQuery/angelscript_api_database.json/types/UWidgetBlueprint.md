# UWidgetBlueprint

**继承自**: `UBaseWidgetBlueprint`

The widget blueprint enables extending UUserWidget the user extensible UWidget.

## 属性

### bCanCallInitializedWithoutPlayerContext
- **类型**: `bool`
- **描述**: Run the initialize event on widget that doesn't have a player context.

### ThumbnailSizeMode
- **类型**: `EThumbnailPreviewSizeMode`

### ThumbnailCustomSize
- **类型**: `FVector2D`

### ThumbnailImage
- **类型**: `UTexture2D`
