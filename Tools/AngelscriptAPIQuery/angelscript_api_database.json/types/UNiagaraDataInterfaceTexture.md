# UNiagaraDataInterfaceTexture

**继承自**: `UNiagaraDataInterface`

Data Interface allowing sampling of a texture

## 属性

### Texture
- **类型**: `UTexture`

### TextureUserParameter
- **类型**: `FNiagaraUserParameterBinding`
- **描述**: When valid the user parameter is used as the texture rather than the one on the data interface
