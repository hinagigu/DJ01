# FEditorImportWorkflowDefinition

Holds settings for the asset import workflow test

## 属性

### ImportFilePath
- **类型**: `FFilePath`
- **描述**: The file to import

### FactorySettings
- **类型**: `TArray<FImportFactorySettingValues>`
- **描述**: Settings for the import factory
