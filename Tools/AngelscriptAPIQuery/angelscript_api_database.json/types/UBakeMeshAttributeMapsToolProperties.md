# UBakeMeshAttributeMapsToolProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### MapTypes
- **类型**: `int`
- **描述**: The bake output types to generate

### MapPreview
- **类型**: `FString`
- **描述**: The baked output type used for preview in the viewport

### Resolution
- **类型**: `EBakeTextureResolution`
- **描述**: The pixel resolution of the generated textures

### BitDepth
- **类型**: `EBakeTextureBitDepth`
- **描述**: The bit depth for each channel of the generated textures

### SamplesPerPixel
- **类型**: `EBakeTextureSamplesPerPixel`
- **描述**: Number of samples per pixel

### SampleFilterMask
- **类型**: `UTexture2D`
- **描述**: Mask texture for filtering out samples/pixels from the output texture
