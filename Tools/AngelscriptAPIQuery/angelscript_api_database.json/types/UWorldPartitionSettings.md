# UWorldPartitionSettings

**继承自**: `UDeveloperSettings`

## 属性

### NewMapsDataLayersLogicOperator
- **类型**: `EWorldPartitionDataLayersLogicOperator`
- **描述**: Set the default logical operator for actor data layers activation for new maps

### bNewMapsEnableWorldPartition
- **类型**: `bool`
- **描述**: Set the default to whether enable world partition for new maps created in the content broswer

### bNewMapsEnableWorldPartitionStreaming
- **类型**: `bool`
- **描述**: Set the default to whether enable world partition streaming for new maps created in the content broswer

### RuntimeHashDefaultClass
- **类型**: `TSubclassOf<UWorldPartitionRuntimeHash>`
- **描述**: Set the default runtime hash class to use for new maps
