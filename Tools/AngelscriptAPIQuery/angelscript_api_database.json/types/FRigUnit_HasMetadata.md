# FRigUnit_HasMetadata

Returns true if a given item in the hierarchy has a specific set of metadata

## 属性

### Item
- **类型**: `FRigElementKey`

### Name
- **类型**: `FName`

### Type
- **类型**: `ERigMetadataType`

### NameSpace
- **类型**: `ERigMetaDataNameSpace`

### Found
- **类型**: `bool`
- **描述**: True if the item has the metadata

## 方法

### opAssign
```angelscript
FRigUnit_HasMetadata& opAssign(FRigUnit_HasMetadata Other)
```
