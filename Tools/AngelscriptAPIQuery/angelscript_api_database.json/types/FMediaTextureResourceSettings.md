# FMediaTextureResourceSettings

This struct is used to expose Media Texture settings via Media Plate Component and is a mirror of some
of the settings.

## 属性

### bEnableGenMips
- **类型**: `bool`
- **描述**: Enable mips generation

### CurrentNumMips
- **类型**: `uint8`
- **描述**: Current number of mips to be generated as output
