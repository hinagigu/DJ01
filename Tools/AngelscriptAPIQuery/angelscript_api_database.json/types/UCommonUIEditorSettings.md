### TemplateButtonStyle
- **类型**: `TSoftClassPtr<UCommonButtonStyle>`
- **描述**: Newly created CommonButton Widgets will use this style.

### TemplateBorderStyle
- **类型**: `TSoftClassPtr<UCommonBorderStyle>`
- **描述**: Newly created CommonBorder Widgets will use this style.
