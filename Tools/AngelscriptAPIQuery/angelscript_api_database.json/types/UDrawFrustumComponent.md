# UDrawFrustumComponent

**继承自**: `UPrimitiveComponent`

Utility component for drawing a view frustum. Origin is at the component location, frustum points down position X axis.

## 属性

### bFrustumEnabled
- **类型**: `bool`

### FrustumColor
- **类型**: `FColor`

### FrustumAngle
- **类型**: `float32`

### FrustumAspectRatio
- **类型**: `float32`

### FrustumStartDist
- **类型**: `float32`

### FrustumEndDist
- **类型**: `float32`

### Texture
- **类型**: `UTexture`
