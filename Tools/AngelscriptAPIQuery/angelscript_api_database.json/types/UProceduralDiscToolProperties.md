# UProceduralDiscToolProperties

**继承自**: `UProceduralShapeToolProperties`

## 属性

### DiscType
- **类型**: `EProceduralDiscType`
- **描述**: Type of disc

### Radius
- **类型**: `float32`
- **描述**: Radius of the disc

### RadialSlices
- **类型**: `int`
- **描述**: Number of radial slices for the disc

### RadialSubdivisions
- **类型**: `int`
- **描述**: Number of radial subdivisions for each radial slice of the disc

### HoleRadius
- **类型**: `float32`
- **描述**: Radius of the hole in the disc. This is only available for Punctured Discs.
