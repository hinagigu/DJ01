# UMeshDrawCommandStatsSettings

**继承自**: `UDeveloperSettings`

User settings used by the MeshDrawCommandStats system.

## 属性

### Budgets
- **类型**: `TArray<FMeshDrawCommandStatsBudget>`
- **描述**: Budgets used by r.MeshDrawCommands.Stats

### BudgetTotals
- **类型**: `TArray<FMeshDrawCommandStatsBudgetTotals>`
- **描述**: The total primitive budget for a collection.

### CollectionForCsvProfiler
- **类型**: `int`
- **描述**: Which collection to export to CSV
