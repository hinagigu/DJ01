# UAnimSequenceBase

**继承自**: `UAnimationAsset`

## 属性

### SequenceLength
- **类型**: `float32`

### RateScale
- **类型**: `float32`
- **描述**: Number for tweaking playback rate of this animation globally.

### bLoop
- **类型**: `bool`
- **描述**: The default looping behavior of this animation.
Asset players can override this

### DataModel
- **类型**: `UAnimDataModel`
