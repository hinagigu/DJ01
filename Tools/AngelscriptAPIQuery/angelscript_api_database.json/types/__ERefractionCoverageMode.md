# __ERefractionCoverageMode

Determines how the refraction account for the coverage with Substrate. It can only be used when Substrate is enabled.

## 属性

### RCM_CoverageIgnored
- **类型**: `ERefractionCoverageMode`

### RCM_CoverageAccountedFor
- **类型**: `ERefractionCoverageMode`

### RCM_MAX
- **类型**: `ERefractionCoverageMode`
