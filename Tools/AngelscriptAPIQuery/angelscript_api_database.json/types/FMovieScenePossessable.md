# FMovieScenePossessable

MovieScenePossessable is a "typed slot" used to allow the MovieScene to control an already-existing object

## 属性

### Tags
- **类型**: `TArray<FName>`
- **描述**: Array of tags that can be used for grouping and categorizing.

### DynamicBinding
- **类型**: `FMovieSceneDynamicBinding`
- **描述**: Optional user-defined possessable lookup information
