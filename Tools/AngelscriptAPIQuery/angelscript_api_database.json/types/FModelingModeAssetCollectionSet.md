# FModelingModeAssetCollectionSet

Defines a Named list/set of content-browser Collection names

## 属性

### Name
- **类型**: `FString`
- **描述**: Name of the set of collections

### Collections
- **类型**: `TArray<FCollectionReference>`
- **描述**: List of Collection names
