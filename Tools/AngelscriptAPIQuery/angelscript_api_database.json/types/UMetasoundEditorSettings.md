# UMetasoundEditorSettings

**继承自**: `UObject`

## 属性

### bPinMetaSoundPatchInAssetMenu
- **类型**: `bool`
- **描述**: Whether to pin the MetaSound Patch asset type when creating new assets.

### bPinMetaSoundSourceInAssetMenu
- **类型**: `bool`
- **描述**: Whether to pin the MetaSound Source asset type when creating new assets.

### DefaultAuthor
- **类型**: `FString`
- **描述**: Default author title to use when authoring a new
MetaSound.  If empty, uses machine name by default.

### DefaultPinTypeColor
- **类型**: `FLinearColor`
- **描述**: Default pin type color

### AudioPinTypeColor
- **类型**: `FLinearColor`
- **描述**: Audio pin type color

### BooleanPinTypeColor
- **类型**: `FLinearColor`
- **描述**: Boolean pin type color

### FloatPinTypeColor
- **类型**: `FLinearColor`
- **描述**: Floating-point pin type color

### IntPinTypeColor
- **类型**: `FLinearColor`
- **描述**: Integer pin type color

### ObjectPinTypeColor
- **类型**: `FLinearColor`
- **描述**: Object pin type color

### StringPinTypeColor
- **类型**: `FLinearColor`
- **描述**: String pin type color

### TimePinTypeColor
- **类型**: `FLinearColor`
- **描述**: Time pin type color

### TriggerPinTypeColor
- **类型**: `FLinearColor`
- **描述**: Trigger pin type color

### WaveTablePinTypeColor
- **类型**: `FLinearColor`
- **描述**: WaveTable pin type color

### NativeNodeTitleColor
- **类型**: `FLinearColor`
- **描述**: Native node class title color

### AssetReferenceNodeTitleColor
- **类型**: `FLinearColor`
- **描述**: Title color for references to MetaSound assets

### InputNodeTitleColor
- **类型**: `FLinearColor`
- **描述**: Input node title color

### OutputNodeTitleColor
- **类型**: `FLinearColor`
- **描述**: Output node title color

### VariableNodeTitleColor
- **类型**: `FLinearColor`
- **描述**: Variable node title color

### DefaultInputWidgetType
- **类型**: `EMetasoundMemberDefaultWidget`
- **描述**: Widget type to show on input nodes by default

### AnalyzerAnimationSettings
- **类型**: `FMetasoundAnalyzerAnimationSettings`
- **描述**: Settings for visualizing analyzed MetaSound connections

### bShowOscilloscopeOnAudioPinMouseOver
- **类型**: `bool`
- **描述**: Whether to show a small oscilloscope when hovering audio pins.
