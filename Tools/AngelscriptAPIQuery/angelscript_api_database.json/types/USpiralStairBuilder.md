# USpiralStairBuilder

**继承自**: `UEditorBrushBuilder`

## 属性

### InnerRadius
- **类型**: `int`
- **描述**: The radius of the inner curve of the stair

### StepWidth
- **类型**: `int`
- **描述**: The width of each step

### StepHeight
- **类型**: `int`
- **描述**: The height of each step

### StepThickness
- **类型**: `int`
- **描述**: The thickness of each step

### NumStepsPer360
- **类型**: `int`
- **描述**: The number of steps in one whole spiral rotation

### NumSteps
- **类型**: `int`
- **描述**: The total number of steps

### SlopedCeiling
- **类型**: `bool`

### SlopedFloor
- **类型**: `bool`

### CounterClockwise
- **类型**: `bool`
