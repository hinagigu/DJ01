# __EClothLODBiasMode

Strategy used for storing additional cloth deformer mappings depending on the
desired use of the RaytracingMinLOD value and of the LODBias console variable.

## 属性

### MappingsToSameLOD
- **类型**: `EClothLODBiasMode`

### MappingsToMinLOD
- **类型**: `EClothLODBiasMode`

### MappingsToAnyLOD
- **类型**: `EClothLODBiasMode`

### EClothLODBiasMode_MAX
- **类型**: `EClothLODBiasMode`
