# URemoveOtherGameplayEffectComponent

**继承自**: `UGameplayEffectComponent`

Remove other Gameplay Effects based on certain conditions

## 属性

### RemoveGameplayEffectQueries
- **类型**: `TArray<FGameplayEffectQuery>`
- **描述**: On Application of the owning Gameplay Effect, any Active GameplayEffects that *match* these queries will be removed.
