# UBakeMeshAttributeVertexToolProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### OutputMode
- **类型**: `EBakeVertexOutput`
- **描述**: The bake output mode

### OutputType
- **类型**: `int`
- **描述**: The bake output type to generate

### OutputTypeR
- **类型**: `int`
- **描述**: The bake output type to generate in the Red channel

### OutputTypeG
- **类型**: `int`
- **描述**: The bake output type to generate in the Green channel

### OutputTypeB
- **类型**: `int`
- **描述**: The bake output type to generate in the Blue channel

### OutputTypeA
- **类型**: `int`
- **描述**: The bake output type to generate in the Alpha channel

### PreviewMode
- **类型**: `EBakeVertexChannel`
- **描述**: The vertex color channel to preview

### bSplitAtNormalSeams
- **类型**: `bool`
- **描述**: If true, compute a separate vertex color for each unique normal on a vertex

### bSplitAtUVSeams
- **类型**: `bool`
- **描述**: If true, Compute a separate vertex color for each unique UV on a vertex.
