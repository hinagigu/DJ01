# UMovieSceneUserThumbnailSettings

**继承自**: `UObject`

## 属性

### bDrawThumbnails
- **类型**: `bool`
- **描述**: Whether to draw thumbnails or not

### bDrawSingleThumbnails
- **类型**: `bool`
- **描述**: Whether to draw a single thumbnail for this section or as many as can fit

### ThumbnailSize
- **类型**: `FIntPoint`
- **描述**: Size at which to draw thumbnails on thumbnail sections

### Quality
- **类型**: `EThumbnailQuality`
- **描述**: Quality to render the thumbnails with
