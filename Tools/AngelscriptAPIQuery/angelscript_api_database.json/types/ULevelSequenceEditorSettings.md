# ULevelSequenceEditorSettings

**继承自**: `UObject`

Level Sequence Editor settings.

## 属性

### TrackSettings
- **类型**: `TArray<FLevelSequenceTrackSettings>`
- **描述**: Specifies class properties for which movie scene tracks will be created automatically.

### bAutoBindToPIE
- **类型**: `bool`
- **描述**: Specifies whether to automatically bind an active sequencer UI to PIE worlds.

### bAutoBindToSimulate
- **类型**: `bool`
- **描述**: Specifies whether to automatically bind an active sequencer UI to simulate worlds.
