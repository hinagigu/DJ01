# UMaterialExpressionPanner

**继承自**: `UMaterialExpression`

## 属性

### SpeedX
- **类型**: `float32`

### SpeedY
- **类型**: `float32`

### ConstCoordinate
- **类型**: `uint`
- **描述**: only used if Coordinate is not hooked up

### bFractionalPart
- **类型**: `bool`
- **描述**: Output only the fractional part of the pan calculation for greater precision.
Output is greater than or equal to 0 and less than 1.
