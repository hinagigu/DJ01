# FPrimaryAssetRulesOverride

Simple structure to allow overriding asset rules for a specific primary asset. This can be used to set chunks

## 属性

### PrimaryAssetId
- **类型**: `FPrimaryAssetId`
- **描述**: Which primary asset to override the rules for

### Rules
- **类型**: `FPrimaryAssetRules`
- **描述**: What to overrides the rules with

## 方法

### opAssign
```angelscript
FPrimaryAssetRulesOverride& opAssign(FPrimaryAssetRulesOverride Other)
```
