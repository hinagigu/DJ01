# FRigUnit_TransformConstraintPerItem

Constrains an item's transform to multiple items' transforms

## 属性

### Item
- **类型**: `FRigElementKey`

### BaseTransformSpace
- **类型**: `ETransformSpaceMode`

### BaseTransform
- **类型**: `FTransform`

### BaseItem
- **类型**: `FRigElementKey`

### Targets
- **类型**: `TArray<FConstraintTarget>`

### bUseInitialTransforms
- **类型**: `bool`
- **描述**: If checked the initial transform will be used for the constraint data

### ExecuteContext
- **类型**: `FControlRigExecuteContext`
