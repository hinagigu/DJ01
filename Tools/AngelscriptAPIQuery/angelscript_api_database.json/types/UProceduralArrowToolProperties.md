# UProceduralArrowToolProperties

**继承自**: `UProceduralShapeToolProperties`

## 属性

### ShaftRadius
- **类型**: `float32`
- **描述**: Radius of the arrow shaft

### ShaftHeight
- **类型**: `float32`
- **描述**: Height of arrow shaft

### HeadRadius
- **类型**: `float32`
- **描述**: Radius of the arrow head base

### HeadHeight
- **类型**: `float32`
- **描述**: Height of arrow head

### RadialSlices
- **类型**: `int`
- **描述**: Number of radial slices for the arrow

### HeightSubdivisions
- **类型**: `int`
- **描述**: Number of subdivisions along each part of the arrow, i.e. shaft, head base, head cone
