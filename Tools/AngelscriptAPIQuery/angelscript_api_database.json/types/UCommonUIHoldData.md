# UCommonUIHoldData

**继承自**: `UObject`

Defines values for hold behavior per input type: for mouse Press and Hold, for gamepad focused Press and Hold, for touch Press and Hold.

## 属性

### KeyboardAndMouse
- **类型**: `FInputHoldData`

### Gamepad
- **类型**: `FInputHoldData`

### Touch
- **类型**: `FInputHoldData`
