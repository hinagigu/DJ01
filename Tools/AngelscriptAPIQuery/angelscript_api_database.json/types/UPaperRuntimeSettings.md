# UPaperRuntimeSettings

**继承自**: `UObject`

Implements the settings for the Paper2D plugin.

## 属性

### bEnableSpriteAtlasGroups
- **类型**: `bool`
- **描述**: Enables experimental *incomplete and unsupported* texture atlas groups that sprites can be assigned to

### bEnableTerrainSplineEditing
- **类型**: `bool`
- **描述**: Enables experimental *incomplete and unsupported* 2D terrain spline editing. Note: You need to restart the editor when enabling this setting for the change to fully take effect.

### bResizeSpriteDataToMatchTextures
- **类型**: `bool`
- **描述**: Enables automatic resizing of various sprite data that is authored in texture space if the source texture gets resized (sockets, the pivot, render and collision geometry, etc...)
