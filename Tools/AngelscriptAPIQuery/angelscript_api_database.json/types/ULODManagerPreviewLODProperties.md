# ULODManagerPreviewLODProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### VisibleLOD
- **类型**: `FString`
- **描述**: LOD to visualise. Default option is equivalent to disabling the Tool, RenderData is the mesh used for rendering derived from the SourceModel (possibly simplified)

### bShowSeams
- **类型**: `bool`
- **描述**: Control whether mesh borders are displayed
