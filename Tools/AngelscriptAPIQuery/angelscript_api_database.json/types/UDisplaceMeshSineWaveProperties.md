# UDisplaceMeshSineWaveProperties

**继承自**: `UInteractiveToolPropertySet`

PropertySet for Sine wave displacement

## 属性

### SineWaveFrequency
- **类型**: `float32`
- **描述**: Sine wave displacement frequency

### SineWavePhaseShift
- **类型**: `float32`
- **描述**: Sine wave phase shift

### SineWaveDirection
- **类型**: `FVector`
- **描述**: Unit vector representing the direction of wave displacement.
