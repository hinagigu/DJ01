# UOnlinePIESettings

**继承自**: `UDeveloperSettings`

Setup login credentials for the Play In Editor (PIE) feature

## 属性

### bOnlinePIEEnabled
- **类型**: `bool`
- **描述**: Attempt to login with user credentials on a backend service before launching the PIE instance.

### Logins
- **类型**: `TArray<FPIELoginSettingsInternal>`
- **描述**: Login credentials, at least one for each instance of PIE that is intended to be run
