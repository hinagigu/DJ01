# FRigUnit_GetItemNameSpace

Returns the namespace of a given item. This may be an empty string if the item doesn't have a namespace.

## 属性

### Item
- **类型**: `FRigElementKey`

### HasNameSpace
- **类型**: `bool`
- **描述**: * True if the item has a valid namespace

### NameSpace
- **类型**: `FString`
- **描述**: * The namespace of the item

### opAssign
```angelscript
FRigUnit_GetItemNameSpace& opAssign(FRigUnit_GetItemNameSpace Other)
```
