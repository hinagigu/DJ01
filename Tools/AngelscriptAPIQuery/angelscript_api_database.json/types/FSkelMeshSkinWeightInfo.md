# FSkelMeshSkinWeightInfo

Vertex skin weight info supplied for a component override.

## 属性

### Bones
- **类型**: `int`
- **描述**: Index of bones that influence this vertex

### Weights
- **类型**: `uint8`
- **描述**: Influence of each bone on this vertex
