# FCurveScaledPayload

### Factor
- **类型**: `float32`
- **描述**: Factor with which the curve was scaled

- **类型**: `float32`
- **描述**: Time used as the origin when scaling the curve

- **类型**: `FAnimationCurveIdentifier`
