# UPolyEditInsetOutsetProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### Softness
- **类型**: `float32`
- **描述**: Amount of smoothing applied to outset boundary

### bBoundaryOnly
- **类型**: `bool`
- **描述**: Controls whether outset operation will move interior vertices as well as border vertices

### AreaScale
- **类型**: `float32`
- **描述**: Tweak area scaling when solving for interior vertices

### bReproject
- **类型**: `bool`
- **描述**: When insetting, determines whether vertices in inset region should be projected back onto input surface
