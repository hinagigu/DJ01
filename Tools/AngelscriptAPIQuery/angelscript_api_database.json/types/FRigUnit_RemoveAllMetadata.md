# FRigUnit_RemoveAllMetadata

Removes an existing metadata filed from an item

## 属性

### Item
- **类型**: `FRigElementKey`

### NameSpace
- **类型**: `ERigMetaDataNameSpace`

### Removed
- **类型**: `bool`
- **描述**: True if any metadata has been removed

### ExecuteContext
- **类型**: `FControlRigExecuteContext`
