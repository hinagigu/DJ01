# UMediaPlayerRecording

**继承自**: `USequenceRecordingBase`

## 属性

### RecordingSettings
- **类型**: `FMediaPlayerRecordingSettings`
- **描述**: Whether this MediaPlayer is active and its event will be recorded when the 'Record' button is pressed.

### MediaPlayerToRecord
- **类型**: `TWeakObjectPtr<UMediaPlayer>`
- **描述**: The MediaPlayer we want to record
