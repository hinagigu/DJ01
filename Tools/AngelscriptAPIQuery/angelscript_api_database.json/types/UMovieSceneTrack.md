# UMovieSceneTrack

**继承自**: `UMovieSceneSignedObject`

Base class for a track in a Movie Scene

## 属性

### EvalOptions
- **类型**: `FMovieSceneTrackEvalOptions`
- **描述**: General evaluation options for a given track

### DisplayOptions
- **类型**: `FMovieSceneTrackDisplayOptions`
- **描述**: General display options for a given track

### TrackTint
- **类型**: `FColor`
- **描述**: This track's tint color
