# FMovieScene3DTransformKeyStruct

Proxy structure for 3D transform section key data.

## 属性

### Location
- **类型**: `FVector`
- **描述**: The key's translation value.

### Rotation
- **类型**: `FRotator`
- **描述**: The key's rotation value.

### Scale
- **类型**: `FVector3f`
- **描述**: The key's scale value.

### Time
- **类型**: `FFrameNumber`
- **描述**: The key's time.
