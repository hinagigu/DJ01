# __EVirtualizationMode

Method of virtualization when a sound is stopped due to playback constraints
(i.e. by concurrency, priority, and/or MaxChannelCount)
for a given sound.

## 属性

### Disabled
- **类型**: `EVirtualizationMode`

### PlayWhenSilent
- **类型**: `EVirtualizationMode`

### Restart
- **类型**: `EVirtualizationMode`

### EVirtualizationMode_MAX
- **类型**: `EVirtualizationMode`
