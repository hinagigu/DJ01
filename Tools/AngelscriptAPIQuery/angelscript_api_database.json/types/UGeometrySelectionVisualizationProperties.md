# UGeometrySelectionVisualizationProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### bShowSelection
- **类型**: `bool`
- **描述**: Render the geometry selection

### bShowTriangleROIBorder
- **类型**: `bool`
- **描述**: Render the geometry selection as the border of the triangles in the ROI (region of influence)

### bShowHidden
- **类型**: `bool`
- **描述**: Show occluded parts of the selection

### bShowEdgeSelectionVertices
- **类型**: `bool`
- **描述**: This tool treats edge selections as vertex selections. Enable this to show the edited vertices

### LineThickness
- **类型**: `float32`
- **描述**: Line thickness used to render the geometry selection

### PointSize
- **类型**: `float32`
- **描述**: Point size used to render the geometry selection

### DepthBias
- **类型**: `float32`
- **描述**: Depth bias used to slightly shift depth of points/lines

### FaceColor
- **类型**: `FColor`
- **描述**: Color used to render the geometry selection

### LineColor
- **类型**: `FColor`
- **描述**: Color used to render the geometry selection

### PointColor
- **类型**: `FColor`
- **描述**: Color used to render the geometry selection

### TriangleROIBorderColor
- **类型**: `FColor`
- **描述**: Color used to render the geometry selection ROI
