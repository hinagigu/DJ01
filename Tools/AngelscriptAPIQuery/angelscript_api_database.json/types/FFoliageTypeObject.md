# FFoliageTypeObject

A wrapper struct used to allow the use of either FoliageType assets or FoliageType blueprint classes

## 属性

### FoliageTypeObject
- **类型**: `UObject`
- **描述**: The foliage type that will be spawned by the procedural foliage simulation
