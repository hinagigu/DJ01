# UPatternTool_BoundingBoxSettings

**继承自**: `UInteractiveToolPropertySet`

Settings for Bounding Box adjustments in the Pattern Tool

## 属性

### bIgnoreTransforms
- **类型**: `bool`
- **描述**: If true, pattern element bounding boxes are not changed to account for StartScale or StartRotation

### Adjustment
- **类型**: `float32`
- **描述**: Value added to the all pattern elements' bounding boxes for adjusting the behavior of packed spacing mode manually

### bVisualize
- **类型**: `bool`
- **描述**: If true, the bounding boxes of each element are rendered in green and the combined bounding box of all source elements is rendered in red
