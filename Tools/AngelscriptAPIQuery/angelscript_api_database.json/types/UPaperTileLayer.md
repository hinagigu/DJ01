# UPaperTileLayer

**继承自**: `UObject`

This class represents a single layer in a tile map.  All layers in the map must have the size dimensions.

## 属性

### LayerName
- **类型**: `FText`
- **描述**: Name of the layer

### LayerWidth
- **类型**: `int`
- **描述**: Width of the layer (in tiles)

### LayerHeight
- **类型**: `int`
- **描述**: Height of the layer (in tiles)

### CollisionThicknessOverride
- **类型**: `float32`

### CollisionOffsetOverride
- **类型**: `float32`

### LayerColor
- **类型**: `FLinearColor`

### bHiddenInEditor
- **类型**: `bool`

### bHiddenInGame
- **类型**: `bool`

### bLayerCollides
- **类型**: `bool`

### bOverrideCollisionThickness
- **类型**: `bool`

### bOverrideCollisionOffset
- **类型**: `bool`
