# FBuildPromotionTestSettings

Holds settings for the editor build promotion test

## 属性

### DefaultStaticMeshAsset
- **类型**: `FFilePath`
- **描述**: Default static mesh asset to apply materials to *

### ImportWorkflow
- **类型**: `FBuildPromotionImportWorkflowSettings`
- **描述**: Import workflow settings *

### OpenAssets
- **类型**: `FBuildPromotionOpenAssetSettings`
- **描述**: Open assets settings *

### NewProjectSettings
- **类型**: `FBuildPromotionNewProjectSettings`
- **描述**: New project settings *

### SourceControlMaterial
- **类型**: `FFilePath`
- **描述**: Material to modify for the content browser step *

## 方法

### opAssign
```angelscript
FBuildPromotionTestSettings& opAssign(FBuildPromotionTestSettings Other)
```
