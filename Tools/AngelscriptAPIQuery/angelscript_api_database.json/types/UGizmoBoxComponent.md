# UGizmoBoxComponent

**继承自**: `UGizmoBaseComponent`

Simple Component intended to be used as part of 3D Gizmos.
Draws outline of 3D Box based on parameters.

## 属性

### Origin
- **类型**: `FVector`

### Rotation
- **类型**: `FQuat`

### Dimensions
- **类型**: `FVector`

### LineThickness
- **类型**: `float32`

### bRemoveHiddenLines
- **类型**: `bool`

### bEnableAxisFlip
- **类型**: `bool`
