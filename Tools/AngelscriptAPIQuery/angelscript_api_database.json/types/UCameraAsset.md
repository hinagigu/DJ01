# UCameraAsset

**继承自**: `UObject`

A complete camera asset.

## 属性

### CameraDirector
- **类型**: `UCameraDirector`
- **描述**: The camera director to use in this camera.

### EnterTransitions
- **类型**: `TArray<FCameraModeTransition>`
- **描述**: A list of default enter transitions for all the camera modes in this asset.

### ExitTransitions
- **类型**: `TArray<FCameraModeTransition>`
- **描述**: A list of default exit transitions for all the camera modes in this asset.
