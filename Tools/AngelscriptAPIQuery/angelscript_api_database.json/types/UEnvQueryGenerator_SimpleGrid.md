# UEnvQueryGenerator_SimpleGrid

**继承自**: `UEnvQueryGenerator_ProjectedPoints`

Simple grid, generates points in 2D square around context

## 属性

### GridSize
- **类型**: `FAIDataProviderFloatValue`
- **描述**: half of square's extent, like a radius

### SpaceBetween
- **类型**: `FAIDataProviderFloatValue`
- **描述**: generation density

### GenerateAround
- **类型**: `TSubclassOf<UEnvQueryContext>`
- **描述**: context
