# UMVVMBlueprintViewEvent

**继承自**: `UObject`

## 属性

### bEnabled
- **类型**: `bool`
- **描述**: Whether the event is enabled or disabled by default. The instance may enable the event at runtime.

### bCompile
- **类型**: `bool`
- **描述**: The event is visible in the editor, but is not compiled and cannot be used at runtime.
