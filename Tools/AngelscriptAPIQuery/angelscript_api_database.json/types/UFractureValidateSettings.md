# UFractureValidateSettings

**继承自**: `UFractureToolSettings`

Settings for the Validate tool *

## 属性

### bRemoveUnreferencedGeometry
- **类型**: `bool`
- **描述**: Find and remove any unused geometry data

### bRemoveClustersOfOne
- **类型**: `bool`
- **描述**: Whether to collapse any clusters with only a single child

### bRemoveDanglingClusters
- **类型**: `bool`
- **描述**: Remove dangling clusters -- Note this can invalidate caches
