**继承自**: `UParticleModuleEventReceiverBase`

## 属性

### SpawnCount
- **类型**: `FRawDistributionFloat`
- **描述**: The number of particles to spawn.

### InheritVelocityScale
- **类型**: `FRawDistributionVector`
- **描述**: If bInheritVelocity is true, scale the velocity with this.

### PhysicalMaterials
- **类型**: `TArray<TObjectPtr<UPhysicalMaterial>>`
- **描述**: Array of physical materials that can be used to allow or ban a specific set of materials when receiving collision events.
