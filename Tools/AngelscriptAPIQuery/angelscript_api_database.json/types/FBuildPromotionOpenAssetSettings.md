# FBuildPromotionOpenAssetSettings

Holds settings for the open assets stage of the build promotion test

## 属性

### BlueprintAsset
- **类型**: `FFilePath`
- **描述**: The blueprint asset to open

### MaterialAsset
- **类型**: `FFilePath`
- **描述**: The material asset to open

### ParticleSystemAsset
- **类型**: `FFilePath`
- **描述**: The particle system asset to open

### SkeletalMeshAsset
- **类型**: `FFilePath`
- **描述**: The skeletal mesh asset to open

### StaticMeshAsset
- **类型**: `FFilePath`
- **描述**: The static mesh asset to open

### TextureAsset
- **类型**: `FFilePath`
- **描述**: The texture asset to open
