# UEllipseSplineGenerator

**继承自**: `USplineGeneratorBase`

## 属性

### NumberOfPoints
- **类型**: `int`

### Length
- **类型**: `float32`

### Width
- **类型**: `float32`

### bReverseDir
- **类型**: `bool`
- **描述**: If enabled, will reverse the direction of the arc

### bKeepFirstKeyTangent
- **类型**: `bool`
- **描述**: If enabled, will start the shape tangent to the current path

### bBranchRight
- **类型**: `bool`
- **描述**: If enabled, will switch the shape's center to the right of the curve
