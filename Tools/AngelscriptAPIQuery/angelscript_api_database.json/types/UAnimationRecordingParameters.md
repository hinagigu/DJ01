# UAnimationRecordingParameters

**继承自**: `UObject`

## 属性

### SampleFrameRate
- **类型**: `FFrameRate`
- **描述**: Sample frame-rate of the recorded animation

### bEndAfterDuration
- **类型**: `bool`
- **描述**: If enabled, this animation recording will automatically end after a set amount of time

### MaximumDurationSeconds
- **类型**: `float32`
- **描述**: The maximum duration of this animation recording
