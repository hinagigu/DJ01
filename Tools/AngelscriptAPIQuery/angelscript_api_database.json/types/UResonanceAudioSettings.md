# UResonanceAudioSettings

**继承自**: `UObject`

## 属性

### OutputSubmix
- **类型**: `FSoftObjectPath`
- **描述**: Reference to submix where reverb plugin audio is routed.

### QualityMode
- **类型**: `ERaQualityMode`
- **描述**: Global Quality mode to use for directional sound sources.

### GlobalReverbPreset
- **类型**: `FSoftObjectPath`
- **描述**: Default settings for global reverb: This is overridden when a player enters Audio Volumes.

### GlobalSourcePreset
- **类型**: `FSoftObjectPath`
- **描述**: Default settings for global source settings
