# UCollisionGeometryVisualizationProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### bShowCollision
- **类型**: `bool`
- **描述**: Show/hide collision geometry

### bShowSolid
- **类型**: `bool`
- **描述**: Whether to show solid shapes in addition to wireframes

### LineThickness
- **类型**: `float32`
- **描述**: Thickness of lines used to visualize collision shapes

### bShowHidden
- **类型**: `bool`
- **描述**: Show occluded parts of the collision geometry, rendered with dashed lines

### bRandomColors
- **类型**: `bool`
- **描述**: Render each collision geometry with a randomly-chosen color

### Color
- **类型**: `FColor`
- **描述**: The color to use for all collision visualizations, if random colors are not used
