# UAbilitySystemGlobals

**继承自**: `UObject`

Holds global data for the ability system. Can be configured per project via config file

## 属性

### GameplayModEvaluationChannelAliases
- **类型**: `FName`
- **描述**: Game-specified named aliases for gameplay mod evaluation channels; Only those with valid aliases are eligible to be used in a game (except Channel0, which is always valid)
