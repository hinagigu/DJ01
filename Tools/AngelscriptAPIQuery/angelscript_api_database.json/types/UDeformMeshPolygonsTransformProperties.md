# UDeformMeshPolygonsTransformProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### DeformationStrategy
- **类型**: `EGroupTopologyDeformationStrategy`
- **描述**: Type of deformation used.

### TransformMode
- **类型**: `EQuickTransformerMode`
- **描述**: Type of transformation used.

### bSelectFaces
- **类型**: `bool`
- **描述**: Allow for faces (PolyGroups) to be selected.

### bSelectEdges
- **类型**: `bool`
- **描述**: Allow for edges to be selected.

### bSelectVertices
- **类型**: `bool`
- **描述**: Allow for vertices to be selected.

### bShowWireframe
- **类型**: `bool`
- **描述**: If true, overlays preview with wireframe.
