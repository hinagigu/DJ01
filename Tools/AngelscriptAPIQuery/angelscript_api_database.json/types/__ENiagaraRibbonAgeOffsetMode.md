# __ENiagaraRibbonAgeOffsetMode

Defines different modes for offsetting UVs by age when ordering ribbon particles using normalized age.

- **类型**: `ENiagaraRibbonAgeOffsetMode`

### Clip
- **类型**: `ENiagaraRibbonAgeOffsetMode`

### ENiagaraRibbonAgeOffsetMode_MAX
