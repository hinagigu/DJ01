# USimpleCameraShakePattern

**继承自**: `UCameraShakePattern`

A base class for a simple camera shake.

## 属性

### Duration
- **类型**: `float32`
- **描述**: Duration in seconds of this shake. Zero or less means infinite.

### BlendInTime
- **类型**: `float32`
- **描述**: Blend-in time for this shake. Zero or less means no blend-in.

### BlendOutTime
- **类型**: `float32`
- **描述**: Blend-out time for this shake. Zero or less means no blend-out.
