# FTemplateMapInfoOverride

Used by new level dialog.

## 属性

### Thumbnail
- **类型**: `FSoftObjectPath`
- **描述**: The thumbnail to display in the new level dialog

### Map
- **类型**: `FSoftObjectPath`
- **描述**: The path to the template map

### DisplayName
- **类型**: `FText`
- **描述**: The display name of the map template in the level dialog. If this is empty the thumbnail name will be used
