# UTileMapEditorSettings

**继承自**: `UObject`

Settings for the Paper2D tile map editor

## 属性

### DefaultBackgroundColor
- **类型**: `FColor`
- **描述**: Default background color for new tile map assets

### bShowGridByDefault
- **类型**: `bool`
- **描述**: Should the grid be shown by default when the editor is opened?

### DefaultTileGridColor
- **类型**: `FColor`
- **描述**: Default per tile grid color for new tile map assets

### DefaultMultiTileGridColor
- **类型**: `FColor`
- **描述**: Default multi tile grid color for new tile map assets

### DefaultMultiTileGridWidth
- **类型**: `int`
- **描述**: Default number of tiles the multi tile grid spans horizontally for new tile map assets. 0 removes vertical lines

### DefaultMultiTileGridHeight
- **类型**: `int`
- **描述**: Default number of tiles the multi tile grid spans vertically for new tile map assets. 0 removes horizontal lines

### DefaultMultiTileGridOffsetX
- **类型**: `int`
- **描述**: Default number of tiles the multi tile grid is shifted to the right for new tile map assets

### DefaultMultiTileGridOffsetY
- **类型**: `int`
- **描述**: Default number of tiles the multi tile grid is shifted downwards for new tile map assets

### DefaultLayerGridColor
- **类型**: `FColor`
- **描述**: Default layer grid color for new tile map assets
