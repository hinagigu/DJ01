# FEngineServiceNotification

Implements a message that contains a notification or log output.

## 属性

### Text
- **类型**: `FString`
- **描述**: Holds the notification text.

### TimeSeconds
- **类型**: `float`
- **描述**: Holds the time in seconds since the engine started.
