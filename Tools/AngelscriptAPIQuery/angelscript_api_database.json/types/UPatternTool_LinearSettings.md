# UPatternTool_LinearSettings

**继承自**: `UInteractiveToolPropertySet`

Settings for Linear Patterns in the Pattern Tool

## 属性

### SpacingMode
- **类型**: `EPatternToolAxisSpacingMode`
- **描述**: Spacing Technique used to distribute Pattern Elements

### Count
- **类型**: `int`
- **描述**: Number of Pattern Elements to place

### StepSize
- **类型**: `float`
- **描述**: Fixed Increment used to place Pattern Elements

### Extent
- **类型**: `float`
- **描述**: Length of Pattern along the Axis

### bCentered
- **类型**: `bool`
- **描述**: If true, Pattern is centered at the Origin, otherwise Pattern starts at the Origin
