# FAnimatedBoneAttribute

Structure encapsulating animated (bone) attribute data.

## 属性

### Identifier
- **类型**: `FAnimationAttributeIdentifier`
- **描述**: Identifier to reference this attribute by

### Curve
- **类型**: `FAttributeCurve`
- **描述**: Curve containing the (animated) attribute data
