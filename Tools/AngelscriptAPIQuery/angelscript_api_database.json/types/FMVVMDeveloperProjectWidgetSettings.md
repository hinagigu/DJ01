# FMVVMDeveloperProjectWidgetSettings

## 属性

### DisallowedFieldNames
- **类型**: `TSet<FName>`
- **描述**: Properties or functions name that should not be use for binding (read or write).

### AdvancedFieldNames
- **类型**: `TSet<FName>`
- **描述**: Properties or functions name that are displayed in the advanced category.
