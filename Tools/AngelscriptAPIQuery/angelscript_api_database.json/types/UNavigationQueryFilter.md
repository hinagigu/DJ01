# UNavigationQueryFilter

**继承自**: `UObject`

Class containing definition of a navigation query filter

## 属性

### Areas
- **类型**: `TArray<FNavigationFilterArea>`
- **描述**: list of overrides for navigation areas

### IncludeFlags
- **类型**: `FNavigationFilterFlags`
- **描述**: required flags of navigation nodes

### ExcludeFlags
- **类型**: `FNavigationFilterFlags`
- **描述**: forbidden flags of navigation nodes
