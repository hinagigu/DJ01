# UNiagaraDataInterfaceLandscape

**继承自**: `UNiagaraDataInterface`

Data Interface allowing sampling of a Landscape

## 属性

### SourceLandscape
- **类型**: `AActor`

### SourceMode
- **类型**: `ENDILandscape_SourceMode`

### PhysicalMaterials
- **类型**: `TArray<TObjectPtr<UPhysicalMaterial>>`

### bVirtualTexturesSupported
- **类型**: `bool`
- **描述**: Can be used to ignore virtual textures even if they are defined for the landscape.
