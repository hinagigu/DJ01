# UGameplayAbility_Montage

**继承自**: `UGameplayAbility`

A gameplay ability that plays a single montage and applies a GameplayEffect

## 属性

### MontageToPlay
- **类型**: `UAnimMontage`

### PlayRate
- **类型**: `float32`

### SectionName
- **类型**: `FName`

### GameplayEffectClassesWhileAnimating
- **类型**: `TArray<TSubclassOf<UGameplayEffect>>`
- **描述**: GameplayEffects to apply and then remove while the animation is playing
