# UBrushRemeshProperties

**继承自**: `URemeshProperties`

## 属性

### bEnableRemeshing
- **类型**: `bool`
- **描述**: Toggle remeshing on/off

### TriangleSize
- **类型**: `int`
- **描述**: Desired size of triangles after Remeshing, relative to average initial triangle size. Larger value results in larger triangles.

### PreserveDetail
- **类型**: `int`
- **描述**: Control the amount of simplification during sculpting. Higher values will avoid wiping out fine details on the mesh.

### Iterations
- **类型**: `int`
