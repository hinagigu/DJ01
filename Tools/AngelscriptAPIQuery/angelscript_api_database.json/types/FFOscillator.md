# FFOscillator

Defines oscillation of a single number.

## 属性

### Amplitude
- **类型**: `float32`

### Frequency
- **类型**: `float32`

### InitialOffset
- **类型**: `EInitialOscillatorOffset`
- **描述**: Defines how to begin (either at zero, or at a randomized value.

### Waveform
- **类型**: `EOscillatorWaveform`
