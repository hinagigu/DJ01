# UMergeMeshesToolProperties

**继承自**: `UInteractiveToolPropertySet`

Standard properties of the Merge Meshes operation

## 属性

### VoxelCount
- **类型**: `int`
- **描述**: The size of the geometry bounding box major axis measured in voxels.

### MeshAdaptivity
- **类型**: `float32`
- **描述**: Remeshing adaptivity, prior to optional simplification

### bAutoSimplify
- **类型**: `bool`
- **描述**: Automatically simplify the result of voxel-based merge.
