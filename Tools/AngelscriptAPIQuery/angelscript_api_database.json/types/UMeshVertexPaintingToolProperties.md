# UMeshVertexPaintingToolProperties

**继承自**: `UBrushBaseProperties`

## 属性

### PaintColor
- **类型**: `FLinearColor`
- **描述**: Color used for Applying Vertex Color Painting

### EraseColor
- **类型**: `FLinearColor`
- **描述**: Color used for Erasing Vertex Color Painting

### bEnableFlow
- **类型**: `bool`
- **描述**: Enables "Flow" painting where paint is continually applied from the brush every tick

### bOnlyFrontFacingTriangles
- **类型**: `bool`
- **描述**: Whether back-facing triangles should be ignored

### VertexPreviewSize
- **类型**: `float32`
- **描述**: Size of vertex points drawn when mesh painting is active.
