# USkinWeightsBindingToolProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### BindingType
- **类型**: `ESkinWeightsBindType`
- **描述**: Binding type to use

### Stiffness
- **类型**: `float32`
- **描述**: Stiffness of binding. Lower values allow more distant bones to contribute more

### MaxInfluences
- **类型**: `int`
- **描述**: Maximum bones that will influence each vertex

### VoxelResolution
- **类型**: `int`
- **描述**: The resolution of the voxel grid if doing geodesic voxel binding
