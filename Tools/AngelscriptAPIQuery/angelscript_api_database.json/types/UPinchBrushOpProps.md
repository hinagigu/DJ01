# UPinchBrushOpProps

**继承自**: `UMeshSculptBrushOpProps`

## 属性

### Strength
- **类型**: `float32`
- **描述**: Strength of the Brush

### Falloff
- **类型**: `float32`
- **描述**: Amount of falloff to apply

### Depth
- **类型**: `float32`
- **描述**: Depth of Brush into surface along surface normal

### bPerpDamping
- **类型**: `bool`
- **描述**: When enabled, brush will damp motion of vertices that would move perpendicular to brush stroke direction
