# UMaterialFunctionInstance

**继承自**: `UMaterialFunctionInterface`

A material function instance defines parameter overrides for a parent material function.

## 属性

### Parent
- **类型**: `UMaterialFunctionInterface`

### ScalarParameterValues
- **类型**: `TArray<FScalarParameterValue>`
- **描述**: Scalar parameters.

### VectorParameterValues
- **类型**: `TArray<FVectorParameterValue>`
- **描述**: Vector parameters.

### DoubleVectorParameterValues
- **类型**: `TArray<FDoubleVectorParameterValue>`
- **描述**: DoubleVector parameters.

### TextureParameterValues
- **类型**: `TArray<FTextureParameterValue>`
- **描述**: Texture parameters.

### FontParameterValues
- **类型**: `TArray<FFontParameterValue>`
- **描述**: Font parameters.

### RuntimeVirtualTextureParameterValues
- **类型**: `TArray<FRuntimeVirtualTextureParameterValue>`
- **描述**: Runtime virtual texture parameters.

### SparseVolumeTextureParameterValues
- **类型**: `TArray<FSparseVolumeTextureParameterValue>`
- **描述**: Sparse volume texture parameters.
