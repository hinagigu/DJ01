# FNiagaraEmitterScalabilitySettings

Scalability settings for Niagara Emitters on a particular platform set.

## 属性

### Platforms
- **类型**: `FNiagaraPlatformSet`
- **描述**: The platforms on which these settings are active (unless overridden).

### SpawnCountScale
- **类型**: `float32`
- **描述**: Scale factor applied to spawn counts for this emitter.

### bScaleSpawnCount
- **类型**: `bool`
