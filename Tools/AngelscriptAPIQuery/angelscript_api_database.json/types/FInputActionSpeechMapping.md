# FInputActionSpeechMapping

Defines a mapping between an action and speech recognition

@see https://docs.unrealengine.com/latest/INT/Gameplay/Input/index.html

## 属性

### ActionName
- **类型**: `FName`
- **描述**: Friendly name of action, e.g "jump"

### SpeechKeyword
- **类型**: `FName`
- **描述**: Key to bind it to.

## 方法

### opAssign
```angelscript
FInputActionSpeechMapping& opAssign(FInputActionSpeechMapping Other)
```
