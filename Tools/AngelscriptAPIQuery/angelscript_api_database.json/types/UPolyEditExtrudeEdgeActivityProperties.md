# UPolyEditExtrudeEdgeActivityProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### DirectionMode
- **类型**: `EPolyEditExtrudeEdgeDirectionMode`
- **描述**: How direction to move the vertices is determined.

### DistanceMode
- **类型**: `EPolyEditExtrudeEdgeDistanceMode`
- **描述**: How distance to extrude along a given direction is determined.

### Distance
- **类型**: `float`

### bUseUnselectedForFrames
- **类型**: `bool`
- **描述**: When a vertex has both a selected and nonselected neighbor, use the unselected neighbor in picking an extrude frame as well.

### bAdjustToExtrudeEvenly
- **类型**: `bool`
- **描述**: Adjust individual extrude directions in an effort to make extruded edges parallel to their original edges.
