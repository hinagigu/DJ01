# UUVEditorBackgroundPreviewProperties

**继承自**: `UInteractiveToolPropertySet`

Visualization settings for the UUVEditorBackgroundPreview

## 属性

### bVisible
- **类型**: `bool`
- **描述**: Should the background be shown (Alt+B)

### SourceType
- **类型**: `EUVEditorBackgroundSourceType`
- **描述**: Source of background visuals

### SourceTexture
- **类型**: `UTexture2D`
- **描述**: Display a background based on the selected texture

### SourceMaterial
- **类型**: `UMaterial`
- **描述**: Display a background based on the selected material
