# ULevelEditor2DSettings

**继承自**: `UDeveloperSettings`

Configure settings for the 2D Level Editor

## 属性

### bEnable2DWidget
- **类型**: `bool`
- **描述**: If enabled, shows the 2D combined translate and rotate tool in the viewport toolbar.

### bEnableSnapLayers
- **类型**: `bool`
- **描述**: If enabled, shows the 2D layer snapping controls in the viewport toolbar.

### SnapAxis
- **类型**: `ELevelEditor2DAxis`
- **描述**: Sets the world space axis for 2D snap layers.

### SnapLayers
- **类型**: `TArray<FMode2DLayer>`
- **描述**: Snap layers that are displayed in the viewport toolbar.
