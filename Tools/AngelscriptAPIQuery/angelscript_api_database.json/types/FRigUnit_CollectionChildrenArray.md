# FRigUnit_CollectionChildrenArray

Creates an item array based on the direct or recursive children
of a provided parent item. Returns an empty array for an invalid parent item.

## 属性

### Parent
- **类型**: `FRigElementKey`

### bIncludeParent
- **类型**: `bool`

### bRecursive
- **类型**: `bool`

### TypeToSearch
- **类型**: `ERigElementType`

### Items
- **类型**: `TArray<FRigElementKey>`
