# UNiagaraParameterCollection

**继承自**: `UObject`

Asset containing a collection of global parameters usable by Niagara.

## 属性

### Namespace
- **类型**: `FName`
- **描述**: Namespace for this parameter collection. Is enforced to be unique across all parameter collections.

### SourceMaterialCollection
- **类型**: `UMaterialParameterCollection`
- **描述**: Optional set of MPC that can drive scalar and vector parameters
