# URenderCaptureProperties

**继承自**: `UInteractiveToolPropertySet`

Tool Properties

## 属性

### Resolution
- **类型**: `EBakeTextureResolution`

### bBaseColorMap
- **类型**: `bool`
- **描述**: Whether to generate a texture for the Base Color property

### bNormalMap
- **类型**: `bool`
- **描述**: Whether to generate a texture for the World Normal property

### bPackedMRSMap
- **类型**: `bool`
- **描述**: Whether to generate a packed texture with Metallic, Roughness and Specular properties

### bMetallicMap
- **类型**: `bool`
- **描述**: Whether to generate a texture for the Metallic property

### bRoughnessMap
- **类型**: `bool`
- **描述**: Whether to generate a texture for the Roughness property

### bSpecularMap
- **类型**: `bool`
- **描述**: Whether to generate a texture for the Specular property

### bEmissiveMap
- **类型**: `bool`
- **描述**: Whether to generate a texture for the Emissive property

### bOpacityMap
- **类型**: `bool`
- **描述**: Whether to generate a texture for the Opacity property

### bSubsurfaceColorMap
- **类型**: `bool`
- **描述**: Whether to generate a texture for the SubsurfaceColor property

### bAntiAliasing
- **类型**: `bool`
- **描述**: Whether to use anti-aliasing in the render captures, this may introduce artefacts if pixels at different scene depths get blended
