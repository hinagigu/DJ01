# FEditorMapPerformanceTestDefinition

Holds settings for the asset import / export automation test

## 属性

### PerformanceTestmap
- **类型**: `FSoftObjectPath`
- **描述**: Map to be used for the Performance Capture *

### TestTimer
- **类型**: `int`
- **描述**: This is the length of time in seconds that this test will run for before stopping.
