# FChaosDebugDrawColorsByShapeType

Structure holding the settings using to debug draw Particles shape based on their shape type on the Chaos Visual Debugger

## 属性

### SimpleTypeColor
- **类型**: `FColor`
- **描述**: Color used for Sphere, Plane, Cube, Capsule, Cylinder, tapered shapes

### ConvexColor
- **类型**: `FColor`
- **描述**: Color used for convex shapes

### HeightFieldColor
- **类型**: `FColor`
- **描述**: Color used for heightfield

### TriangleMeshColor
- **类型**: `FColor`
- **描述**: Color used for triangle meshes

### LevelSetColor
- **类型**: `FColor`
- **描述**: Color used for triangle LevelSets
