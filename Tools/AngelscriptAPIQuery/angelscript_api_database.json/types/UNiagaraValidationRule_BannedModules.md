# UNiagaraValidationRule_BannedModules

**继承自**: `UNiagaraValidationRule`

This validation rule can ban the use of certain modules on all or a subset of platforms.

## 属性

### Platforms
- **类型**: `FNiagaraPlatformSet`
- **描述**: Platforms this validation rule will apply to.

### BannedModules
- **类型**: `TArray<TObjectPtr<UNiagaraScript>>`
