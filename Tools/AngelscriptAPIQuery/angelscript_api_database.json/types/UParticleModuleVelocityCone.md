# UParticleModuleVelocityCone

**继承自**: `UParticleModuleVelocityBase`

## 属性

### Angle
- **类型**: `FRawDistributionFloat`
- **描述**: The Min value represents the inner cone angle value and the Max value represents the outer cone angle value.

### Velocity
- **类型**: `FRawDistributionFloat`
- **描述**: The initial velocity of the particles.

### Direction
- **类型**: `FVector`
- **描述**: The direction FVector of the cone.
