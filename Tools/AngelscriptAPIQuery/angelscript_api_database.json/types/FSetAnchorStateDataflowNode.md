# FSetAnchorStateDataflowNode

Sets the anchored state on the selected bones in a Collection

## 属性

### AnchorState
- **类型**: `EAnchorStateEnum`
- **描述**: What anchor state to set on selected bones

### bSetNotSelectedBonesToOppositeState
- **类型**: `bool`
- **描述**: If true, sets the non selected bones to opposite anchor state

### bActive
- **类型**: `bool`
