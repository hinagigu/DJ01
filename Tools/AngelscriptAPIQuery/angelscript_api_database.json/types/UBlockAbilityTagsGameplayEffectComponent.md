# UBlockAbilityTagsGameplayEffectComponent

**继承自**: `UGameplayEffectComponent`

Handles blocking the activation of Gameplay Abilities based on Gameplay Tags for the Target actor of the owner Gameplay Effect

## 属性

### InheritableBlockedAbilityTagsContainer
- **类型**: `FInheritedTagContainer`
- **描述**: These tags are applied to the target actor of the Gameplay Effect.  Blocked Ability Tags prevent Gameplay Abilities with these tags from executing.
