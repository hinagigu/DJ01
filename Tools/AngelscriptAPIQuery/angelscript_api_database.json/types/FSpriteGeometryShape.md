# FSpriteGeometryShape

A single piece of geometry (e.g., a polygon which may be convex or concave, a box, or a circle)

## 属性

### Vertices
- **类型**: `TArray<FVector2D>`
- **描述**: Vertices for the polygon (valid for Box and Polygon, but empty for Circle)
