Data about an input device's current state

## 属性

### OwningPlatformUser
- **类型**: `FPlatformUserId`
- **描述**: The platform user that this input device belongs to

### ConnectionState
- **类型**: `EInputDeviceConnectionState`
- **描述**: The connection state of this input device
