# UGameFeatureAction_AddGameplayCuePath

**继承自**: `UGameFeatureAction`

GameFeatureAction responsible for adding gameplay cue paths to the gameplay cue manager.

@see UAbilitySystemGlobals::GameplayCueNotifyPaths

## 属性

### DirectoryPathsToAdd
- **类型**: `TArray<FDirectoryPath>`
- **描述**: List of paths to register to the gameplay cue manager. These are relative tot he game content directory
