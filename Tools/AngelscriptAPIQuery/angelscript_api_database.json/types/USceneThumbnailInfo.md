# USceneThumbnailInfo

**继承自**: `UThumbnailInfo`

## 属性

### OrbitPitch
- **类型**: `float32`
- **描述**: The pitch of the orbit camera around the asset

### OrbitYaw
- **类型**: `float32`
- **描述**: The yaw of the orbit camera around the asset

### OrbitZoom
- **类型**: `float32`
- **描述**: The offset from the bounds sphere distance from the asset
