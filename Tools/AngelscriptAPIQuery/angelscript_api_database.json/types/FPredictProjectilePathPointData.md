# FPredictProjectilePathPointData

Data about a single point in a projectile path trace.

## 属性

### Location
- **类型**: `FVector`
- **描述**: Location of this point

### Velocity
- **类型**: `FVector`
- **描述**: Velocity at this point

### Time
- **类型**: `float32`
- **描述**: Elapsed time at this point from the start of the trace.
