# UGameplayDebuggerUserSettings

**继承自**: `UDeveloperSettings`

## 属性

### MaxViewDistance
- **类型**: `float32`
- **描述**: Distance from view location under which actors can be selected
This distance can also be used by some categories to apply culling.

### MaxViewAngle
- **类型**: `float32`
- **描述**: Angle from view direction under which actors can be selected
This angle can also be used by some categories to apply culling.

### FontSize
- **类型**: `int`
- **描述**: Font Size used by Gameplay Debugger

### bEnableGameplayDebuggerInEditor
- **类型**: `bool`
