# UMovieSceneToolsProjectSettings

**继承自**: `UObject`

Settings for the level sequences

## 属性

### DefaultStartTime
- **类型**: `float32`
- **描述**: The default start time for new level sequences, in seconds.

### DefaultDuration
- **类型**: `float32`
- **描述**: The default duration for new level sequences in seconds.

### SubsequenceDirectory
- **类型**: `FString`
- **描述**: The default directory for the subsequences.

### ShotDirectory
- **类型**: `FString`
- **描述**: The default directory for the shots.

### SubsequencePrefix
- **类型**: `FString`
- **描述**: The default prefix for subsequence names.

### ShotPrefix
- **类型**: `FString`
- **描述**: The default prefix for shot names.

### FirstShotNumber
- **类型**: `uint`
- **描述**: The first shot number.

### ShotIncrement
- **类型**: `uint`
- **描述**: The default shot increment.

### ShotNumDigits
- **类型**: `uint`
- **描述**: The number of digits for the shot number.

### TakeNumDigits
- **类型**: `uint`
- **描述**: The number of digits for the take number.

### FirstTakeNumber
- **类型**: `uint`
- **描述**: The first take number.

### TakeSeparator
- **类型**: `FString`
- **描述**: Character separator between the shot number and the take number.

### FbxSettings
- **类型**: `TArray<FMovieSceneToolsFbxSettings>`
- **描述**: Mapping between fbx property name and property track path
