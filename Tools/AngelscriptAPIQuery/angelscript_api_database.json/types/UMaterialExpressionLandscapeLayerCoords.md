# UMaterialExpressionLandscapeLayerCoords

**继承自**: `UMaterialExpression`

## 属性

### MappingType
- **类型**: `ETerrainCoordMappingType`
- **描述**: Determines the mapping place to use on the terrain.

### CustomUVType
- **类型**: `ELandscapeCustomizedCoordType`
- **描述**: Determines the mapping place to use on the terrain.

### MappingScale
- **类型**: `float32`
- **描述**: Uniform scale to apply to the mapping.

### MappingRotation
- **类型**: `float32`
- **描述**: Rotation to apply to the mapping.

### MappingPanU
- **类型**: `float32`
- **描述**: Offset to apply to the mapping along U.

### MappingPanV
- **类型**: `float32`
- **描述**: Offset to apply to the mapping along V.
