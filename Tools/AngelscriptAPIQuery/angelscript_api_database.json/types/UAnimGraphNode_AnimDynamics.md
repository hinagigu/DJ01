# UAnimGraphNode_AnimDynamics

**继承自**: `UAnimGraphNode_SkeletalControlBase`

## 属性

### Node
- **类型**: `FAnimNode_AnimDynamics`

### bPreviewLive
- **类型**: `bool`
- **描述**: Preview the live physics object on the mesh

### bShowLinearLimits
- **类型**: `bool`
- **描述**: Show linear (prismatic) limits in the viewport

### bShowAngularLimits
- **类型**: `bool`
- **描述**: Show angular limit ranges in the viewport

### bShowPlanarLimit
- **类型**: `bool`
- **描述**: Show planar limit info (actual plane, plane normal) in the viewport

### bShowSphericalLimit
- **类型**: `bool`
- **描述**: Show spherical limits in the viewport (preview live only)

### bShowCollisionSpheres
- **类型**: `bool`
- **描述**: If planar limits are enabled and the collision mode isn't CoM, draw sphere collision sizes
