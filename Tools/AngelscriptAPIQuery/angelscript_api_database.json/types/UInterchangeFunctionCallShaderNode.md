# UInterchangeFunctionCallShaderNode

**继承自**: `UInterchangeShaderNode`

A function call shader node has a named set of inputs and outputs which corresponds to the inputs and outputs of the shader function it instances.

## 方法

### GetCustomMaterialFunction
```angelscript
bool GetCustomMaterialFunction(FString& AttributeValue)
```

### SetCustomMaterialFunction
```angelscript
bool SetCustomMaterialFunction(FString AttributeValue)
```
Set the unique id of the material function referenced by the function call expression.
