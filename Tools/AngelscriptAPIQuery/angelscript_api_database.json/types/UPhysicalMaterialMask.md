# UPhysicalMaterialMask

**继承自**: `UObject`

Physical material masks are used to map multiple physical materials to a single rendering material

## 属性

### AssetImportData
- **类型**: `UAssetImportData`

### MaskTexture
- **类型**: `UTexture`
- **描述**: Mask input texture, square aspect ratio recommended. Recognized mask colors include: white, black, red, green, yellow, cyan, turquoise, and magenta.

### UVChannelIndex
- **类型**: `int`

### AddressX
- **类型**: `TextureAddress`

### AddressY
- **类型**: `TextureAddress`
