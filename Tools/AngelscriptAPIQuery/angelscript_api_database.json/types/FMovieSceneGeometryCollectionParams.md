# FMovieSceneGeometryCollectionParams

## 属性

### GeometryCollectionCache
- **类型**: `FSoftObjectPath`
- **描述**: The animation this section plays

### StartFrameOffset
- **类型**: `FFrameNumber`
- **描述**: The offset into the beginning of the animation clip

### EndFrameOffset
- **类型**: `FFrameNumber`
- **描述**: The offset into the end of the animation clip

### PlayRate
- **类型**: `float32`
- **描述**: The playback rate of the animation clip
