# FRigUnit_SetSpaceInitialTransform

SetSpaceInitialTransform is used to perform a change in the hierarchy by setting a single space's initial transform.

## 属性

### SpaceName
- **类型**: `FName`

### Transform
- **类型**: `FTransform`

### Result
- **类型**: `FTransform`
- **描述**: The transform value result (after weighting)

### Space
- **类型**: `ERigVMTransformSpace`

### ExecuteContext
- **类型**: `FControlRigExecuteContext`
