# __EButtonClickMethod

Enumerates different methods that a button click can be triggered. Normally, DownAndUp is appropriate.

## 属性

### DownAndUp
- **类型**: `EButtonClickMethod`

### MouseDown
- **类型**: `EButtonClickMethod`

### MouseUp
- **类型**: `EButtonClickMethod`

### PreciseClick
- **类型**: `EButtonClickMethod`

### EButtonClickMethod_MAX
- **类型**: `EButtonClickMethod`
