## 属性

### CueVolume
- **类型**: `float32`
- **描述**: The volume of the created sound cue

### CuePackageSuffix
- **类型**: `FString`
- **描述**: If not empty, generated SoundCues will be placed in PackageCuePackageSuffix, but only if bAutoCreateCue is true
