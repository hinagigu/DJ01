# UGizmoArrowComponent

**继承自**: `UGizmoBaseComponent`

Simple Component intended to be used as part of 3D Gizmos.
Currently draws the "arrow" as a single 3D line.

## 属性

### Direction
- **类型**: `FVector`

### Gap
- **类型**: `float32`

### Length
- **类型**: `float32`

### Thickness
- **类型**: `float32`
