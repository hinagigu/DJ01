# FHeaderRowStyle

Represents the appearance of an SHeaderRow

## 属性

### ColumnStyle
- **类型**: `FTableColumnHeaderStyle`
- **描述**: Style of the normal header row columns

### LastColumnStyle
- **类型**: `FTableColumnHeaderStyle`
- **描述**: Style of the last header row column

### ColumnSplitterStyle
- **类型**: `FSplitterStyle`
- **描述**: Style of the splitter used between the columns

### SplitterHandleSize
- **类型**: `float32`
- **描述**: Size of the splitter used between the columns

### BackgroundBrush
- **类型**: `FSlateBrush`
- **描述**: Brush used to draw the header row background

### ForegroundColor
- **类型**: `FSlateColor`
- **描述**: Color used to draw the header row foreground

### HorizontalSeparatorBrush
- **类型**: `FSlateBrush`
- **描述**: Brush used to draw the splitter between the header and the contents below it

### HorizontalSeparatorThickness
- **类型**: `float32`
