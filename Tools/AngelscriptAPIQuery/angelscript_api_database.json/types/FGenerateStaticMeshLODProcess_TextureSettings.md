# FGenerateStaticMeshLODProcess_TextureSettings

## 属性

### BakeResolution
- **类型**: `EGenerateStaticMeshLODBakeResolution`
- **描述**: Resolution for normal map and texture baking

### BakeThickness
- **类型**: `float32`
- **描述**: How far away from the output mesh to search for input mesh during baking

### bCombineTextures
- **类型**: `bool`
- **描述**: Generate an atlassed texture where multiple textures are combined into a single output texture
