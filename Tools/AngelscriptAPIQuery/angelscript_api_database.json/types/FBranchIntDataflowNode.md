# FBranchIntDataflowNode

Branch between two int inputs based on boolean condition

## 属性

### A
- **类型**: `int`
- **描述**: Int input

### B
- **类型**: `int`
- **描述**: Int input

### bCondition
- **类型**: `bool`
- **描述**: If true, Output = A, otherwise Output = B

### bActive
- **类型**: `bool`
