# UHardwareTargetingSettings

**继承自**: `UObject`

Hardware targeting settings, stored in default config, per-project

## 属性

### TargetedHardwareClass
- **类型**: `EHardwareClass`
- **描述**: Enum specifying which class of hardware this game is targeting

### DefaultGraphicsPerformance
- **类型**: `EGraphicsPreset`
- **描述**: Enum specifying a graphics preset to use for this game
