# UEditorProjectAppearanceSettings

**继承自**: `UDeveloperSettings`

Editor project appearance settings. Stored in default config, per-project

## 属性

### bDisplayUnits
- **类型**: `bool`
- **描述**: Whether to display units on editor properties where the property has units set.

### bDisplayUnitsOnComponentTransforms
- **类型**: `bool`
- **描述**: Whether to display units on component transform properties

### DistanceUnits
- **类型**: `TArray<EUnit>`
- **描述**: Choose a set of units in which to display distance/length values.

### MassUnits
- **类型**: `TArray<EUnit>`
- **描述**: Choose a set of units in which to display masses.

### TimeUnits
- **类型**: `TArray<EUnit>`
- **描述**: Choose the units in which to display time.

### AngleUnits
- **类型**: `EUnit`
- **描述**: Choose the units in which to display angles.

### SpeedUnits
- **类型**: `EUnit`
- **描述**: Choose the units in which to display speeds and velocities.

### AngularSpeedUnits
- **类型**: `EUnit`
- **描述**: Choose the units in which to display angular speeds.

### AccelerationUnits
- **类型**: `EUnit`
- **描述**: Choose the units in which to display acceleration.

### TemperatureUnits
- **类型**: `EUnit`
- **描述**: Choose the units in which to display temperatures.

### ForceUnits
- **类型**: `EUnit`
- **描述**: Choose the units in which to display forces.

### TorqueUnits
- **类型**: `EUnit`
- **描述**: Choose the units in which to display torques.

### ImpulseUnits
- **类型**: `EUnit`
- **描述**: Choose the units in which to display impulses.

### PositionalImpulseUnits
- **类型**: `EUnit`
- **描述**: Choose the units in which to display positional impulses.

### ShowSearchableNames
- **类型**: `EReferenceViewerSettingMode`
- **描述**: Should the Reference Viewer have 'Show Searchable Names' checked by default when opened in this project

### ReferenceViewerDefaultMaxSearchBreadth
- **类型**: `int`
- **描述**: The default maximum search breadth for the reference viewer when opened
