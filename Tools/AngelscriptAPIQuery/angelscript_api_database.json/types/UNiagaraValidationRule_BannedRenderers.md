# UNiagaraValidationRule_BannedRenderers

**继承自**: `UNiagaraValidationRule`

This validation rule can ban the use of certain renderers on all or a subset of platforms.

## 属性

### Platforms
- **类型**: `FNiagaraPlatformSet`
- **描述**: Platforms this validation rule will apply to.

### BannedRenderers
- **类型**: `TArray<TSubclassOf<UNiagaraRendererProperties>>`
