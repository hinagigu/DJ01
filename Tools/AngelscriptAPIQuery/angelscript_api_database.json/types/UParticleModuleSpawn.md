# UParticleModuleSpawn

**继承自**: `UParticleModuleSpawnBase`

## 属性

### Rate
- **类型**: `FRawDistributionFloat`
- **描述**: The rate at which to spawn particles.

### RateScale
- **类型**: `FRawDistributionFloat`
- **描述**: The scalar to apply to the rate.

### BurstList
- **类型**: `TArray<FParticleBurst>`
- **描述**: The array of burst entries.

### BurstScale
- **类型**: `FRawDistributionFloat`
- **描述**: Scale all burst entries by this amount.

### ParticleBurstMethod
- **类型**: `EParticleBurstMethod`
- **描述**: The method to utilize when burst-emitting particles.

### bApplyGlobalSpawnRateScale
- **类型**: `bool`
