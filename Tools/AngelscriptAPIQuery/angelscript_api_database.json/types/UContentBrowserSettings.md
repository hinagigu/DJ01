# UContentBrowserSettings

**继承自**: `UObject`

Implements content browser settings.  These are global not per-project

## 属性

### NumObjectsToLoadBeforeWarning
- **类型**: `int`
- **描述**: The number of objects to load at once in the Content Browser before displaying a warning about loading many assets

### bDisplayContentFolderSuffix
- **类型**: `bool`
- **描述**: Whether to append 'Content' text to displayed folder names

### bDisplayFriendlyNameForPluginFolders
- **类型**: `bool`
- **描述**: Whether display friendly name as plugin folder names

### bEnableRealtimeMaterialInstanceThumbnails
- **类型**: `bool`
- **描述**: Enables the rendering of Material Instance thumbnail previews
