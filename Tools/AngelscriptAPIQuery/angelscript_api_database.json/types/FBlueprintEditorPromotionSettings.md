# FBlueprintEditorPromotionSettings

Holds settings for the blueprint editor build promotion tests

## 属性

### FirstMeshPath
- **类型**: `FFilePath`
- **描述**: The starting mesh for the blueprint *

### SecondMeshPath
- **类型**: `FFilePath`
- **描述**: The mesh to set on the blueprint after the delay *

### DefaultParticleAsset
- **类型**: `FFilePath`
- **描述**: Default particle asset to use for tests
