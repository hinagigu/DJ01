# __ESplineBoneAxis

The different axes we can align our bones to.
Note that the values are set to match up with EAxis (but without 'None')

## 属性

### None
- **类型**: `ESplineBoneAxis`

### X
- **类型**: `ESplineBoneAxis`

### Y
- **类型**: `ESplineBoneAxis`

### Z
- **类型**: `ESplineBoneAxis`

### ESplineBoneAxis_MAX
- **类型**: `ESplineBoneAxis`
