# FRigUnit_OffsetTransformForItem

Offset Transform is used to add an offset to an existing transform in the hierarchy. The offset is post multiplied.

## 属性

### Item
- **类型**: `FRigElementKey`

### OffsetTransform
- **类型**: `FTransform`

### Weight
- **类型**: `float32`

### bPropagateToChildren
- **类型**: `bool`

### ExecuteContext
- **类型**: `FControlRigExecuteContext`
