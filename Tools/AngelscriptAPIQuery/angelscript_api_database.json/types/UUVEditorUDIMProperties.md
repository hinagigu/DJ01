# UUVEditorUDIMProperties

**继承自**: `UInteractiveToolPropertySet`

Settings for UDIMs in the UVEditor

## 属性

### UDIMSourceAsset
- **类型**: `FString`

### UDIMSourceTexture
- **类型**: `UTexture2D`
- **描述**: Texture asset to source UDIM information from

### ActiveUDIMs
- **类型**: `TArray<FUDIMSpecifier>`
- **描述**: Currently active UDIM set
