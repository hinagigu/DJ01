# FRigUnit_HasMetadataTagArray

Returns true if a given item in the hierarchy has all of the provided tags

## 属性

### Item
- **类型**: `FRigElementKey`

### Tags
- **类型**: `TArray<FName>`

### NameSpace
- **类型**: `ERigMetaDataNameSpace`

### Found
- **类型**: `bool`
- **描述**: True if the item has the metadata
