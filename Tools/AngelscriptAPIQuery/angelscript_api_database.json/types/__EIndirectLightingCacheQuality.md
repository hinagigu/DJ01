# __EIndirectLightingCacheQuality

Quality of indirect lighting for Movable primitives. This has a large effect on Indirect Lighting Cache update time.

## 属性

### ILCQ_Off
- **类型**: `EIndirectLightingCacheQuality`

### ILCQ_Point
- **类型**: `EIndirectLightingCacheQuality`

### ILCQ_Volume
- **类型**: `EIndirectLightingCacheQuality`

### ILCQ_MAX
- **类型**: `EIndirectLightingCacheQuality`
