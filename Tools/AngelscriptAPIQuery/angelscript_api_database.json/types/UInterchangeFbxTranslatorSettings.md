# UInterchangeFbxTranslatorSettings

**继承自**: `UInterchangeTranslatorSettings`

Fbx translator class support import of texture, material, static mesh, skeletal mesh,

## 属性

### bConvertScene
- **类型**: `bool`
- **描述**: Whether to convert FBX scene axis system to Unreal axis system.

### bForceFrontXAxis
- **类型**: `bool`
- **描述**: Whether to force the front axis to be align with X instead of -Y default.

### bConvertSceneUnit
- **类型**: `bool`
- **描述**: Whether to convert the scene from FBX unit to UE unit (centimeter).
