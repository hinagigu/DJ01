# __EGameplayCueEvent

Indicates what type of action happened to a specific gameplay cue tag. Sometimes you will get multiple events at once

## 属性

### OnActive
- **类型**: `EGameplayCueEvent`

### WhileActive
- **类型**: `EGameplayCueEvent`

### Executed
- **类型**: `EGameplayCueEvent`

### Removed
- **类型**: `EGameplayCueEvent`

### EGameplayCueEvent_MAX
- **类型**: `EGameplayCueEvent`
