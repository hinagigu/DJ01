# FRigUnit_CurveExists

CurveExists is used to check whether a curve exists or not.

## 属性

### Curve
- **类型**: `FName`

### Exists
- **类型**: `bool`
- **描述**: Boolean indicating whether the named curve exists or not.
Does not indicate whether the curve's value is valid or not.
