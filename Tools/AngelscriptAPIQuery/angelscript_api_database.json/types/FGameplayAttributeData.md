# FGameplayAttributeData

Place in an AttributeSet to create an attribute that can be accesed using FGameplayAttribute. It is strongly encouraged to use this instead of raw float attributes

## 属性

### BaseValue
- **类型**: `float32`

### CurrentValue
- **类型**: `float32`
