# FTableColumnHeaderStyle

Represents the appearance of an STableColumnHeader

## 属性

### SortPrimaryAscendingImage
- **类型**: `FSlateBrush`
- **描述**: Image used when a column is primarily sorted in ascending order

### SortPrimaryDescendingImage
- **类型**: `FSlateBrush`
- **描述**: Image used when a column is primarily sorted in descending order

### SortSecondaryAscendingImage
- **类型**: `FSlateBrush`
- **描述**: Image used when a column is secondarily sorted in ascending order

### SortSecondaryDescendingImage
- **类型**: `FSlateBrush`
- **描述**: Image used when a column is secondarily sorted in descending order

### NormalBrush
- **类型**: `FSlateBrush`
- **描述**: Brush used to draw the header in its normal state

### HoveredBrush
- **类型**: `FSlateBrush`
- **描述**: Brush used to draw the header in its hovered state

### MenuDropdownImage
- **类型**: `FSlateBrush`
- **描述**: Image used for the menu drop-down button

### MenuDropdownNormalBorderBrush
- **类型**: `FSlateBrush`
- **描述**: Brush used to draw the menu drop-down border in its normal state

### MenuDropdownHoveredBorderBrush
- **类型**: `FSlateBrush`
- **描述**: Brush used to draw the menu drop-down border in its hovered state
