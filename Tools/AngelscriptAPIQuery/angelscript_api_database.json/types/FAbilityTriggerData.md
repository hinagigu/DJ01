# FAbilityTriggerData

Structure that defines how an ability will be triggered by external events

## 属性

### TriggerTag
- **类型**: `FGameplayTag`
- **描述**: The tag to respond to

### TriggerSource
- **类型**: `EGameplayAbilityTriggerSource`
- **描述**: The type of trigger to respond to
