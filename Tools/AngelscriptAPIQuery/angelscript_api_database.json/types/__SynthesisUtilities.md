# __SynthesisUtilities

## 方法

### GetLinearFrequency
```angelscript
float32 GetLinearFrequency(float32 InLogFrequencyValue, float32 InDomainMin, float32 InDomainMax, float32 InRangeMin, float32 InRangeMax)
```
Returns the log frequency of the input value. Maps linear domain and range values to log output (good for linear slider controlling frequency)
