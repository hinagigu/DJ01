# UNiagaraParameterDefinitions

**继承自**: `UNiagaraParameterDefinitionsBase`

Collection of UNiagaraScriptVariables to synchronize between UNiagaraScripts.

## 属性

### bPromoteToTopInAddMenus
- **类型**: `bool`
- **描述**: If true then these parameters will appear as top level entry in add menus (e.g. in the module editor)

### MenuSortOrder
- **类型**: `int`
- **描述**: Defines the sort order in add menus. Entries with smaller numbers are displayed first.
