# UModelingComponentsEditorSettings

**继承自**: `UDeveloperSettings`

Editor preferences for the Modeling Components plug-in.

## 属性

### GridMode
- **类型**: `EModelingComponentsPlaneVisualizationMode`
- **描述**: The type of grid to draw in the viewport for modeling mode tools

### NumGridLines
- **类型**: `int`
- **描述**: The number of lines to be drawn for the plane

### GridSpacing
- **类型**: `float32`
- **描述**: The space between grid lines in world space

### GridScale
- **类型**: `float32`
- **描述**: The base scale used to determine the size of the hierarchical plane

### GridSize
- **类型**: `float32`
- **描述**: The fraction of the viewport that the grid should occupy if looking at the plane's center
