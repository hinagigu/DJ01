# UMeshConstraintProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### bPreserveSharpEdges
- **类型**: `bool`
- **描述**: If true, sharp edges are preserved

### MeshBoundaryConstraint
- **类型**: `EMeshBoundaryConstraint`
- **描述**: Mesh Boundary Constraint Type

### GroupBoundaryConstraint
- **类型**: `EGroupBoundaryConstraint`
- **描述**: Group Boundary Constraint Type

### MaterialBoundaryConstraint
- **类型**: `EMaterialBoundaryConstraint`
- **描述**: Material Boundary Constraint Type

### bPreventNormalFlips
- **类型**: `bool`
- **描述**: Prevent normal flips

### bPreventTinyTriangles
- **类型**: `bool`
- **描述**: Prevent introduction of tiny triangles or slivers
