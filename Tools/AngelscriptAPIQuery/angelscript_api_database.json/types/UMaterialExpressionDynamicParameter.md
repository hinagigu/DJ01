# UMaterialExpressionDynamicParameter

**继承自**: `UMaterialExpression`

## 属性

### ParamNames
- **类型**: `TArray<FString>`
- **描述**: The names of the parameters.
These will show up in Cascade when editing a particle system
that uses the material it is in...

### DefaultValue
- **类型**: `FLinearColor`

### ParameterIndex
- **类型**: `uint`
- **描述**: The index of the dynamic parameter for use in tools that allow > 1
