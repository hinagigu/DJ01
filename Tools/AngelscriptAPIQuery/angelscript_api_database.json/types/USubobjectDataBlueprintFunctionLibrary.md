# USubobjectDataBlueprintFunctionLibrary

**继承自**: `UBlueprintFunctionLibrary`

A function library with wrappers around the getter/setter functions for FSubobjectData
that will make it easier to use within blueprint contexts.

## 方法

### GetBlueprint
```angelscript
UBlueprint GetBlueprint(FSubobjectData Data)
```

### GetDisplayName
```angelscript
FText GetDisplayName(FSubobjectData Data)
```
