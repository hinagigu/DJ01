# UMeshColorPaintingToolProperties

**继承自**: `UMeshVertexPaintingToolProperties`

## 属性

### bWriteRed
- **类型**: `bool`
- **描述**: Whether or not to apply Vertex Color Painting to the Red Channel

### bWriteGreen
- **类型**: `bool`
- **描述**: Whether or not to apply Vertex Color Painting to the Green Channel

### bWriteBlue
- **类型**: `bool`
- **描述**: Whether or not to apply Vertex Color Painting to the Blue Channel

### bWriteAlpha
- **类型**: `bool`
- **描述**: Whether or not to apply Vertex Color Painting to the Alpha Channel

### bPaintOnSpecificLOD
- **类型**: `bool`
- **描述**: When unchecked the painting on the base LOD will be propagate automatically to all other LODs when exiting the mode or changing the selection

### LODIndex
- **类型**: `int`
- **描述**: LOD Index to which should specifically be painted
