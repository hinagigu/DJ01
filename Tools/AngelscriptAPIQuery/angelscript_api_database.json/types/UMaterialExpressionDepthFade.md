# UMaterialExpressionDepthFade

**继承自**: `UMaterialExpression`

## 属性

### OpacityDefault
- **类型**: `float32`
- **描述**: Opacity which will be scaled by the result of the fade.  This is used when InOpacity is unconnected.

### FadeDistanceDefault
- **类型**: `float32`
- **描述**: World space distance over which the fade should take place.  This is used when FadeDistance is unconnected.
