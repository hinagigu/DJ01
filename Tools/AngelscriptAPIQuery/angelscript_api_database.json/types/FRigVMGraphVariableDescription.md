# FRigVMGraphVariableDescription

The variable description is used to convey information
about unique variables within a Graph. Multiple Variable
Nodes can share the same variable description.

## 属性

### Name
- **类型**: `FName`

### CPPType
- **类型**: `FString`

### CPPTypeObject
- **类型**: `UObject`

### DefaultValue
- **类型**: `FString`
