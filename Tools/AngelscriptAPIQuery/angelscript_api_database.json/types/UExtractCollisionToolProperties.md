# UExtractCollisionToolProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### CollisionType
- **类型**: `EExtractCollisionOutputType`
- **描述**: Type of collision geometry to convert to Mesh

### bOutputSeparateMeshes
- **类型**: `bool`
- **描述**: Whether or not to generate a seperate Mesh Object for each Simple Collision Shape

### bShowPreview
- **类型**: `bool`
- **描述**: Show/Hide a preview of the generated mesh (overlaps source mesh)

### bShowInputMesh
- **类型**: `bool`
- **描述**: Show/Hide input mesh

### bWeldEdges
- **类型**: `bool`
- **描述**: Whether or not to weld coincident border edges of the Complex Collision Mesh (if possible)
