# FRigUnit_AddParent

Adds a new parent to an element. The weight for the new parent will be 0.0.
You can use the SetParentWeights node to change the parent weights later.

## 属性

### Child
- **类型**: `FRigElementKey`

### Parent
- **类型**: `FRigElementKey`

### ExecuteContext
- **类型**: `FControlRigExecuteContext`
