# UMovieScene3DTransformSection

**继承自**: `UMovieSceneSection`

A 3D transform section

## 属性

### bUseQuaternionInterpolation
- **类型**: `bool`
- **描述**: Whether to use a quaternion linear interpolation between keys. This finds the 'shortest' rotation between keyed orientations.

### Show3DTrajectory
- **类型**: `EShow3DTrajectory`
- **描述**: Whether to show the 3d trajectory
