# UEnhancedInputPlatformSettings

**继承自**: `UPlatformSettings`

Settings for Enhanced Input that can be set per-platform

## 属性

### InputData
- **类型**: `TArray<TSoftClassPtr<UEnhancedInputPlatformData>>`
- **描述**: Input data that can be populated with Enhanced Input Platform Data blueprints

### bShouldLogMappingContextRedirects
- **类型**: `bool`
- **描述**: If true, then Enhanced Input will log which mapping contexts have been redirected when building the control mappings.
