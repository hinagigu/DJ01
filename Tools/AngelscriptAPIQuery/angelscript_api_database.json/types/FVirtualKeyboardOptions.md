# FVirtualKeyboardOptions

## 属性

### bEnableAutocorrect
- **类型**: `bool`
- **描述**: Enables autocorrect for this widget, if supported by the platform's virtual keyboard. Autocorrect must also be enabled in Input settings for this to take effect.
