# __EGraphType

Distinguishes between different graph types. Graphs can have different properties; for example: functions have one entry point, ubergraphs can have multiples.

## 属性

### GT_Function
- **类型**: `EGraphType`

### GT_Ubergraph
- **类型**: `EGraphType`

### GT_Macro
- **类型**: `EGraphType`

### GT_Animation
- **类型**: `EGraphType`

### GT_StateMachine
- **类型**: `EGraphType`

### GT_MAX
- **类型**: `EGraphType`
