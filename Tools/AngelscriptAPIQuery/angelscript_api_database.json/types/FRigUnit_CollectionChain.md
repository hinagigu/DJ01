# FRigUnit_CollectionChain

Creates a collection based on a first and last item within a chain.
Chains can refer to bone chains or chains within a control hierarchy.

## 属性

### FirstItem
- **类型**: `FRigElementKey`

### LastItem
- **类型**: `FRigElementKey`

### Reverse
- **类型**: `bool`

### Collection
- **类型**: `FRigElementKeyCollection`
