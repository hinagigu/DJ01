# FNiagaraOutlinerViewSettings

View settings used in the Niagara Outliner.

## 属性

### ViewMode
- **类型**: `ENiagaraOutlinerViewModes`

### FilterSettings
- **类型**: `FNiagaraOutlinerFilterSettings`

### bSortDescending
- **类型**: `bool`
- **描述**: Whether to sort ascending or descending.

### SortMode
- **类型**: `ENiagaraOutlinerSortMode`

### TimeUnits
- **类型**: `ENiagaraOutlinerTimeUnits`
- **描述**: Units used to display time data in performance view mode.
