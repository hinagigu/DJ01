# UWorldPartitionEditorSettings

**继承自**: `UDeveloperSettings`

## 属性

### CommandletClass
- **类型**: `TSubclassOf<UWorldPartitionConvertCommandlet>`
- **描述**: Commandlet class to use for World Partition conversion

### InstancedFoliageGridSize
- **类型**: `int`
- **描述**: Editor grid size used for instance foliage actors in World Partition worlds

### MinimapLowQualityWorldUnitsPerPixelThreshold
- **类型**: `int`
- **描述**: Threshold from which minimap generates a warning if its WorldUnitsPerPixel is above this value

### bEnableLoadingInEditor
- **类型**: `bool`
- **描述**: Whether to enable dynamic loading in the editor through loading regions

### bEnableStreamingGenerationLogOnPIE
- **类型**: `bool`
- **描述**: Whether to enable streaming generation log on PIE

### bShowHLODsInEditor
- **类型**: `bool`
- **描述**: Whether to show HLODs in the editor

### bShowHLODsOverLoadedRegions
- **类型**: `bool`
- **描述**: Control display of HLODs in case actors are loaded

### HLODMinDrawDistance
- **类型**: `float`
- **描述**: Minimum distance at which HLODs should be displayed in editor

### HLODMaxDrawDistance
- **类型**: `float`
- **描述**: Maximum distance at which HLODs should be displayed in editor
