# UAssetTagsGameplayEffectComponent

**继承自**: `UGameplayEffectComponent`

These are tags that the Gameplay Effect Asset itself 'has' (owns). These do _not_ transfer to any Actors

## 属性

### InheritableAssetTags
- **类型**: `FInheritedTagContainer`
- **描述**: The GameplayEffect's Tags: tags the the GE *has* and DOES NOT give to the actor.
