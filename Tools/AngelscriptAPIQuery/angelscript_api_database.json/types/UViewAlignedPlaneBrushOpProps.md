# UViewAlignedPlaneBrushOpProps

**继承自**: `UBasePlaneBrushOpProps`

## 属性

### Strength
- **类型**: `float32`
- **描述**: Strength of the Brush

### Falloff
- **类型**: `float32`
- **描述**: Amount of falloff to apply

### Depth
- **类型**: `float32`
- **描述**: Depth of Brush into surface along view ray

### WhichSide
- **类型**: `EPlaneBrushSideMode`
- **描述**: Control whether effect of brush should be limited to one side of the Plane
