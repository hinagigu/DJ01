# FInterpCurveVector

Describes an entire curve that is used to compute a 3D vector output value from a float input.
@note This is a mirror of TInterpCurve<FVector>, defined in InterpCurve.h

## 属性

### Points
- **类型**: `TArray<FInterpCurvePointVector>`

### bIsLooped
- **类型**: `bool`

### LoopKeyOffset
- **类型**: `float32`
