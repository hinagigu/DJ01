# FExpandableAreaStyle

Represents the appearance of an SExpandableArea

## 属性

### CollapsedImage
- **类型**: `FSlateBrush`
- **描述**: Image to use when the area is collapsed

### ExpandedImage
- **类型**: `FSlateBrush`
- **描述**: Image to use when the area is expanded

### RolloutAnimationSeconds
- **类型**: `float32`
- **描述**: How long the rollout animation lasts
