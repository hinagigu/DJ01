# UCameraModifier_CameraShake

**继承自**: `UCameraModifier`

A UCameraModifier_CameraShake is a camera modifier that can apply a UCameraShakeBase to
the owning camera.

## 属性

### SplitScreenShakeScale
- **类型**: `float32`
- **描述**: Scaling factor applied to all camera shakes in when in splitscreen mode. Normally used to reduce shaking, since shakes feel more intense in a smaller viewport.
