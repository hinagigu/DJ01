# UBasicOverlays

**继承自**: `UOverlays`

Implements an asset that contains a set of overlay data (which includes timing, text, and position) to be displayed for any
given source (including, but not limited to, audio, dialog, and movies)

## 属性

### Overlays
- **类型**: `TArray<FOverlayItem>`
- **描述**: The overlay data held by this asset. Contains info on timing, position, and the subtitle to display

### AssetImportData
- **类型**: `UAssetImportData`
- **描述**: The import data used to make this overlays asset
