# UMaterialExpressionFresnel

**继承自**: `UMaterialExpression`

## 属性

### Exponent
- **类型**: `float32`
- **描述**: The exponent to pass into the pow() function

### BaseReflectFraction
- **类型**: `float32`
- **描述**: Specifies the fraction of specular reflection when the surfaces is viewed from straight on.
A value of 1 effectively disables Fresnel.
