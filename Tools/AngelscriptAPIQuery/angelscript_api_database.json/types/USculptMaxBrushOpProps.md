# USculptMaxBrushOpProps

**继承自**: `UMeshSculptBrushOpProps`

## 属性

### Strength
- **类型**: `float32`
- **描述**: Strength of the Brush

### Falloff
- **类型**: `float32`
- **描述**: Amount of falloff to apply

### MaxHeight
- **类型**: `float32`
- **描述**: Maximum height as fraction of brush size

### bUseFixedHeight
- **类型**: `bool`
- **描述**: If true, maximum height is defined using the FixedHeight constant instead of brush-relative size

### FixedHeight
- **类型**: `float32`
- **描述**: Maximum height in world-space dimension
