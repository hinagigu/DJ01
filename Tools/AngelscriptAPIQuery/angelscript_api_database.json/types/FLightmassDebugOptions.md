# FLightmassDebugOptions

Debug options for Lightmass

## 属性

### CoplanarTolerance
- **类型**: `float32`
- **描述**: The tolerance level used when gathering BSP surfaces.

### ExecutionTimeDivisor
- **类型**: `float32`
- **描述**: The amount of time that will be count as full red when bColorByExecutionTime is enabled

### bDebugMode
- **类型**: `bool`

### bStatsEnabled
- **类型**: `bool`

### bGatherBSPSurfacesAcrossComponents

### bUseImmediateImport

### bColorByExecutionTime
- **类型**: `bool`
