# __EButtonPressMethod

Enumerates different methods that a button can be triggered with keyboard/controller. Normally, DownAndUp is appropriate.

## 属性

### DownAndUp
- **类型**: `EButtonPressMethod`

### ButtonPress
- **类型**: `EButtonPressMethod`

### ButtonRelease
- **类型**: `EButtonPressMethod`

### EButtonPressMethod_MAX
- **类型**: `EButtonPressMethod`
