# FRigUnit_GetControlTransform

GetControlTransform is used to retrieve a single transform from a hierarchy.

## 属性

### Control
- **类型**: `FName`

### Space
- **类型**: `ERigVMTransformSpace`

### Transform
- **类型**: `FTransform`
- **描述**: The current value of the control.

### Minimum
- **类型**: `FTransform`
- **描述**: The minimum value of the control.

### Maximum
- **类型**: `FTransform`
- **描述**: The maximum value of the control.
