# FVolumeControlStyle

Represents the appearance of an SVolumeControl

## 属性

### SliderStyle
- **类型**: `FSliderStyle`
- **描述**: The style of the volume control slider

### HighVolumeImage
- **类型**: `FSlateBrush`
- **描述**: Image to use when the volume is set to high

### MidVolumeImage
- **类型**: `FSlateBrush`
- **描述**: Image to use when the volume is set to mid-range

### LowVolumeImage
- **类型**: `FSlateBrush`
- **描述**: Image to use when the volume is set to low

### NoVolumeImage
- **类型**: `FSlateBrush`
- **描述**: Image to use when the volume is set to off

### MutedImage
- **类型**: `FSlateBrush`
- **描述**: Image to use when the volume is muted

FVolumeControlStyle& opAssign(FVolumeControlStyle Other)
