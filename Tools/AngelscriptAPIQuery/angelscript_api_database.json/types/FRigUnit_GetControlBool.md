# FRigUnit_GetControlBool

GetControlBool is used to retrieve a single Bool from a hierarchy.

## 属性

### Control
- **类型**: `FName`

### BoolValue
- **类型**: `bool`
- **描述**: The current bool of the given bone - or identity in case it wasn't found.

### opAssign
