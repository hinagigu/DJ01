# UDJ01GameData

**继承自**: `UPrimaryDataAsset`

UDJ01GameData

     Non-mutable data asset that contains global game data.

## 属性

### DamageGameplayEffect_SetByCaller
- **类型**: `TSoftClassPtr<UGameplayEffect>`
- **描述**: Gameplay effect used to apply damage.  Uses SetByCaller for the damage magnitude.

### HealGameplayEffect_SetByCaller
- **类型**: `TSoftClassPtr<UGameplayEffect>`
- **描述**: Gameplay effect used to apply healing.  Uses SetByCaller for the healing magnitude.

### DynamicTagGameplayEffect
- **类型**: `TSoftClassPtr<UGameplayEffect>`
- **描述**: Gameplay effect used to add and remove dynamic tags.
