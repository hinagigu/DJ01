# FRigUnit_ShapeExists

Checks whether or not a shape is available in the rig's shape libraries

## 属性

### ShapeName
- **类型**: `FName`

### Result
- **类型**: `bool`
- **描述**: * True if the shape name exists in any of the shape libraries
