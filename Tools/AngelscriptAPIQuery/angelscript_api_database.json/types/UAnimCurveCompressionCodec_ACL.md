# UAnimCurveCompressionCodec_ACL

**继承自**: `UAnimCurveCompressionCodec`

Uses the open source Animation Compression Library with default settings suitable for general purpose animation curves.

## 属性

### CurvePrecision
- **类型**: `float32`
- **描述**: The curve precision to target when compressing the animation curves.

### MorphTargetPositionPrecision
- **类型**: `float32`
- **描述**: The mesh deformation precision to target when compressing morph target animation curves.

### MorphTargetSource
- **类型**: `USkeletalMesh`
- **描述**: The skeletal mesh used to estimate the morph target deformation during compression.
