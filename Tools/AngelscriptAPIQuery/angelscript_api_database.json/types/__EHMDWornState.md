# __EHMDWornState

Stores if the user is wearing the HMD or not. For HMDs without a sensor to detect the user wearing it, the state defaults to Unknown.

## 属性

### Unknown
- **类型**: `EHMDWornState`

### Worn
- **类型**: `EHMDWornState`

### NotWorn
- **类型**: `EHMDWornState`

### EHMDWornState_MAX
- **类型**: `EHMDWornState`
