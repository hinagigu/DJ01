# FChaosDebugDrawColorsByState

Structure holding the settings using to debug draw Particles shape based on their state on the Chaos Visual Debugger

## 属性

### DynamicColor
- **类型**: `FColor`
- **描述**: Color used for dynamic particles

### SleepingColor
- **类型**: `FColor`
- **描述**: Color used for sleeping particles

### KinematicColor
- **类型**: `FColor`
- **描述**: Color used for kinematic particles

### StaticColor
- **类型**: `FColor`
- **描述**: Color used for static particles
