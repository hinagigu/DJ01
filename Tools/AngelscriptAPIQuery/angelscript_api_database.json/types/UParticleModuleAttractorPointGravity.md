# UParticleModuleAttractorPointGravity

**继承自**: `UParticleModuleAttractorBase`

## 属性

### Position
- **类型**: `FVector`
- **描述**: The position of the point gravity source.

### Radius
- **类型**: `float32`
- **描述**: The distance at which the influence of the point begins to falloff.

### StrengthRaw
- **类型**: `FRawDistributionFloat`
- **描述**: The strength of the point source.
