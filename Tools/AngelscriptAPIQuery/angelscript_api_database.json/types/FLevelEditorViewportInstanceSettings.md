# FLevelEditorViewportInstanceSettings

Implements the Level Editor's per-instance view port settings.

## 属性

### bShowFullToolbar
- **类型**: `bool`
- **描述**: When enabled, the full viewport toolbar will be shown. When disabled, a compact toolbar is used.
