# FPerlinLayerProperties

Per-layer properties for Perlin noise. Each layer has independent Frequency and Intensity.

## 属性

### Frequency
- **类型**: `float32`
- **描述**: Frequency of Perlin noise layer

### Intensity
- **类型**: `float32`
- **描述**: Intensity/amplitude of Perlin noise layer
