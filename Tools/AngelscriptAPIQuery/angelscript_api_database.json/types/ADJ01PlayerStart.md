# ADJ01PlayerStart

**继承自**: `APlayerStart`

ADJ01PlayerStart

Base player starts that can be used by a lot of modes.

## 属性

### ExpirationCheckInterval
- **类型**: `float32`
- **描述**: Interval in which we'll check if this player start is not colliding with anyone anymore

### StartPointTags
- **类型**: `FGameplayTagContainer`
- **描述**: Tags to identify this player start
