# UISMEditorToolProperties

**继承自**: `UInteractiveToolPropertySet`

Standard properties of the Transform Meshes operation

## 属性

### TransformMode
- **类型**: `EISMEditorTransformMode`
- **描述**: Transformation Mode controls the overall behavior of the Gizmos in the Tool

### bSetPivotMode
- **类型**: `bool`
- **描述**: When true, the Gizmo can be moved independently without affecting objects. This allows the Gizmo to be repositioned before transforming.

### bShowSelectable
- **类型**: `bool`
- **描述**: Show a highlight around all selectable instances

### bShowSelected
- **类型**: `bool`
- **描述**: Show a highlight around the selected instances

### bHideWhenDragging
- **类型**: `bool`
- **描述**: Hide the Selectable and Selected highlights when dragging with the Gizmos
