# FAddCustomCollectionAttributeDataflowNode

Adds custom attribute to Collection

## 属性

### GroupName
- **类型**: `EStandardGroupNameEnum`
- **描述**: Standard group names

### CustomGroupName
- **类型**: `FString`
- **描述**: User specified group name

### AttrName
- **类型**: `FString`
- **描述**: Attribute name

### CustomAttributeType
- **类型**: `ECustomAttributeTypeEnum`
- **描述**: Attribute type

### NumElements
- **类型**: `int`
- **描述**: Number of elements for the attribute

### bActive
- **类型**: `bool`
