# UMaterialExpressionFunctionOutput

**继承自**: `UMaterialExpression`

## 属性

### OutputName
- **类型**: `FName`

### Description
- **类型**: `FString`
- **描述**: The output's description, which will be used as a tooltip on the connector in function call expressions that use this function.

### SortPriority
- **类型**: `int`
- **描述**: Controls where the output is displayed relative to the other outputs.
