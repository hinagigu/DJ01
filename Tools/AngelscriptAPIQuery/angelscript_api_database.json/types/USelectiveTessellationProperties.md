# USelectiveTessellationProperties

**继承自**: `UInteractiveToolPropertySet`

PropertySet for properties affecting the selective tessellation.

## 属性

### SelectionType
- **类型**: `EDisplaceMeshToolTriangleSelectionType`
- **描述**: Optionally, restrict tessellation to a subset of the triangles.

### ActiveMaterial
- **类型**: `FName`
- **描述**: Name of the selected material.
