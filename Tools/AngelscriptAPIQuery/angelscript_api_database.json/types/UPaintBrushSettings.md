# UPaintBrushSettings

**继承自**: `UObject`

## 属性

### BrushRadius
- **类型**: `float32`
- **描述**: Radius of the Brush used for Painting

### BrushStrength
- **类型**: `float32`
- **描述**: Strength of the brush (0.0 - 1.0)

### BrushFalloffAmount
- **类型**: `float32`
- **描述**: Amount of falloff to apply (0.0 - 1.0)

### bEnableFlow
- **类型**: `bool`
- **描述**: Enables "Flow" painting where paint is continually applied from the brush every tick

### bOnlyFrontFacingTriangles
- **类型**: `bool`
- **描述**: Whether back-facing triangles should be ignored

### ColorViewMode
- **类型**: `EMeshPaintColorViewMode`
- **描述**: Color view mode used to display Vertex Colors
