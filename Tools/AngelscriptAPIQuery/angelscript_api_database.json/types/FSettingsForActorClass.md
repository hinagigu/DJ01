# FSettingsForActorClass

## 属性

### Class
- **类型**: `TSubclassOf<AActor>`
- **描述**: The class of the actor we want to record

### bRecordToPossessable
- **类型**: `bool`
- **描述**: Whether to record to 'possessable' (i.e. level-owned) or 'spawnable' (i.e. sequence-owned) actors.
