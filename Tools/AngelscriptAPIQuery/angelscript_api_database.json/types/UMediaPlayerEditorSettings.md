# UMediaPlayerEditorSettings

**继承自**: `UObject`

## 属性

### DesiredPlayerName
- **类型**: `FName`
- **描述**: The name of the desired native media player to use for playback.

### ShowTextOverlays
- **类型**: `bool`
- **描述**: Whether to display overlay texts.

### ViewportScale
- **类型**: `EMediaPlayerEditorScale`
- **描述**: How the video viewport should be scaled.
