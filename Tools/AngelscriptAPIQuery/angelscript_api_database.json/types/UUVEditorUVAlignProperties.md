# UUVEditorUVAlignProperties

**继承自**: `UUVEditorUVTransformPropertiesBase`

UV Align Settings

## 属性

### AlignAnchor
- **类型**: `EUVEditorAlignAnchor`
- **描述**: Controls what geometry the alignment is to be relative to when performed.

### ManualAnchor
- **类型**: `FVector2D`
- **描述**: Manual anchor location for relative alignment

### AlignDirection
- **类型**: `EUVEditorAlignDirection`
- **描述**: Controls what side of the island bounding boxes are being aligned

### Grouping
- **类型**: `EUVEditorAlignDistributeGroupingMode`
- **描述**: Controls how alignment considers grouping selected objects with respect to the alignment behavior.
