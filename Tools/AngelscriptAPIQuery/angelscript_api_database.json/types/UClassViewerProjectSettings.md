# UClassViewerProjectSettings

**继承自**: `UObject`

Implements the settings for the Class Viewer Project Settings

## 属性

### InternalOnlyPaths
- **类型**: `TArray<FDirectoryPath>`
- **描述**: The base directories to be considered Internal Only for the class picker.

### InternalOnlyClasses
- **类型**: `TArray<FSoftClassPath>`
- **描述**: The base classes to be considered Internal Only for the class picker.
