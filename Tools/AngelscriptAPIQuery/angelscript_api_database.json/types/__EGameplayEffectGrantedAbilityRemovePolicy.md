# __EGameplayEffectGrantedAbilityRemovePolicy

Describes what happens when a GameplayEffect, that is granting an active ability, is removed from its owner.

## 属性

### CancelAbilityImmediately
- **类型**: `EGameplayEffectGrantedAbilityRemovePolicy`

### RemoveAbilityOnEnd
- **类型**: `EGameplayEffectGrantedAbilityRemovePolicy`

### DoNothing
- **类型**: `EGameplayEffectGrantedAbilityRemovePolicy`

### EGameplayEffectGrantedAbilityRemovePolicy_MAX
- **类型**: `EGameplayEffectGrantedAbilityRemovePolicy`
