# UDataRegistrySource_DataTable

**继承自**: `UDataRegistrySource`

Data source that loads from a specific data table containing the same type of structs as the registry

## 属性

### SourceTable
- **类型**: `TSoftObjectPtr<UDataTable>`
- **描述**: What table to load from

### TableRules
- **类型**: `FDataRegistrySource_DataTableRules`
- **描述**: Access rules
