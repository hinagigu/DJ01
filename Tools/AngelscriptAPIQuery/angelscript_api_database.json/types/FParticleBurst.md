# FParticleBurst

## 属性

### Count
- **类型**: `int`
- **描述**: The number of particles to burst

### CountLow
- **类型**: `int`
- **描述**: If >= 0, use as a range [CountLow..Count]

### Time
- **类型**: `float32`
- **描述**: The time at which to burst them (0..1: emitter lifetime)
