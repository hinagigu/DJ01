# FRigUnit_SetControlVisibility

SetControlVisibility is used to change the visibility on a control at runtime

## 属性

### Item
- **类型**: `FRigElementKey`

### Pattern
- **类型**: `FString`
- **描述**: If the ControlName is set to None this can be used to look for a series of Controls

### bVisible
- **类型**: `bool`

### ExecuteContext
- **类型**: `FControlRigExecuteContext`

## 方法

### opAssign
```angelscript
FRigUnit_SetControlVisibility& opAssign(FRigUnit_SetControlVisibility Other)
```
