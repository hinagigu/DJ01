# FBeamTargetData

Beam Multi-target Variables.

## 属性

### TargetName
- **类型**: `FName`
- **描述**: Name of the target.

### TargetPercentage
- **类型**: `float32`
- **描述**: Percentage chance the target will be selected (100 = always).
