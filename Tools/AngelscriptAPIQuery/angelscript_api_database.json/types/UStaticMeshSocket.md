# UStaticMeshSocket

**继承自**: `UObject`

## 属性

### SocketName
- **类型**: `FName`
- **描述**: Defines a named attachment location on the UStaticMesh.
These are set up in editor and used as a shortcut instead of specifying
everything explicitly to AttachComponent in the StaticMeshComponent.
The Outer of a StaticMeshSocket should always be the UStaticMesh.

### RelativeLocation
- **类型**: `FVector`

### RelativeRotation
- **类型**: `FRotator`

### RelativeScale
- **类型**: `FVector`

### Tag
- **类型**: `FString`

### PreviewStaticMesh
- **类型**: `UStaticMesh`
