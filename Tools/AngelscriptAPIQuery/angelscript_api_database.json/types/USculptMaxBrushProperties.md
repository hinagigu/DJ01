# USculptMaxBrushProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### MaxHeight
- **类型**: `float32`
- **描述**: Specify maximum displacement height (relative to brush size)

### bFreezeCurrentHeight
- **类型**: `bool`
- **描述**: Use maximum height from last brush stroke, regardless of brush size. Note that spatial brush falloff still applies.
