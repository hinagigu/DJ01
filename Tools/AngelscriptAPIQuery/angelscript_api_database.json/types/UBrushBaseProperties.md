# UBrushBaseProperties

**继承自**: `UInteractiveToolPropertySet`

Standard properties for a Brush-type Tool

## 属性

### BrushSize
- **类型**: `float32`
- **描述**: Relative size of brush

### bSpecifyRadius
- **类型**: `bool`
- **描述**: If true, ignore relative Brush Size and use explicit world Radius

### BrushRadius
- **类型**: `float32`
- **描述**: Radius of brush

### BrushStrength
- **类型**: `float32`
- **描述**: Strength of the brush (0.0 - 1.0)

### BrushFalloffAmount
- **类型**: `float32`
- **描述**: Amount of falloff to apply (0.0 - 1.0)
