# __EActorUpdateOverlapsMethod

Chooses a method for actors to update overlap state (objects it is touching) on initialization, currently only used during level streaming.

## 属性

### UseConfigDefault
- **类型**: `EActorUpdateOverlapsMethod`

### AlwaysUpdate
- **类型**: `EActorUpdateOverlapsMethod`

### OnlyUpdateMovable
- **类型**: `EActorUpdateOverlapsMethod`

### NeverUpdate
- **类型**: `EActorUpdateOverlapsMethod`

### EActorUpdateOverlapsMethod_MAX
- **类型**: `EActorUpdateOverlapsMethod`
