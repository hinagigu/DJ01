# FDataflowOverrideNode

FDataflowOverrideNode
             Base class for override nodes within the Dataflow graph.

             Override Nodes allow to access to Override property on
             the asset. They can read the values by the key.

## 属性

### Key
- **类型**: `FName`

### Default
- **类型**: `FString`

### bActive
- **类型**: `bool`
