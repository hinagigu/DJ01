# UUVLayoutPreviewProperties

**继承自**: `UInteractiveToolPropertySet`

Visualization settings for UV layout preview

## 属性

### bEnabled
- **类型**: `bool`
- **描述**: Enable the preview UV layout

### Side
- **类型**: `EUVLayoutPreviewSide`
- **描述**: Which side of the selected object the preview UV layout is shown

### Scale
- **类型**: `float32`
- **描述**: Uniform scaling factor

### Offset
- **类型**: `FVector2D`
- **描述**: Offset relative to the center of the selected object

### bShowWireframe
- **类型**: `bool`
- **描述**: Show wireframe mesh in the preview UV layout
