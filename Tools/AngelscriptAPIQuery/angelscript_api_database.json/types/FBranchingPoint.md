# FBranchingPoint

Remove FBranchingPoint when VER_UE4_MONTAGE_BRANCHING_POINT_REMOVAL is removed.

## 属性

### EventName
- **类型**: `FName`

### SlotIndex
- **类型**: `int`
- **描述**: The slot index we are currently using within LinkedMontage

### LinkMethod
- **类型**: `EAnimLinkMethod`
- **描述**: The method we are using to calculate our times
