# UKelvinBrushProperties

**继承自**: `UInteractiveToolPropertySet`

## 属性

### FallOffDistance
- **类型**: `float32`
- **描述**: Brush Fall off as fraction of brush size

### Stiffness
- **类型**: `float32`
- **描述**: How much the mesh resists shear

### Incompressiblity
- **类型**: `float32`
- **描述**: How compressible the spatial region is: 1 - 2 x Poisson ratio

### BrushSteps
- **类型**: `int`
- **描述**: Integration steps
