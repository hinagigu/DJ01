# FGizmosParameters

## 属性

### RotateMode
- **类型**: `EAxisRotateMode`
- **描述**: Rotate mode used when manipulating axis rotation handles.

### bCtrlMiddleDoesY
- **类型**: `bool`
- **描述**: Property used to define whether the Y axis is indirectly controlled via CTRL+MMB or CTRL+RMB (and Z via CTRL+RMB or CTRL+LMB+RMB).
