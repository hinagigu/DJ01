# UPatternTool_RadialSettings

**继承自**: `UInteractiveToolPropertySet`

Settings for Radial Patterns in the Pattern Tool

## 属性

### SpacingMode
- **类型**: `EPatternToolAxisSpacingMode`
- **描述**: Spacing Technique used to distribute Pattern Elements around the Circle/Arc

### Count
- **类型**: `int`
- **描述**: Number of  Pattern Elements to place

### StepSize
- **类型**: `float`
- **描述**: Fixed Increment (in Degrees) used to position Pattern Elements around the Circle/Arc

### Radius
- **类型**: `float`
- **描述**: Radius of the Circle/Arc

### StartAngle
- **类型**: `float`
- **描述**: Start angle of the Circle/Arc

### EndAngle
- **类型**: `float`
- **描述**: End angle of the Circle/Arc

### AngleShift
- **类型**: `float`
- **描述**: Fixed offset added to Start/End Angles

### bOriented
- **类型**: `bool`
- **描述**: If true, Pattern elements are rotated to align with the Circle tangent
