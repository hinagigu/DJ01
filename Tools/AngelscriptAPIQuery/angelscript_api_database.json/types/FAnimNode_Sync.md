# FAnimNode_Sync

## 属性

### Source
- **类型**: `FPoseLink`

### GroupName
- **类型**: `FName`
- **描述**: The group name that we synchronize with (NAME_None if it is not part of any group).

### GroupRole
- **类型**: `EAnimGroupRole`
- **描述**: The role this animation can assume within the group (ignored if GroupName is not set)
