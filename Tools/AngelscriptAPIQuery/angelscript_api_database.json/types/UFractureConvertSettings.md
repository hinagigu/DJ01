# UFractureConvertSettings

**继承自**: `UFractureToolSettings`

Settings related to geometry collection -> static mesh conversion *

## 属性

### bPromptForBaseName
- **类型**: `bool`
- **描述**: Whether to prompt user for a location and base name for the generated meshes, or automatically place them next to the source geometry collections

### bPerBone
- **类型**: `bool`
- **描述**: Whether to generate a separate mesh for each bone, or one combined mesh

### bCenterPivots
- **类型**: `bool`
- **描述**: Whether to center the pivot for each mesh, or use the pivot from the geometry collection

### bPlaceInWorld
- **类型**: `bool`
- **描述**: Whether to place new static mesh actors in the level

### bSelectNewActors
- **类型**: `bool`
- **描述**: Whether to select new static mesh actors
