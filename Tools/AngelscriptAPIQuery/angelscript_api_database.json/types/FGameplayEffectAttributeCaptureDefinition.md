# FGameplayEffectAttributeCaptureDefinition

Struct defining gameplay attribute capture options for gameplay effects

## 属性

### AttributeToCapture
- **类型**: `FGameplayAttribute`
- **描述**: Gameplay attribute to capture

### AttributeSource
- **类型**: `EGameplayEffectAttributeCaptureSource`
- **描述**: Source of the gameplay attribute

### bSnapshot
- **类型**: `bool`
- **描述**: Whether the attribute should be snapshotted or not
