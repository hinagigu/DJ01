# UParticleModuleSizeScaleBySpeed

**继承自**: `UParticleModuleSizeBase`

## 属性

### SpeedScale
- **类型**: `FVector2D`
- **描述**: By how much speed affects the size of the particle in each dimension.

### MaxScale
- **类型**: `FVector2D`
- **描述**: The maximum amount by which to scale a particle in each dimension.
