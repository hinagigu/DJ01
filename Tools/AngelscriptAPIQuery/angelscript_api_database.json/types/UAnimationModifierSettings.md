# UAnimationModifierSettings

**继承自**: `UDeveloperSettings`

## 属性

### DefaultAnimationModifiers
- **类型**: `TArray<TSubclassOf<UAnimationModifier>>`
- **描述**: Set of Animation Modifiers to be added whenever a new Animation Sequence is imported

### bApplyAnimationModifiersOnImport
- **类型**: `bool`
- **描述**: Whether or not to apply animation modifiers post (re)import
