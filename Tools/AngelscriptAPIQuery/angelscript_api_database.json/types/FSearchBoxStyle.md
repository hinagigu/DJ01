# FSearchBoxStyle

Represents the appearance of an SSearchBox

## 属性

### TextBoxStyle
- **类型**: `FEditableTextBoxStyle`
- **描述**: Style to use for the text box part of the search box

### ActiveFontInfo
- **类型**: `FSlateFontInfo`
- **描述**: Font to use for the text box part of the search box when a search term is entered

### UpArrowImage
- **类型**: `FSlateBrush`
- **描述**: Image to use for the search "up" arrow

### DownArrowImage
- **类型**: `FSlateBrush`
- **描述**: Image to use for the search "down" arrow

### GlassImage
- **类型**: `FSlateBrush`
- **描述**: Image to use for the search "glass"

### ClearImage
- **类型**: `FSlateBrush`
- **描述**: Image to use for the search "clear" button

### ImagePadding
- **类型**: `FMargin`
- **描述**: Padding to use around the images

### bLeftAlignSearchResultButtons
- **类型**: `bool`
- **描述**: If true, search result buttons appear to the left of the search text

### bLeftAlignGlassImageAndClearButton
- **类型**: `bool`
- **描述**: If true, glass image and clear button appear to the left of the search text
