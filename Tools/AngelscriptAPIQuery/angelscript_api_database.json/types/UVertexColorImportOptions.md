# UVertexColorImportOptions

**继承自**: `UObject`

## 属性

### UVIndex
- **类型**: `int`
- **描述**: Texture Coordinate Channel to use for Sampling the Texture

### LODIndex
- **类型**: `int`
- **描述**: LOD Index to import the Vertex Colors to

### bRed
- **类型**: `bool`
- **描述**: Red Texture Channel

### bBlue
- **类型**: `bool`
- **描述**: Blue Texture Channel

### bGreen
- **类型**: `bool`
- **描述**: Green Texture Channel

### bAlpha
- **类型**: `bool`
- **描述**: Alpha Texture Channel

### bImportToInstance
- **类型**: `bool`
- **描述**: Whether or not to import the Vertex Colors to Mesh Component instance or the underlying Static Mesh
