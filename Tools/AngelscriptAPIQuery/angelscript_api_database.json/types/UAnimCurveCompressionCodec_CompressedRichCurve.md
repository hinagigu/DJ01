# UAnimCurveCompressionCodec_CompressedRichCurve

**继承自**: `UAnimCurveCompressionCodec`

## 属性

### MaxCurveError
- **类型**: `float32`
- **描述**: Max error allowed when compressing the rich curves

### UseAnimSequenceSampleRate
- **类型**: `bool`
- **描述**: Whether to use the animation sequence sample rate or an explicit value

### ErrorSampleRate
- **类型**: `float32`
- **描述**: Sample rate to use when measuring the curve error
