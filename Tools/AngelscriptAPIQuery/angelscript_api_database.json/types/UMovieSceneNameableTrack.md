# UMovieSceneNameableTrack

**继承自**: `UMovieSceneTrack`

Base class for movie scene tracks that can be renamed by the user.

## 属性

### DisplayName
- **类型**: `FText`
- **描述**: The track's human readable display name.

### TrackRowDisplayNames
- **类型**: `TArray<FText>`
- **描述**: The track display name per row.
