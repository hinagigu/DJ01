# FPaperTileInfo

This is the contents of a tile map cell

## 属性

### TileSet
- **类型**: `UPaperTileSet`
- **描述**: The tile set that this tile comes from

### PackedTileIndex
- **类型**: `int`
- **描述**: This is the index of the current tile within the tile set
