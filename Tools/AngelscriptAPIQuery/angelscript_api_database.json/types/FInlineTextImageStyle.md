# FInlineTextImageStyle

Represents the appearance of an inline image used by rich text

## 属性

### Image
- **类型**: `FSlateBrush`
- **描述**: Image to use when the slider thumb is in its normal state

### Baseline
- **类型**: `int16`
- **描述**: The offset from the bottom of the image height to the baseline.
