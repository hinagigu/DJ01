# UCameraMode

**继承自**: `UObject`

A camera mode asset, which runs a hierarchy of camera nodes to drive
the behavior of a camera.

## 属性

### RootNode
- **类型**: `UCameraNode`
- **描述**: Root camera node.

### EnterTransitions
- **类型**: `TArray<FCameraModeTransition>`
- **描述**: List of enter transitions for this camera mode.

### ExitTransitions
- **类型**: `TArray<FCameraModeTransition>`
- **描述**: List of exist transitions for this camera mode.
