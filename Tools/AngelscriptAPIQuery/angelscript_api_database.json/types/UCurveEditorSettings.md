# UCurveEditorSettings

**继承自**: `UObject`

Serializable options for curve editor.

## 属性

### bAutoFrameCurveEditor
- **类型**: `bool`

### bShowBars
- **类型**: `bool`

### FrameInputPadding
- **类型**: `int`
- **描述**: Number of pixels to add as padding in the input axis when framing curve keys

### FrameOutputPadding
- **类型**: `int`
- **描述**: Number of pixels to add as padding in the output axis when framing curve keys

### bShowBufferedCurves
- **类型**: `bool`

### bShowCurveEditorCurveToolTips
- **类型**: `bool`

### TangentVisibility

### bSnapTimeToSelection

### CustomColors

### ParentSpaceCustomColor
- **类型**: `FLinearColor`

### WorldSpaceCustomColor
- **类型**: `FLinearColor`

### ControlSpaceCustomColors
- **类型**: `TArray<FCustomColorForSpaceSwitch>`

### TreeViewWidth
- **类型**: `float32`
