# FRigUnit_GetCurveValue

GetCurveValue is used to retrieve a single float from a Curve

## 属性

### Curve
- **类型**: `FName`

### Valid
- **类型**: `bool`

### Value
- **类型**: `float32`
- **描述**: The current transform of the given Curve - or identity in case it wasn't found.
