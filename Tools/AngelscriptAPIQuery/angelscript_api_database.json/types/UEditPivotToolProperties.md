# UEditPivotToolProperties

**继承自**: `UInteractiveToolPropertySet`

Standard properties of the Edit Pivot operation

## 属性

### bApplyToAllLODs
- **类型**: `bool`
- **描述**: If checked, the baked transform will be applied to all available LODs. Has no effect on selections without LODs.

### bEnableSnapDragging
- **类型**: `bool`
- **描述**: When enabled, click-drag to reposition the Pivot

### RotationMode
- **类型**: `EEditPivotSnapDragRotationMode`
- **描述**: When Snap-Dragging, align source and target normals
