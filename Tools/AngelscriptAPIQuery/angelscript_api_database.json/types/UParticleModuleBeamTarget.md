# UParticleModuleBeamTarget

**继承自**: `UParticleModuleBeamBase`

## 属性

### TargetMethod
- **类型**: `Beam2SourceTargetMethod`
- **描述**: The method flag.

### TargetName
- **类型**: `FName`
- **描述**: The target point sources of each beam, when using the end point method.

### Target
- **类型**: `FRawDistributionVector`
- **描述**: Default target-point information to use if the beam method is endpoint.

### TargetTangentMethod
- **类型**: `Beam2SourceTargetTangentMethod`
- **描述**: The method to use for the Target tangent.

### TargetTangent
- **类型**: `FRawDistributionVector`
- **描述**: The tangent for the Target point for each beam.

### TargetStrength
- **类型**: `FRawDistributionFloat`
- **描述**: The strength of the tangent from the Target point for each beam.

### LockRadius
- **类型**: `float32`
- **描述**: Default target-point information to use if the beam method is endpoint.

### bTargetAbsolute
- **类型**: `bool`

### bLockTarget
- **类型**: `bool`

### bLockTargetTangent
- **类型**: `bool`

### bLockTargetStength
- **类型**: `bool`
