# __EMonoChannelUpmixMethod

Enumeration defines how to treat mono 2D playback. Mono sounds need to upmixed to stereo when played back in 2D.

## 属性

### Linear
- **类型**: `EMonoChannelUpmixMethod`

### EqualPower
- **类型**: `EMonoChannelUpmixMethod`

### FullVolume
- **类型**: `EMonoChannelUpmixMethod`

### EMonoChannelUpmixMethod_MAX
- **类型**: `EMonoChannelUpmixMethod`
