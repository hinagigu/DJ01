# FBakeMultiMeshDetailProperties

## 属性

### SourceTexture
- **类型**: `UTexture2D`
- **描述**: Source mesh color texture that is to be resampled into a new texture

### SourceTextureUVLayer
- **类型**: `int`
- **描述**: UV channel to use for the source mesh color texture
