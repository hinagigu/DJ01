Specifies which shader generated offsets should included in the world position (displacement/WPO etc.)

### WPT_ExcludeAllShaderOffsets
- **类型**: `EWorldPositionIncludedOffsets`

### WPT_CameraRelative
- **类型**: `EWorldPositionIncludedOffsets`

### WPT_CameraRelativeNoOffsets
- **类型**: `EWorldPositionIncludedOffsets`

### WPT_MAX
- **类型**: `EWorldPositionIncludedOffsets`
