# DJ01 RPG游戏开发计划

## 项目概述
基于UE5和Lyra框架开发的RPG游戏，利用ModularGameplay和GameFeatures系统实现模块化开发。

## 开发时间安排
- 工作日每天约1小时开发时间
- 周末可能有额外开发时间
- 预计总体开发周期6-8个月

## 文档目录
- `development_plan.md`: 详细的开发计划和里程碑
- `features/`: 具体功能模块的详细设计文档
- `technical/`: 技术实现相关的文档
- `assets/`: 资源需求和规划文档
- `weekly_goals.md`: 每周目标和进度跟踪

## 开发原则
1. 保持小步快跑，每次开发专注于一个小功能点
2. 充分利用Lyra框架的已有功能
3. 优先实现核心玩法
4. 保持代码和设计文档的同步更新
5. 定期回顾和调整计划

## 版本控制
- 使用Git进行版本控制
- 每个功能开发使用独立分支
- 定期合并到主分支
