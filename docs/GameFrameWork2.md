# 🎮 "太多该死勇者" 游戏框架文档 v2.0

## 🌍 游戏基本概念

### 核心设定

【游戏类型】3D动作RPG + 社会模拟 + 剧情解谜
【开发引擎】Unreal Engine 5（参考Lyra框架）
【游戏节奏】有意义的战斗 + 深度社交 + 逐层揭露的真相
【核心主题】从英雄梦想到发现产业骗局的冲击之旅
【目标时长】主线20-30小时，完全体验50+小时

### 世界观简化版

【背景】格兰维尔城邦联盟，一个魔法朋克的城邦社会
【人口】约14万人的独立城邦
【核心产业】讨伐军（勇者）+ 魔物威胁应对体系
【隐藏真相】所有魔物都是批量繁殖的产品，魔王是人造的生物武器
【玩家旅程】从相信英雄神话，到发现制造业真相

### 与原设定的关键变更

| 对比项 | v1.0 (旧) | v2.0 (新) |
|--------|-----------|-----------|
| 游戏类型 | 2D像素日常模拟 | 3D动作RPG+社会模拟 |
| 开发引擎 | Unity 2D / Godot | Unreal Engine 5 (Lyra) |
| 核心玩法 | 职业工作+社交 | 战斗+探索+社交+调查 |
| 战斗系统 | 几乎没有 | GAS驱动的动作战斗 |
| 魔物真相 | 演员假扮 | 繁殖场批量生产 |
| 主角定位 | 普通魔法师 | 新人勇者 |

## 👤 玩家角色设定

### 主角背景

【身份】刚通过考核的新人勇者（讨伐军编外人员）
【来源】来自边远村庄，怀揣成为英雄的梦想
【状态】试用期勇者，需要完成任务证明自己
【特点】战斗中观察力敏锐，容易发现异常
【初始认知】完全相信官方叙事：魔物是天然威胁，勇者是保护者

### 角色成长路径

【战斗成长】
- 技能树：近战/远程/魔法三大分支
- 装备系统：武器、防具、饰品的升级路线
- 能力获取：通过GAS系统实现技能解锁和强化
- 战斗风格：玩家可以培养独特的战斗方式

【认知成长】
- 第一阶段：纯粹的英雄主义者
- 第二阶段：开始注意到违和感
- 第三阶段：主动调查真相
- 第四阶段：面临立场选择
- 第五阶段：做出最终决定

### 可选起始背景（影响初始技能和剧情）

【战士之子】
- 父亲是退役勇者（或"失踪"勇者）
- 初始近战技能+1
- 特殊剧情：追查父亲的真正下落

【魔法学徒】
- 曾在魔法学院学习基础魔法
- 初始魔法技能+1
- 特殊剧情：学院隐藏的研究项目

【农村青年】
- 来自被"魔物袭击"的村庄
- 初始生存技能+1
- 特殊剧情：当年袭击的真相

【工人后代】
- 父母在"军事禁区"工作（繁殖场）
- 初始调查技能+1
- 特殊剧情：父母的秘密工作

## ⚔️ 战斗系统框架

### 基于GAS的技能体系

【Gameplay Ability System 应用】

基础能力类型：
- 主动技能（Active Abilities）：玩家主动释放的攻击/防御技能
- 被动技能（Passive Abilities）：持续生效的增益效果
- 反应技能（Reaction Abilities）：特定条件触发的技能
- 终极技能（Ultimate Abilities）：需要积累资源释放的大招

技能获取方式：
- 等级解锁：达到特定等级自动获得
- 任务奖励：完成特定任务获得
- 装备附带：某些装备自带技能
- 发现学习：在探索中发现技能书

【Gameplay Effect 应用】

状态效果：
- 增益效果：攻击力提升、防御强化、速度加快
- 减益效果：中毒、灼烧、冰冻、眩晕
- 持续效果：生命恢复、魔力回复
- 特殊效果：无敌、隐身、嘲讽

### 敌人设计（魔物分类）

【繁殖魔物】（正常投放的产品）
- 行为特征：攻击模式可预测，有明显的"弱点"
- 战斗难度：按等级设计，从F级到S级
- 隐藏线索：可能掉落"编号标签"、愈合速度异常
- AI设计：使用行为树，但刻意设计"规律"

| 等级 | 代表魔物 | 难度描述 | 投放频率 |
|------|----------|----------|----------|
| F级 | 史莱姆、小型昆虫 | 新手练习 | 最高 |
| E级 | 哥布林、野狼 | 日常任务 | 高 |
| D级 | 兽人、巨蜘蛛 | 团队配合 | 中 |
| C级 | 食人魔、石像鬼 | 需要策略 | 低 |
| B级 | 小型龙、死灵法师 | 精英挑战 | 很低 |
| A级 | 巨龙、恶魔将军 | 季度事件 | 极低 |
| S级 | 古代魔兽、灾厄之子 | 年度事件 | 稀有 |

【失控魔物】（投放后失去控制的产品）
- 行为特征：比正常魔物更加狂暴、不可预测
- 战斗难度：同等级下更危险
- 隐藏线索：可能有"收容小队"同时在追踪
- AI设计：行为树更加激进，少有"示弱"模式

【人造魔王】（最高级定制产品）
- 行为特征：拥有"智慧"表现，会发表宣言
- 战斗难度：需要大规模团队协作
- 隐藏线索：某些"能力"过于"恰到好处"
- AI设计：剧本化的战斗流程，设计好的"弱点"

### 战斗中的真相发现

【第一层：个体违和感】
- 同一区域的魔物外观高度相似
- 攻击模式过于"规律"
- 某些魔物在即将致命时会"示弱"
- 战利品中偶尔发现奇怪的金属片（编号标签）

【第二层：系统性异常】
- 魔物出现的时间地点过于"巧合"
- 勇者伤亡率的官方数据与实际不符
- 某些区域被标记为"污染区"禁止进入
- 夜间有神秘车队在运输"货物"

【第三层：产业链证据】
- 发现繁殖场的位置
- 接触到繁殖场工人
- 获得内部文件或记录
- 亲眼目睹魔物的"生产"过程

## 🏙️ 世界地图设计

### 格兰维尔首都（主要活动区域）

【冒险者区】
- 讨伐军总部：接受任务、领取报酬
- 冒险者公会：编外勇者的聚集地
- 装备商店街：购买和升级装备
- 训练场：学习新技能、练习战斗
- 玩家住所：个人基地，可升级扩建

【平民区】
- 市场广场：信息交流、购买物资
- 酒馆：NPC聚集地，获取谣言和任务
- 住宅区：NPC居住地，社交互动
- 神庙：特殊服务和仪式

【贵族区】（后期解锁）
- 皇宫外围：高级任务来源
- 贵族俱乐部：接触上层社会
- 隐藏线索：上层人士的可疑对话

【地下区】（隐藏区域）
- 黑市：购买非法物品和情报
- 反抗组织据点：接触异议者
- 秘密通道：连接各个区域

### 郊外任务区域

【东部平原】
- 难度：F-E级魔物
- 特色：开阔地形，适合新手
- 隐藏：某些区域"魔物密度"异常规律

【南部森林】
- 难度：E-D级魔物
- 特色：复杂地形，伏击战术
- 隐藏：发现可疑的运输痕迹

【西部山地】
- 难度：D-C级魔物
- 特色：垂直战斗，地形利用
- 隐藏：被标记为"矿区禁区"的区域

【北部边境】
- 难度：C-B级魔物
- 特色：恶劣环境，团队挑战
- 隐藏：繁殖场所在地

### 机密区域（剧情解锁）

【繁殖场设施】
- 12座分布在偏远地区
- 伪装为"军事禁区"或"污染区"
- 需要特定剧情进度才能接近
- 最终揭露真相的关键场所

【禁忌研究所】
- 位于北方冰原
- 魔王制造的最高机密设施
- 游戏后期的终极探索目标

## ⏰ 游戏流程设计

### 第一章：英雄的起点 (5-8小时)

【开场】
- 主角通过新人考核，正式成为编外勇者
- 分配到首都冒险者区，开始新生活
- 接受导师指导，学习基本战斗技能
- 完成第一次正式讨伐任务

【核心体验】
- 学习战斗系统，掌握基本技能
- 接受各种等级的讨伐任务
- 与其他勇者和NPC建立关系
- 享受"英雄成长"的满足感

【隐藏细节】（玩家可能注意不到）
- 魔物出现的时间太"准时"
- 同批魔物外观高度相似
- 偶尔捡到的奇怪金属片
- 某些老勇者的讽刺性言论

【章节事件】
- 第一次击败D级魔物的成就感
- 被官方媒体采访的"英雄时刻"
- 导师透露的一些"经验之谈"

### 第二章：裂痕的开始 (5-8小时)

【转折事件】
- 一次任务中遭遇"失控魔物"
- 与官方描述不符的战斗体验
- 发现编号标签并开始困惑
- 目击"收容小队"的可疑行动

【新增内容】
- 解锁更多战斗技能和装备
- 更高难度的讨伐任务
- 与"老油条"勇者的深度交流
- 接触首都的不同社会阶层

【疑点显现】
- 伤亡统计数据的矛盾
- 禁区限制的不合理性
- 某些NPC的欲言又止
- 官方解释的漏洞

【章节事件】
- 一位同期勇者的"意外死亡"
- 调查导师失踪/沉默的原因
- 首次接触质疑官方的异议者

### 第三章：地下的真相 (5-8小时)

【调查选择】
- 玩家可以选择深入调查或继续当"好勇者"
- 接触地下反抗组织
- 潜入禁区收集证据
- 与繁殖场工人/知情者对话

【分歧路径】

路线A - 体制内：
- 继续正常的勇者工作
- 获得更高的军衔和待遇
- 逐渐发现即使在体制内也能接触真相
- 面临"知情不报"的道德困境

路线B - 调查者：
- 主动追查真相
- 承担被发现的风险
- 接触更多内幕信息
- 面临人身安全威胁

【章节事件】
- 第一次看到繁殖场的照片/记录
- 与知情高层的一次深入对话
- 发现父母/导师/朋友与真相的关联

### 第四章：抉择时刻 (5-8小时)

【真相揭露】
- 完全理解繁殖产业的运作机制
- 知道魔王也是人造产品
- 了解200年的历史演变
- 面临最终的立场选择

【多重路线】

结局A - 维持现状：
- 选择融入体系
- 利用真相为自己谋取利益
- 代价：成为共谋者

结局B - 渐进改良：
- 在体制内推动改变
- 尝试减少繁殖量和伤亡
- 代价：漫长且可能失败

结局C - 真相曝光：
- 公开一切证据
- 引发社会剧变
- 代价：可能的混乱和牺牲

结局D - 逃离：
- 离开格兰维尔
- 寻找没有谎言的世界
- 代价：放弃一切关系

结局E - 毁灭：
- 释放繁殖场的所有魔物
- 终结这个虚伪的体系
- 代价：毁灭性后果

【最终Boss】
- 根据选择的路线，最终Boss不同
- 可能是：人造魔王 / 体制高层 / 收容部队 / 失控的繁殖场

## 🤝 社交系统设计

### 关系类型

【友谊系统】
- 与NPC的基础关系
- 影响对话选项和任务获取
- 通过互动、赠礼、帮助提升

【信任系统】
- 影响NPC是否愿意分享秘密
- 某些真相只有高信任度才能获取
- 关键NPC的信任度决定剧情分支

【派系声誉】
- 讨伐军声誉：影响官方任务和待遇
- 平民声誉：影响社区支持和信息
- 地下声誉：影响反抗组织的信任
- 贵族声誉：影响上层接触机会

### 关键NPC设计

【导师 - 雷克斯老兵】
- 退役的资深勇者
- 知道一些真相但选择沉默
- 可以教授高级战斗技能
- 剧情关键：他的过去隐藏着重要线索

【同期 - 艾拉】
- 与主角同期入职的女勇者
- 乐观热血的英雄主义者
- 可发展的恋爱对象
- 剧情关键：她的家人在繁殖场工作

【情报商 - 老鼠】
- 地下情报网络的核心人物
- 出售各种情报和线索
- 道德模糊的生存主义者
- 剧情关键：连接地上和地下世界

【异议者 - 玛丽安】
- 被"治疗"过但逃脱的前记者
- 拥有关于真相的碎片信息
- 反抗组织的联络人
- 剧情关键：引导玩家发现真相

【知情高层 - 公爵夫人】
- 表面的上流社会贵妇
- 实际是繁殖总局的高层
- 对体制有复杂的看法
- 剧情关键：可能的内部改革者或敌人

### 社交活动

【日常互动】
- 对话：了解NPC的想法和信息
- 赠礼：提升好感度
- 帮助：完成NPC的个人请求
- 邀约：参与特殊社交事件

【特殊事件】
- 节日活动：全城庆典，特殊任务
- 个人事件：NPC的个人剧情线
- 派系活动：各派系的专属事件
- 隐藏事件：满足特定条件触发

## 🛠️ 技术实现框架

### UE5 + Lyra 架构应用

【Experience System】
- 用于管理不同游戏场景和状态
- 战斗场景 / 城镇场景 / 剧情场景的切换
- 每个Experience定义自己的规则和系统

【Gameplay Ability System】
- 所有战斗技能基于GAS实现
- Ability：技能定义
- Effect：状态效果
- Attribute：属性系统
- Cue：技能表现

【Modular Gameplay Features】
- 战斗系统作为独立Feature
- 社交系统作为独立Feature
- 调查系统作为独立Feature
- 支持后期扩展和DLC

### AI系统设计

【魔物AI - 行为树】

普通繁殖魔物：
- 巡逻行为：固定路线（故意设计成可预测）
- 战斗行为：规律的攻击模式
- 特殊行为：血量低时"示弱"
- 隐藏行为：偶尔的"异常"作为线索

失控魔物：
- 更激进的攻击模式
- 更少的"示弱"行为
- 不遵循标准战斗流程
- 给玩家不同的战斗体验

Boss/魔王：
- 多阶段战斗设计
- 剧本化的能力释放
- 设计好的"弱点时机"
- 史诗级的战斗表现

【NPC AI - 状态机】
- 日程系统：NPC有自己的作息
- 反应系统：对玩家行为的反应
- 对话系统：基于好感度和剧情的对话
- 行为系统：NPC的独特行为模式

### 开发优先级

【Phase 1：核心战斗】
- 基础移动和战斗系统
- GAS技能框架搭建
- 基础魔物AI
- 一个完整的战斗区域

【Phase 2：城镇基础】
- 首都冒险者区的构建
- 基础NPC和对话系统
- 任务系统框架
- 商店和装备系统

【Phase 3：剧情框架】
- 主线剧情第一章
- 关键NPC的完整剧情线
- 真相发现的触发系统
- 分支选择系统

【Phase 4：系统深化】
- 完整的技能树
- 社交系统完善
- 派系系统
- 更多区域和内容

【Phase 5：内容完善】
- 全部主线剧情
- 多结局系统
- 隐藏内容和彩蛋
- 平衡性调整

## 🎨 视觉和音效设计

### 视觉风格

【整体风格】
- 魔法朋克：魔法替代科技的世界观
- 写实与风格化的平衡
- 明亮的城镇 vs 阴暗的真相
- 随剧情推进的视觉氛围变化

【场景设计】
- 城镇：温暖、繁华、生活气息
- 战斗区域：危险、紧张、变化多样
- 繁殖场：工业、冰冷、压抑
- 禁忌研究所：黑暗、诡异、禁忌感

【魔物设计】
- 繁殖魔物：标准化、批量感、微妙的"工业产品"特征
- 失控魔物：更野性、更混乱、更危险
- 魔王：设计感、戏剧性、"定制产品"的精致

### 音效设计

【背景音乐】
- 城镇：活泼、温暖、日常感
- 战斗：紧张、激昂、英雄感
- 调查：神秘、不安、悬疑感
- 真相揭露：震撼、压抑、冲击感

【环境音效】
- 城镇：人声、脚步、商业活动
- 野外：风声、虫鸣、远处的战斗
- 繁殖场：机械运转、魔物嘶吼、压抑的回声

【战斗音效】
- 武器打击：有力、清晰的反馈
- 魔法释放：魔幻感、力量感
- 魔物反应：痛苦、愤怒、（偶尔的）困惑
